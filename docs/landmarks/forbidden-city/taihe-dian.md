# 太和殿

![太和殿](https://images.unsplash.com/photo-1763058075410-d86b870594af?w=1200&q=80)

> 金銮高踞，万殿之尊。这里是中国古代建筑的最高规格，也是皇权天命的实体化身。

## 简介

太和殿，俗称"金銮殿"，是紫禁城外朝三大殿之首，也是中国现存体量最大、等级最高的木结构大殿。它矗立在三层汉白玉须弥座台基之上，通高35.05米，面阔十一间，进深五间，建筑面积达2377平方米——在中国古建筑的等级序列中，"十一间"是绝无仅有的孤例，太和殿是唯一的。

明清两代，凡遇皇帝登基、大婚、册立皇后、命将出征，以及元旦、冬至、万寿节三大朝贺，均在太和殿举行最隆重的典礼。这里不处理日常政务，而是纯粹的国家礼仪空间——每一次大殿门开，都意味着帝国最重大的时刻到来。

太和殿始建于明永乐十八年（1420年），初名"奉天殿"，嘉靖年间改名"皇极殿"，清顺治二年（1645年）定名"太和殿"，取《周易》"保合太和"之意。现存建筑为清康熙三十四年（1695年）重建。

## 建筑与景观

| 项目 | 数据 |
|:---|:---|
| **面阔** | 十一间，63.96米 |
| **进深** | 五间，37.17米 |
| **通高** | 35.05米（含三层台基） |
| **建筑面积** | 2377平方米 |
| **屋顶形制** | 重檐庑殿顶（中国古建筑最高等级） |
| **蟠龙金柱** | 殿内72根（其中6根为沥粉贴金蟠龙柱） |
| **殿前广场** | 约25000平方米，可容万人朝贺 |
| **台基螭首** | 三层须弥座共1142个螭首排水口 |

太和殿的建筑语言处处彰显"至高无上"：屋顶采用最高等级的**重檐庑殿顶**，正脊两端安放中国现存最大的琉璃鸱吻（高3.4米）；檐角走兽多达十只（龙、凤、狮、天马、海马、狻猊、押鱼、獬豸、斗牛、行什），加一位骑凤仙人，是全国古建中走兽数量唯一的"十样俱全"——其他任何建筑最多只能用九只。

殿内的核心是那把**金漆雕龙宝座**（俗称"龙椅"），高约1.7米，髹金漆、雕蟠龙，是明代嘉靖年间遗物。宝座上方悬挂"建极绑猷"匾额（乾隆御笔），头顶是深邃的**蟠龙藻井**——一条巨龙从井底倒悬而下，口衔宝珠，即传说中的"轩辕镜"。殿内六根沥粉贴金蟠龙柱环绕宝座，与藻井金龙上下呼应，构成一个金碧辉煌的权力圣殿。

殿外三层汉白玉须弥座台基，栏板望柱雕龙刻凤，1142个螭首排水口在暴雨时会形成"千龙吐水"的壮观景象。

## 历史故事

### 李自成的火与康熙的重建

明末崇祯十七年（1644年），李自成攻入北京，撤离前纵火焚烧紫禁城，太和殿（时称皇极殿）化为灰烬。清军入关后，顺治帝在残破的宫城中即位，直到康熙年间才得以重建。康熙三十四年（1695年），太和殿重建工程历时数年完工，我们今天看到的太和殿，正是那次重建的成果。它比明代原版略小（明奉天店面阔三十丈），但依然冠绝天下。

### 袁世凯的"龙椅"闹剧

1915年，袁世凯复辟称帝，嫌太和殿的明代龙椅"不够气派"，竟命人搬走原椅，换上一把中西合璧的高背大椅。那把不伦不类的椅子很快随着帝制覆灭被移走，明代龙椅却不知所踪。直到1959年，文物专家朱家溍在一处库房角落里发现了残破的原椅，经修复后重新安放回太和殿——如今游客透过殿门望见的龙椅，正是那把失而复得的明代真品。

## 必看亮点

- **金漆雕龙宝座与蟠龙藻井**：透过殿门向内望去，龙椅、金柱、藻井、轩辕镜层层纵深，是紫禁城最震撼的视觉中轴线。
- **十只檐角走兽**：绕到殿侧仰望檐角，数一数全国唯一的"十样俱全"脊兽队列。
- **千龙吐水**：若恰逢暴雨，三层台基的1142个螭首同时喷水，是太和殿最难得的奇景。
- **殿前广场的尺度**：站在2.5万平方米的空旷广场上仰望大殿，体会"万人朝贺"的空间压迫感。

## 参观贴士

- **最佳观赏点**：太和门内的金水桥畔是拍摄大殿全景的经典机位；清晨开馆时广场空旷，光线从东南斜照，最显恢宏。
- **只能外观**：太和殿内部不对外开放，游客只能在殿门外隔栏观赏，建议带望远镜或长焦镜头看清藻井细节。
- **人最多的时段**：10:00-14:00是太和殿广场人流高峰，想拍空镜请赶在8:30开馆后直奔此处。
- **别错过铜器**：殿前月台上的日晷、嘉量、铜龟、铜鹤，是皇权"授时"与"度量"的象征，值得细看。
