# 劳动人民文化宫

> "明清太庙，皇家祭祖圣地。"

## 简介

劳动人民文化宫位于北京市中心，天安门东侧，是明清两代皇帝祭祀祖先的太庙。1950年，改为劳动人民文化宫，成为北京市总工会下属的文化活动场所。

## 历史沿革

### 太庙始建

明永乐十八年（1420年），明成祖朱棣按照"左祖右社"的制度，在天安门东侧修建太庙，祭祀祖先。

### 明清太庙

明清两代，太庙是皇帝祭祀祖先的圣地，也是皇家祭祖的重要场所。

### 改为文化宫

1950年，太庙改为"劳动人民文化宫"，成为北京市总工会下属的文化活动场所，对劳动人民开放。

## 建筑特色

### 太庙

太庙是劳动人民文化宫的核心建筑，是明清皇帝祭祀祖先的地方。

#### 布局

太庙按照周礼"左祖右社"的制度布局，规模宏大。

| 建筑 | 说明 |
|:---|:---|
| **戟门** | 正门 |
| **前殿** | 祭祀前准备的地方 |
| **中殿** | 供奉祖先牌位的地方 |
| **后殿** | 供奉祖先遗物的地方 |

### 古柏

劳动人民文化宫内有大片古柏林，是明永乐年间种植的，已有600余年历史。

### 红墙

劳动人民文化宫的红墙与故宫的红墙相连，是北京中轴线的重要组成部分。

## 文化价值

劳动人民文化宫是明清太庙，是中国古代宗庙礼制的重要遗存。

劳动人民文化宫与故宫、中山公园相对应，体现了中国古代"左祖右社"的礼制。
