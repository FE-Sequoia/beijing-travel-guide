## 简介

什刹海位于北京市西城区，是北京城内面积最大、风貌保存最完整的一片历史街区。什刹海由前海、后海和西海（积水潭）三个相连的湖泊组成，周边分布着大量保存完好的胡同和四合院，是老北京风貌最集中的地区之一。

## 历史沿革

### 形成与发展

什刹海地区在金代就已经形成湖泊。元代时，这里是京杭大运河的终点码头，也是当时世界上最繁华的商业区之一。

#### 元代

元代定都北京（当时称大都），什刹海成为大运河的终点，商贾云集，舟楫往来，"舳舻蔽水"，极为繁华。

#### 明清时期

明清时期，什刹海周边逐渐形成了众多王府、寺庙和民居。这里的胡同和四合院保存了大量老北京的风貌。

### 近代变迁

20世纪90年代以来，什刹海地区进行了大规模的保护性修缮，成为北京最具特色的旅游区之一。

## 主要区域

### 前海

前海是什刹海最热闹的区域，周边酒吧、餐厅林立，是夜生活的聚集地。

#### 银锭桥

银锭桥是前海的标志性建筑，站在桥上可以远眺西山。桥名取自其形似倒置的银锭。

### 后海

后海相对安静，周边有许多名人故居和传统四合院。

#### 宋庆龄故居

宋庆龄故居位于后海北沿，是一座保存完好的四合院。宋庆龄晚年在此居住。

### 西海（积水潭）

西海最为安静，适合散步和休闲。周边有郭守敬纪念馆等景点。

## 游览攻略

### 推荐路线

**半日游（胡同游）**：

银锭桥 → 烟袋斜街 → 前海 → 恭王府 → 后海 → 宋庆龄故居

**一日游（深度游）**：

上午：恭王府 → 前海 → 银锭桥 → 烟袋斜街
下午：后海 → 宋庆龄故居 → 西海 → 郭守敬纪念馆
晚上：后海酒吧街

### 实用贴士

1. **最佳时间**：傍晚时分景色最美，夕阳洒在湖面上，别有一番风味
2. **胡同游**：建议租一辆人力三轮车，听老北京讲胡同故事
3. **酒吧消费**：后海酒吧价格偏高，建议先问价再消费
4. **拍照位置**：银锭桥、恭王府门前、后海湖边都是绝佳拍照点
5. **注意安全**：夜间人流量大，注意保管好个人财物

## 文化意义

什刹海是老北京生活的缩影。在这里，你可以看到胡同里晾晒的衣服、巷子里下棋的老人、湖边垂钓的市民，感受到最地道的北京生活。

"先有什刹海，后有北京城"，这句话虽然有些夸张，但也说明了什刹海在北京历史上的重要地位。
