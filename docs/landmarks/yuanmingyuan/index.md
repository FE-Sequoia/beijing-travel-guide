# 圆明园

![圆明园遗址](https://images.unsplash.com/photo-1747001539757-285c9443a55b?w=1200&q=80)

> "有一天，两个强盗闯进了圆明园，一个强盗洗劫，另一个强盗放火……在历史面前，这两个强盗一个叫法兰西，一个叫英吉利。"
> —— 维克多·雨果《就英法联军远征中国给巴特勒上尉的信》（1861年）

## 简介

圆明园位于北京西北郊，与[颐和园](/landmarks/summer-palace/)毗邻，由**圆明园、长春园、绮春园**三园组成，统称"圆明三园"，占地350多公顷，其中水面面积约140公顷，建筑面积达20万平方米。自康熙四十八年（1709年）始建，历经雍正、乾隆、嘉庆、道光、咸丰五朝150余年营建，园中有景观150余处，曾被誉为"万园之园"和"一切造园艺术的典范"。

1860年10月，英法联军攻入北京，对圆明园大肆劫掠后纵火焚毁，大火三日三夜不熄。1900年八国联军侵华期间，园内残存建筑再遭洗劫。曾经的琼楼玉宇，如今只剩断壁残垣静立于荒草之间。

今天的圆明园是**全国重点文物保护单位**和**爱国主义教育基地**。那些矗立在夕阳下的大理石残柱，无声地诉说着那段屈辱的历史，提醒后人：勿忘国耻，吾辈自强。

## 推荐路线

### 精华游（约3小时）

地铁4号线圆明园站 → 南门入园 → **绮春园**（正觉寺）→ 三园交界 → 福海西岸漫步 → **西洋楼遗址区**（谐奇趣 → 万花阵 → 海晏堂 → **大水法** → 远瀛观）→ 东门出

> 💡 此路线以西洋楼遗址为核心，是绝大多数游客的选择。遗址区需另购15元门票。

### 深度游（约5-6小时）

南门 → **绮春园**（正觉寺、鉴碧亭）→ **福海景区**（蓬岛瑶台环湖）→ 方外观 → 养雀笼 → **西洋楼全景区**（谐奇趣、蓄水楼、万花阵、海晏堂、大水法、观水法、线法山）→ **文源阁**遗址 → 圆明园博物馆（全景模型展）→ 南门出

> 💡 园区面积广阔，深度游建议乘坐园内观光电瓶车（10-15元/站）节省体力。

## 历史意义

圆明园不仅是一座园林，更是一段民族记忆的载体：

| 时间 | 事件 |
|:---|:---|
| 1709年 | 康熙帝赐园予皇四子胤禛（后来的雍正帝），"圆明"二字为康熙御笔 |
| 1709-1860年 | 历经五朝150余年营建，成为"万园之园"，清代政治中心之一 |
| 1860年10月 | 英法联军劫掠焚毁，大火三日不熄 |
| 1900年 | 八国联军侵华，残存建筑再遭劫掠 |
| 1976年 | 成立圆明园管理处，开始系统保护 |
| 1988年 | 被国务院列为全国重点文物保护单位 |

圆明园的毁灭是中华民族的奇耻大辱，也是人类文明史上的一场浩劫。今天的遗址公园选择"保留残迹、不事重建"，让每一块残石都成为历史的见证。正如雨果所言，这是"世界文明史上的惨痛损失"。

更多圆明园的历史，可参阅[圆明园遗址历史专题](/history/yuanmingyuan-ruins)。
