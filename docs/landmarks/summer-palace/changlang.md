# 长廊

![长廊](https://images.unsplash.com/photo-1645526629357-16bbd762c8d5?w=1200&q=80)

> "一步一景，一画一世界。"
> —— 世界上最长的画廊，七百二十八米的丹青长卷

## 简介

长廊东起邀月门，西至石丈亭，沿昆明湖北岸蜿蜒逶迤，全长728米，共273间，是中国古典园林中最长的游廊，被载入吉尼斯世界纪录，誉为"世界第一长廊"。

它像一条彩带，将万寿山前的[排云殿](/landmarks/summer-palace/paiyun-dian)、[佛香阁](/landmarks/summer-palace/foxiang-ge)、乐寿堂、听鹂馆等景点串珠成链。游人循廊而行，晴不知日，雨不湿衣，山色湖光在廊柱间次第展开。

长廊始建于乾隆十五年（1750年）。咸丰十年（1860年）遭英法联军焚毁，现存建筑为光绪年间重建，廊内彩画亦为彼时重绘。1992年，长廊以"世界上最长的游廊"之姿被吉尼斯世界纪录收录，成为中国园林向世界递出的第一张名片。

## 建筑与景观

| 项目 | 内容 |
|:---|:---|
| **全长** | 728米，共273间 |
| **彩画数量** | 14000余幅苏式彩画，无一雷同 |
| **四季亭** | 留佳（春）、寄澜（夏）、秋水（秋）、清遥（冬）四座八角亭 |
| **附属建筑** | 对鸥舫、鱼藻轩两座临湖水榭 |
| **整体形态** | 随湖岸曲折，俯瞰如展翅之蝠，"蝠"谐"福" |

长廊并非笔直，而是随湖岸曲折，中部以排云门为界向两翼展开。廊中每隔一段便有一座八角亭歇脚，其中留佳、寄澜、秋水、清遥四亭分别象征春夏秋冬四时，亭名皆取自诗文，意趣隽永。

廊内梁枋之上，14000余幅苏式彩画题材大致可分四类：

| 彩画类别 | 内容举例 |
|:---|:---|
| **山水** | 西湖十景、江南风光，底本多出自乾隆南巡画师写生 |
| **人物故事** | 《西游记》《三国演义》《水浒传》《红楼梦》《聊斋》等 |
| **花鸟鱼虫** | 四季花卉、翎毛草虫，工写兼备 |
| **博古纹样** | 器物、几何图案，点缀于枋心箍头之间 |

## 历史故事

### 为母筑廊的孝心

乾隆十六年是崇庆皇太后的六十大寿。乾隆知道母亲最爱西湖山水，又怕北方风寒伤身，便下令沿昆明湖北岸修建长廊，让太后"不出宫门而览尽江南"。此后每逢雨雪，太后便由宫女搀扶漫步廊中，看廊外湖山烟雨，听檐下风铃叮咚。这条长廊因此又被称为"孝廊"——帝王家的奢华背后，也有一份寻常的母子之情。

### 一万四千幅画，没有一幅重样

光绪年间重绘长廊彩画时，朝廷从江南请来大批画师，限期完工。画师们以《芥子园画谱》为底本，博采历代名画粉本，山水取法西湖十景，人物囊括四大名著与民间传说。最难的是"无一雷同"——273间廊段，每一间的包袱画都需另起炉灶。据传验收时慈禧逐幅过目，稍有重复或败笔即令重绘，最终成就这座"走不完的画廊"。如今游人见到的彩画，大部分为1959年迎国庆时由京津画师按光绪旧制再次重绘，笔法虽新，格局依旧。

### 蝙蝠廊里藏"福"字

从空中俯瞰，长廊以排云门为中心向东西两翼伸展，中部微微向南凸出，整体恰似一只展翅的蝙蝠。"蝠"与"福"同音，蝙蝠纹本就是清宫最常见的吉祥图案。乾隆以"蝠形"布廊，等于在昆明湖畔画了一个巨大的"福"字，把对太后的祝寿之意藏进了平面布局里——造园家的心思，要飞起来才看得见。

## 必看亮点

1. **万幅彩画**：边走边找《西游记》《三国演义》故事画，是老少咸宜的"寻宝游戏"，人物画多集中于廊内侧梁枋。
2. **四季亭**：留意留佳、寄澜、秋水、清遥四座八角亭的匾额与位置，体会造园家以亭点景、以名寄时的巧思。
3. **框景成画**：廊柱与横枋天然构成画框，昆明湖、南湖岛、十七孔桥皆入画中，随手一拍即是明信片。
4. **对鸥舫与鱼藻轩**：廊中部伸向湖面的两座水榭，是凭栏观湖、喂鱼赏荷的最佳处。
5. **听鹂馆**：廊西侧的皇家戏台旧址，如今是著名的宫廷菜餐厅，逛累了可在此歇脚。

## 拍照机位

1. **廊内透视**：站在廊中央，利用立柱与彩绘的重复韵律拍纵深，广角镜头效果极佳。
2. **框景构图**：以廊柱为框拍昆明湖与十七孔桥，"画中游"变成"画外看画"。
3. **彩画特写**：侧光时段（清晨或傍晚）拍梁枋包袱画，色彩最饱和。

## 参观贴士

1. 长廊全天开放、无需额外购票，但旺季人潮汹涌，想拍空镜务必赶早（开园后一小时内）或傍晚。
2. 建议自东向西走（邀月门→石丈亭方向），顺光看彩画，且越走越接近[石舫](/landmarks/summer-palace/shifang)和画中游。
3. 细看彩画需仰头，颈椎不佳者量力而行；重点欣赏排云门两侧的"包袱画"精华段即可。
4. 廊内禁止倚坐栏杆、攀爬彩绘，苏式彩画怕汗怕蹭，人人爱护才能长久。
5. 长廊与[佛香阁](/landmarks/summer-palace/foxiang-ge)中轴线十字相交，逛完长廊正好从排云门拾级登山，动线无缝衔接。
6. 带本《西游记》或《三国》故事清单来"按图索骥"，比走马观花有趣十倍。

## 常见问题

**Q：长廊的彩画都是光绪年间的原作吗？**

不全是。光绪年间重绘后，1959年迎国庆又由京津画师按光绪旧制大规模重绘，如今所见大部分为彼时手笔，格局一仍其旧。

**Q：走完长廊需要多久？**

长廊全长728米，正常步速15-20分钟可走完；若逐间看彩画、亭中歇脚，建议留足40-60分钟。

**Q：雨天逛长廊体验如何？**

极佳。长廊本就是为雨雪天游湖而建，雨打湖面、廊外烟水朦胧，而廊内片雨不沾——这是颐和园最地道的"隐藏玩法"。

**Q：长廊适合老人小孩吗？**

非常适合。全程平路无坡，每隔数十米就有八角亭歇脚，是颐和园无障碍程度最高的路线之一。
