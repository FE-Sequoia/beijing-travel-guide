# 万福阁（白檀木弥勒大佛）

![万福阁高耸楼阁](https://plus.unsplash.com/premium_photo-1676035245908-e844e8d1664b?w=1200&q=80)

> "一木参天二十六米，飞桥三阁万福同归——登阁仰瞻，方知人间巨佛，竟由独木而生。"

## 简介

万福阁是雍和宫最北端、也是全寺最高大的建筑，外观三层、内为两层，黄瓦飞檐，巍峨冠于全寺。阁内供奉的白檀木弥勒大佛，以一整根白檀木雕刻而成，地上部分高18米，地下还埋入8米，通高26米，是世界上最大的独木雕刻佛像，1990年被载入吉尼斯世界纪录。

万福阁的建造与七世达赖喇嘛的一份厚礼直接相关：乾隆年间，为报答清廷出兵平定珠尔默特之乱、护持西藏的恩典，七世达赖喇嘛格桑嘉措以巨资购得一根产自印度的巨大白檀木，经西藏辗转运抵北京，历时三载。乾隆帝得此巨木，遂建万福阁以奉弥勒——一说亦为给生母崇庆皇太后祝寿祈福。巨木成佛像，高阁护巨佛，"万福"之名，寄寓万福骈臻、福泽绵长。

## 建筑与景观

| 项目 | 规格与内容 |
|:---|:---|
| 阁宇规制 | 外观三层、内为两层，全寺最高建筑，黄琉璃瓦歇山顶 |
| 弥勒大佛 | 整根白檀木雕弥勒立像，地上18米 + 地下8米，通高26米 |
| 世界纪录 | 1990年吉尼斯世界纪录"最大独木雕刻佛像" |
| 佛身披袍 | 传以数千匹黄缎制成披袍，工程历时三年 |
| 东侧 | 永康阁，以飞桥（悬空廊道）与万福阁相连 |
| 西侧 | 延宁阁，同样以飞桥与万福阁相连 |

弥勒大佛为未来佛弥勒的立像，体态颀长，垂目含笑，左手持净瓶（一说施与愿印），右手下垂，整木纹理顺佛身走势而雕，衣带流畅如生。佛像地下部分以巨井承托，深埋8米以保持稳定——工匠先立木、后建阁，阁因佛起，佛以阁尊。

### 飞桥三阁"阁中阁"

万福阁东西两侧，分峙永康阁与延宁阁两座配阁，各供奉小型佛像与经卷。三阁二层之间以悬空飞桥相通，桥身凌于殿院之上，形成"飞桥三阁"的奇观；从阁外仰望，楼阁重叠、飞桥如虹，又有"阁中阁"之称。这种以飞桥联络楼阁的形制，取法佛教绘画中的兜率天宫意象——弥勒菩萨正居于兜率天内院，等待降世成佛。

## 历史故事

### 白檀木的三年旅程

这根白檀木原产于印度，由七世达赖喇嘛以巨资购得，以报乾隆帝派兵安定西藏之恩。巨木先经陆路入藏，再辗转东运，水陆兼程，历时三年方抵北京。为运此木，沿途拆桥让路、征用牛马，其艰难可想。木至京城，乾隆帝见其巨硕无匹，遂命察罕达尔罕活佛设计佛像，由养心殿造办处"广木作""木作"的能工巧匠雕造，终成旷世巨佛。

### 先立佛，后建阁

26米的立像无处可"搬进"楼阁，工匠遂采用"先立佛、后建阁"之法：先将白檀木植于地基巨井之中，雕成佛像，再沿佛身逐层起阁封顶。故万福阁不是"装下了佛像"，而是"围绕着佛像生长出来"的——这在中国楼阁建筑史上亦属罕见巧思。

## 必看亮点

- **26米独木弥勒**：地上18米、地下8米，整根白檀木雕成，吉尼斯世界纪录保持者。
- **先立佛后建阁**：楼阁因佛而起的营造智慧，中国古建史上的孤例之一。
- **飞桥三阁"阁中阁"**：万福阁与永康阁、延宁阁以悬空飞桥相连，形制奇绝。
- **白檀木的来历**：七世达赖献礼、三年运木入京，汉藏情谊的实物见证。
- **二层平视佛面**：登阁至二层，可与弥勒佛面大致平视，仰望与平观两种视角皆不可错过。

## 参观贴士

- 万福阁为中轴线终点与高潮，建议放在行程最后，留足瞻仰时间。
- 阁内楼梯陡峭、光线幽暗，上下请扶栏慢行，注意脚下。
- 仰观巨佛时宜退至殿门附近，方能尽收全像；二层回廊可观佛身细节。
- 殿内禁止拍照，请勿触摸佛像及披袍。
- 文物保护需要，个别殿阁可能临时关闭，请以现场公告为准。
