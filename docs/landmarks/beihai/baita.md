# 白塔与琼华岛

![白塔与琼华岛](https://images.unsplash.com/photo-1621871183199-4ec39bb3c41c?w=1200&q=80)

> 碧波中央，一岛独秀；岛巅白塔，鎏金宝顶直指苍穹——这是北海的眼睛，也是北京城的天际坐标。

## 简介

**琼华岛**位于北海太液池南部，是全园的布局中心与精神所在。自辽金以来，这里便是皇家苑囿的核心，元代忽必烈营建大都时更以琼华岛为基准点，确立了整座都城的中轴格局。岛上林木葱茏、殿宇层叠，亭台楼阁依山就势，将北方山水的雄浑与江南园林的秀美熔于一炉。

耸立于琼华岛之巅的**白塔**，是北海乃至整个北京城最醒目的地标。它建于**清顺治八年（1651年）**，是在辽金"广寒殿"旧址上兴建的一座**藏式喇嘛塔**，通体洁白，高**35.9米**，塔顶覆以鎏金宝顶，在阳光下熠熠生辉。

登临琼华岛而四望，南可望故宫角楼与中南海的碧波，北可眺景山万春亭的琉璃，东眺可见团城古木，西眺则有五龙亭凌波。可以说，琼华岛是读懂北京中轴线与皇家苑囿格局的最佳观景点。

从造园思想看，北海承袭了中国古典园林"**一池三山**"的传统格局——以一片大湖（太液池）象征东海，以湖中及周边的琼华岛、团城、犀山台等象征海上仙山"蓬莱、方丈、瀛洲"。这一源自秦汉的仙境意象，经辽金元明清历代营建，最终凝定为今日"白塔凌波、仙山在望"的经典景观，琼华岛正是这座人间仙岛的灵魂。

## 历史沿革

| 年代 | 大事 |
|:---|:---|
| 辽代 | 始建离宫别苑，岛上营建"瑶屿"行宫 |
| 金代 | 扩建为皇家苑囿，建"广寒殿"等仙山楼阁 |
| 元代 | 忽必烈以琼华岛为中心营建大都，改名"万寿山" |

| 清乾隆年间 | 大修琼华岛，增建漪澜堂、阅古楼诸景 |
| 当代 | 列为全国重点文物保护单位，对外开放 |

## 建筑与景观

### 白塔

| 项目 | 数据 |
|:---|:---|
| **建成年代** | 清顺治八年（1651年） |
| **前身** | 辽金"广寒殿"旧址 |
| **形制** | 藏式喇嘛塔（覆钵式） |
| **高度** | 35.9米 |
| **塔顶** | 鎏金宝顶，由天盘、地盘、日、月、火焰组成 |
| **结构** | 砖石结构，塔身洁白，南面设"眼光门"（时轮金刚门） |

白塔由塔基、塔身、相轮、宝顶四部分组成。塔基为高大的汉白玉须弥座，束腰处雕有精美的纹饰；塔身呈上圆下方的覆钵形，四面设佛龛，南面的"眼光门"内雕时轮金刚像，象征佛法之眼遍照十方；其上为十三层相轮（俗称"十三天"），层层收分；最顶为鎏金天盘、地盘与日月火焰宝顶，象征佛教的宇宙观。

### 琼华岛诸景

- **永安寺**：白塔脚下的藏传佛教寺院，依山而建、殿宇三重。自南而北依次为**法轮殿、正觉殿、普安殿**，红墙黄瓦、层层递高，是通往白塔的主要朝圣轴线。
- **漪澜堂**：位于琼华岛北麓，是**乾隆皇帝读书静养**之处。仿镇江金山寺"寺裹山"格局而建，堂周环以双层长廊，凭栏可览太液池全景。
- **琼岛春阴碑**：立于琼华岛东麓，碑身镌刻乾隆御笔"**琼岛春阴**"四字，是"**燕京八景**"之一。
- **阅古楼**：位于琼华岛西北，平面呈半月形的两层楼阁，楼内四壁嵌满《三希堂法帖》石刻，收藏自魏晋至明代的书法名迹数百件。

## 历史故事

### 白塔为何而建：两种传说

关于白塔的兴建，历来有**两说**。其一为**镇邪压龙**之说：相传北京地下有"海眼"，常有水患之虞，建塔以镇之，取"镇压龙脉、安定都城"之意。其二为**顺治向佛**之说：清世祖顺治帝笃信藏传佛教，曾召西藏高僧入京，为供奉舍利、弘扬佛法，遂于广寒殿旧址上兴建这座藏式喇嘛塔。两说并行不悖，一为民间附会，一为正史所载。

### 广寒殿的前世：从仙山到佛国

白塔并非凭空而起，它的脚下曾是金代著名的**广寒殿**——一座模仿海上仙山、供帝王游宴的华美宫殿。元代时广寒殿仍是皇家宴饮之所，相传忽必烈曾在此大宴群臣，殿中还陈列着那只后来流落民间的渎山大玉海（见[团城](/landmarks/beihai/tuancheng)）。至明代广寒殿倾圮，清顺治年间改为白塔，琼华岛的主题也由此从道家的"蓬莱仙岛"转变为佛家的"清净梵境"。

## 必看亮点

1. **登塔俯瞰**：琼华岛是全园制高点，南望故宫角楼与中南海，北眺景山万春亭，红墙金瓦尽收眼底。
2. **鎏金宝顶**：仰观塔顶十三天之上的鎏金日月宝顶，在阳光下金光灿灿，是整座白塔最庄严的部分。
3. **永安寺三重殿**：沿法轮殿、正觉殿、普安殿拾级而上，感受依山筑寺的空间层次与朝圣氛围。
4. **琼岛春阴碑**：寻觅燕京八景之一的御碑，体会"春阴"二字的诗意。
5. **漪澜堂凭栏**：在乾隆读书处凭栏北望，太液池水天一色，双层长廊环抱，是岛上最宜观景之所。
6. **阅古楼法帖**：若时间充裕，可入阅古楼细览《三希堂法帖》石刻，品味千年书法墨韵。

## 拍照机位

1. **南岸回望**：从南岸隔太液池拍白塔全景，碧水白塔倒影相映，是最经典的一幅。
2. **漪澜堂长廊**：双层长廊的廊柱与漏窗是现成的框景，可拍出层次丰富的白塔侧影。
3. **塔下仰拍**：白塔脚下仰拍鎏金宝顶与蓝天，庄严挺拔，尽显喇嘛塔的垂直之美。

## 参观贴士

1. **购联票登岛**：琼华岛（含白塔、永安寺）需购联票（20元含琼岛、团城），单买门票10元不含登岛。
2. **顺时针环岛**：建议从南岸过永安桥上山，依次游永安寺、白塔，再下山至漪澜堂、琼岛春阴碑、阅古楼，不走回头路。
3. **下午登塔**：下午阳光西斜，从岛上回望故宫、景山的逆光与侧光最适合拍照。
4. **联游北岸**：下山后可沿湖游[九龙壁](/landmarks/beihai/jiulongbi)、[五龙亭](/landmarks/beihai/wulongting)、[西天梵境](/landmarks/beihai/xitianfanjing)、[静心斋](/landmarks/beihai/jingxinzhai)。
5. **联游团城**：琼华岛南侧过永安桥即达[团城](/landmarks/beihai/tuancheng)，可顺路一游白玉佛与渎山大玉海。
6. **留意体力**：琼华岛台阶较多，登塔往返约需半小时，穿舒适的鞋。

## 常见问题

**Q：白塔可以进去登临吗？**

白塔本身为宗教圣物，通常不入内攀登；但琼华岛是全园制高点，登山即可俯瞰故宫、景山、中南海的开阔全景，视野极佳。

**Q：为什么白塔是白色的喇嘛塔，而不是传统楼阁式塔？**

白塔建于清顺治年间，顺治帝笃信藏传佛教，故采用藏式覆钵喇嘛塔的形制，与内地常见的楼阁式塔风格迥异，是汉藏文化交融的见证。

**Q：琼岛春阴碑的"春阴"是什么意思？**

"春阴"描绘的是春日琼岛林木葱茏、烟岚如黛的清幽意境，是乾隆钦定的"燕京八景"之一，碑为乾隆御笔亲题。
