# 郭沫若故居

![郭沫若故居](https://images.unsplash.com/photo-voV-byLj7qg?w=1200)

> "诗人、学者、革命家，一代文豪的京城岁月。"

## 简介

郭沫若故居位于北京市西城区前海西街18号，是一座典型的北京四合院。郭沫若先生从1963年至1978年在此居住，度过了人生的最后15年。故居现已成为"郭沫若纪念馆"，展示着这位文学巨匠的生平事迹和珍贵文物。

## 历史沿革

### 清代府邸

郭沫若故居原为清代恭王府的马号，后成为一座普通的四合院。1963年，郭沫若先生迁入此院。

### 郭沫若晚年

1963年至1978年，郭沫若先生在此居住，完成了大量重要的学术著作和文学作品。

#### 晚年成就

| 年份 | 成就 |
|:---|:---|
| 1963年 | 迁入前海西街18号 |
| 1965年 | 创作《武则天》 |
| 1971年 | 创作《李白与杜甫》 |
| 1978年 | 逝世于此 |

## 故居布局

### 四合院结构

郭沫若故居是一座典型的北京四合院，分为外院和内院。

#### 建筑布局

| 建筑 | 功能 | 特色 |
|:---|:---|:---|
| **大门** | 入口 | 传统的朱漆大门 |
| **影壁** | 照壁 | 精美的砖雕影壁 |
| **正房** | 郭沫若居室 | 保留原貌 |
| **东厢房** | 书房 | 郭沫若生前书房 |
| **西厢房** | 展览室 | 生平事迹展 |
| **后院** | 花园 | 郭沫若亲自设计 |

### 主要展室

#### 正房展室

正房是郭沫若先生的居室，保留了他生前的生活场景。

- **卧室**：简朴的卧室，摆放着简朴的家具
- **会客室**：会客的地方，摆放着沙发和茶几
- **工作室**：郭沫若生前工作的地方

#### 西厢房展室

西厢房是郭沫若生平事迹展，展示了郭沫若先生一生的重要经历。

#### 东厢房展室

东厢房是郭沫若的藏书室，收藏了大量珍贵书籍。

## 游览攻略

### 推荐路线

**精华路线（约1小时）**：

入口 → 影壁 → 正房（郭沫若居室） → 东厢房（藏书室） → 西厢房（生平展） → 后院（花园） → 出口

### 实用贴士

1. **提前预约**：建议提前在官网或电话预约
2. **最佳时间**：工作日人较少，适合慢慢参观
3. **拍照位置**：影壁、正房前、后院都是绝佳拍照点

5. **周边游览**：参观完后可以到什刹海、恭王府游览

## 文化意义

郭沫若是中国现代文学史上的重要人物，他的诗歌、戏剧、史学著作在中国文学史上占有重要地位。郭沫若故居不仅是一座名人故居，更是研究中国现代文学史的重要场所。

"诗人、学者、革命家"，这三个身份完美地概括了郭沫若先生的一生。
