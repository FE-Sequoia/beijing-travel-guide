# 九龙壁

![九龙壁](https://images.unsplash.com/photo-1767134915079-b3cc2b5daf30?w=1200&q=80)

> 一壁琉璃，九龙腾跃；正反面各有九条龙——这是中国唯一一座双面九龙壁，也是三百年前最华丽的"镇邪之眼"。

## 简介

**九龙壁**位于北海北岸**澄观堂前**，是一座建于**清乾隆二十一年（1756年）**的巨型琉璃影壁（照壁）。壁身以七色琉璃拼砌而成，长**25.5米**、高**6.9米**、厚**1.42米**，体量雄浑、色彩斑斓，是北海北岸最具视觉冲击力的景观。

九龙壁与中国古建筑中的"照壁"一脉相承——设于殿宇之前，用以遮挡视线、聚拢气运、趋吉避凶。北海这座九龙壁原是大圆镜智宝殿山门前的影壁，寺院虽已不存，影壁却巍然独峙，成为琉璃艺术的绝世孤品。

它与**故宫九龙壁**、**山西大同九龙壁**并称为"**中国现存三大九龙壁**"，是其中唯一一座**双面皆有九龙**的作品，堪称三绝之冠。这得益于琉璃材质防水耐晒的特性——正是这一特性，让它比许多木构殿宇更加长寿，历经近三百年风雨依然色彩如新。

## 建筑与景观

| 项目 | 数据 |

| **建成年代** | 清乾隆二十一年（1756年） |
| **长度** | 25.5米 |
| **高度** | 6.9米 |
| **厚度** | 1.42米 |
| **材质** | 七色琉璃构件拼砌（黄、蓝、白、紫、绿、赭、靛） |
| **结构** | 庑殿式壁顶、琉璃斗拱、须弥座壁身 |
| **龙数** | 正、背两面各有九条巨龙，合"九五之尊"之数 |

九龙壁通体由预制琉璃构件拼合而成，壁顶为庑殿式，覆琉璃瓦、设斗拱，仿木构屋顶形制；壁身置于高大的汉白玉须弥座之上，九条巨龙以高浮雕手法塑于壁面，腾挪于云涛海雾之间，姿态无一雷同。最可贵的是**双面九龙**——正面与背面各雕九条龙，两面皆气势磅礴，这在现存三大九龙壁中独一无二。

若仔细端详，会发现九龙的排列极有章法：正中一条龙为正龙，身居壁心、正面昂首，象征皇帝居中驭极；其余八龙分列两侧，两两相对，升龙降龙交替，构成四对对称，共成"九龙"之数。"九"与"五"皆为中国传统文化中的极数与中数，九龙壁暗合"九五之尊"，处处体现皇权的至高无上。

## 历史故事

### 楠木补龙：瞒天过海的匠人急智

北海九龙壁流传最广的传说，是关于**一条龙的腹部构件**。相传烧制琉璃构件时，一块白色龙腹构件在出窑时不慎破损——琉璃构件一旦烧坏便无法补救，重烧又赶不上工期，延误圣工是杀头之罪。情急之下，一位老工匠想出绝招：以一块**楠木**依原样雕刻成龙腹形状，再髹漆彩绘成与琉璃完全一致的色泽，拼接回壁面，竟瞒过了验收的眼睛。今天游客若贴近细看，据说其中一条白龙的腹部颜色与质感略异于周围的琉璃——那或许就是当年那块"楠木补丁"的遗迹。传说未必全真，却为冰冷的琉璃壁添上了一分匠人的机智与温度。

### 三大九龙壁：从大同到北海再到紫禁城

中国现存三大九龙壁各具渊源：**山西大同九龙壁**（明洪武二十五年，1392年，为明太祖第十三子代王府所建，是中国现存最早、最大的九龙壁）、**北海九龙壁**（1756年，双面九龙，体量居中）、**故宫九龙壁**（1772年，单面九龙，工艺最精）。三座壁分属王府、皇家园林、皇宫三个等级，恰好构成一部"九龙壁等级史"。其中唯北海这座双面雕龙，无论从哪个方向看去，九龙皆腾跃欲出，是三者中最特别的一座。

## 必看亮点

1. **双面九龙**：这是中国三大九龙壁中唯一双面皆有九龙的一座，正反两面都值得驻足细看，别处难寻。
2. **七色琉璃**：黄、蓝、白、紫、绿诸色交错，在阳光下的层次感是清代琉璃工艺的巅峰展示。
3. **"楠木补丁"**：寻找那条白龙腹部的异色构件，体会"楠木补龙"传说里的匠人急智。
4. **九龙姿态**：逐一辨认正龙、升龙、降龙的不同姿态，无一雷同，是高浮雕琉璃的教科书。
5. **须弥座与壁顶**：仰望庑殿式壁顶的琉璃斗拱，俯看须弥座的雕饰，体会"以壁仿殿"的建筑匠心。
6. **海水江崖纹**：龙身下方的波涛纹饰雕工极细，云水翻涌的动感是整壁最见功力的部分。

## 拍照机位

1. **正面全景**：壁长逾25米，需退至较远处方能摄入全貌，宜用广角。
2. **局部特写**：龙头、龙爪、云纹、海水江崖的特写，比全景更能体现琉璃工艺之精。
3. **侧视透视**：走到壁的两端侧视，九龙的高浮雕深度与透视效果更震撼。

## 参观贴士

1. **上午光线最佳**：九龙壁主体朝南，上午阳光正面照射时色彩最鲜亮、浮雕立体感最强。
2. **两面都要看**：许多游客只看正面，别忘了绕到背面——这里同样有九条腾飞的巨龙，是此壁最独特之处。
3. **走近看细节**：全景固然壮观，但龙头、云纹、海水江崖的细部，才是琉璃工艺真正的精彩所在。
4. **联游北岸**：九龙壁与[五龙亭](/landmarks/beihai/wulongting)、[西天梵境](/landmarks/beihai/xitianfanjing)、[静心斋](/landmarks/beihai/jingxinzhai)同在北海北岸，可沿湖串联游览。
5. **对比故宫九龙壁**：若行程包含故宫，可将此壁与[故宫九龙壁](/landmarks/forbidden-city/jiulong-bi)对照观看——单面与双面、皇宫与园林，趣味倍增。

## 常见问题

**Q：北海九龙壁和故宫九龙壁有什么区别？**

最核心的区别是**单面与双面**：北海九龙壁正反两面各有九龙，是中国唯一的双面九龙壁；故宫九龙壁只有单面九龙，但工艺更精巧细腻。两座壁一在皇家园林、一在皇宫内廷，可对照欣赏。

**Q："楠木补龙"是真的吗？**

这是流传甚广的民间传说，史无确证。今天游客据传仍能在某条白龙腹部找到与琉璃质感略异的"补丁"，真假已不可考，但它为九龙壁增添了浓厚的人情味与故事性。

**Q：为什么寺院没了，九龙壁还在？**

九龙壁原是大圆镜智宝殿山门前的影壁。清末寺院毁于火灾，木构殿宇荡然无存，而琉璃材质的影壁耐火耐久，得以独存至今，反成了这片旧址上最醒目的见证。
