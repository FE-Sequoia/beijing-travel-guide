# 正阳门（箭楼）

![正阳门](https://images.unsplash.com/photo-3bqa0rGhVSA?w=1200)

> "国门第一楼，守望京城六百年。"

## 简介

正阳门，俗称前门，位于北京市中轴线上，是明清北京内城的正门。正阳门由城楼和箭楼两部分组成，是北京城内唯一一座城楼和箭楼都完整保存下来的城门。正阳门是北京城的标志性建筑之一，也是中轴线上的重要节点。

## 历史沿革

### 明代始建

正阳门始建于明永乐十八年（1420年），是明清北京内城的正门。正阳门的名称取自"圣主当阳，日至中天"，寓意皇帝的统治如日中天。

### 清代重修

清代对正阳门进行了多次重修，保持了明代建筑的原貌。正阳门一直是北京城的正门，皇帝出巡、重要仪式等都要经过正阳门。

### 近代变迁

民国时期，正阳门城楼和箭楼都曾遭受破坏。1980年代以来，正阳门进行了大规模的修复，恢复了原有的建筑风貌。

#### 重要事件

| 时间 | 事件 |
|:---|:---|
| 1420年 | 正阳门始建 |
| 1644年 | 李自成攻入北京，正阳门受损 |
| 1900年 | 八国联军入侵，正阳门被毁 |
| 1915年 | 正阳门城楼被拆除 |
| 1978年 | 正阳门箭楼修复 |
| 1988年 | 正阳门城楼复建 |

## 建筑特色

### 城楼

正阳门城楼是北京城内最高大的城门楼，建筑宏伟壮观。

#### 建筑参数

| 项目 | 数据 |
|:---|:---|
| **高度** | 约42米 |
| **面阔** | 七间 |
| **进深** | 三间 |
| **屋顶形式** | 重檐歇山顶 |
| **瓦色** | 灰筒瓦绿剪边 |

#### 建筑特色

- **高度**：正阳门城楼是北京城内最高大的城门楼
- **结构**：城楼为重檐歇山顶，气势恢宏
- **功能**：城楼用于瞭望和防御

### 箭楼

正阳门箭楼位于城楼前方，是防御性建筑。

#### 建筑参数

| 项目 | 数据 |
|:---|:---|
| **高度** | 约38米 |
| **面阔** | 七间 |
| **进深** | 三间 |
| **屋顶形式** | 重檐歇山顶 |

#### 建筑特色

- **防御功能**：箭楼四面有箭窗，用于射击
- **建筑形式**：重檐歇山顶，与城楼相呼应
- **文化内涵**：箭楼是城门防御体系的重要组成部分

### 瓮城

正阳门原有瓮城，是城门防御体系的重要组成部分。瓮城在民国时期被拆除，现仅存遗址。

## 主要看点

### 城楼展览

正阳门城楼内有丰富的展览，展示了正阳门的历史和文化。

#### 展览内容

- **历史沿革**：正阳门的建筑历史
- **中轴线文化**：北京中轴线的历史和文化
- **城门文化**：中国古代城门建筑的特色

### 箭楼展览

正阳门箭楼内有关于北京城门防御体系的展览。

#### 展览内容

- **防御体系**：北京城门的防御体系
- **军事文化**：中国古代军事文化
- **建筑艺术**：城门建筑的艺术特色

## 文化意义

正阳门是北京城的象征，也是中轴线上的重要节点。"正阳"二字寓意着光明正大、蒸蒸日上，体现了中国古代的文化理念。

正阳门不仅是一座城门，更是北京历史文化的见证。从明清到近代，正阳门见证了北京城的沧桑巨变，也承载了北京人的记忆和情感。
