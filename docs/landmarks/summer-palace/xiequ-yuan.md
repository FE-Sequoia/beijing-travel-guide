# 谐趣园

![谐趣园](https://images.unsplash.com/photo-1762663983944-9acb89c68650?w=1200&q=80)

> "通常拟惠山，肖之为惠山园。"
> —— 乾隆御诗道出这座"园中之园"的身世

## 简介

谐趣园位于颐和园东北角、后湖东端，是一座藏于大园之中的小园，素有"园中之园"的美誉。它仿无锡名园寄畅园而建，小巧精致，自成天地：一池碧水居中，亭台楼阁环池而筑，游廊曲折相接，假山点缀其间，虽处北方皇家园林之中，却尽显江南私家园林的婉约气韵。

从万寿山的巍峨排场走进谐趣园的曲径回廊，恍如从庙堂退入山林，呼吸都为之一缓。

谐趣园始建于乾隆十六年（1751年）。乾隆首次南巡时驻跸无锡寄畅园，对其"山水清音"一见倾心，回京后便命画师携图仿建，初名"惠山园"，取"惠山"之音。嘉庆十六年（1811年）重修后更名"谐趣园"，寓"以物外之静趣，谐寸田之中和"之意——游目骋怀，心与物谐，正是这座小园的精神内核。1860年谐趣园同样毁于英法联军大火，现存建筑为光绪年间依原样重建。

## 建筑与景观

| 项目 | 内容 |
|:---|:---|
| **格局** | 以水池为中心，环池布置亭台楼阁、游廊假山 |
| **涵远堂** | 园中正殿，面阔五间，帝后游园休憩之所 |
| **知鱼桥** | 贴水低桥，典出《庄子》"濠梁之辩" |
| **瞩新楼** | 两层小楼，登楼可俯瞰全园 |
| **澄爽斋** | 临水而筑，推窗即见荷风 |
| **其他** | 引镜、洗秋、饮绿诸亭榭，环池游廊嵌书法石刻 |

谐趣园以一方水池为核心，四周建筑皆面水而立：北岸涵远堂为园中正殿，面阔五间；西岸瞩新楼两层，登楼可俯瞰全园；南岸澄爽斋临水而筑，推窗即见荷风送香。环池游廊将诸建筑连为一体，廊壁嵌有历代书法石刻。

最妙的是那座贴水而建的知鱼桥——桥身极低，几乎与水面相平，人行桥上如在鱼背，俯身即见游鱼往来。

## 历史故事

### 知鱼桥上的"濠梁之辩"

谐趣园最出名的小景是知鱼桥。桥名典出《庄子·秋水》：庄子与惠子游于濠梁之上，庄子曰："鲦鱼出游从容，是鱼之乐也。"惠子反问："子非鱼，安知鱼之乐？"庄子答："子非我，安知我不知鱼之乐？"

乾隆对此典故极为偏爱，不仅将桥命名为"知鱼"，还在桥头题诗："鱼跃鸢飞同活泼，水流山静共悠然。"如今桥栏低矮，游鱼不惧人，春夏之交柳丝拂水、鱼群唼喋，站在桥上发呆十分钟，便能明白这座园子为何叫"谐趣"——不必真的知道鱼乐不乐，此刻你自己是快乐的，就够了。

### 乾隆的"园林搬运术"

乾隆一生六下江南，每到一处名园必令随行画师绘图摹样，回京后择地仿建：仿海宁安澜园、仿苏州狮子林、仿南京瞻园，谐趣园仿的则是无锡寄畅园。这种"移天缩地"的做法，让三山五园集江南名园之大成。

有趣的是，寄畅园借景锡山龙光塔，谐趣园则借景万寿山余脉——仿的是格局，变的是气韵，北方山水给了这座"无锡分园"一副更开阔的骨架。

### 一园之内，四种造园法

谐趣园虽小，却是中国造园技法的微缩教材：**障景**——入园先以假山、照壁挡眼，逼你转过弯才见全池；**框景**——游廊花窗把池柳亭台裁成一幅幅册页；**对景**——涵远堂与瞩新楼隔水相望，互为观赏对象；**借景**——抬头处万寿山余脉入园作背景。300米方圆之内走完这四种手法，等于上了一堂园林课。

## 必看亮点

1. **知鱼桥**：贴水低桥，俯身观鱼，体会庄子"濠梁之辩"的哲学趣味，是全园最不容错过的一景。
2. **涵远堂**：园中正殿，殿前平台临水，是观赏全园倒影的最佳位置，雨日听荷尤佳。
3. **环池游廊**：廊壁书法石刻与框景窗格交替出现，一步一景，尽显江南园林的曲折含蓄。
4. **瞩新楼**：两层小楼为全园制高点，登楼俯瞰，水池、亭台、假山尽收一池之小。
5. **四季小景**：春看柳、夏赏荷、秋听雨、冬观雪——小园的好处在于，任何时候来都有独属此刻的意境。

## 拍照机位

1. **知鱼桥俯拍**：低机位贴水面拍游鱼与桥影，春夏柳丝入镜最佳。
2. **涵远堂前平台**：隔水拍对岸瞩新楼与全园倒影，无风时如镜面。
3. **花窗框景**：游廊漏窗框出的一角池景，是现成的"宋画"构图。

## 参观贴士

1. 谐趣园免费开放（含在大门票内），无需另购票，是颐和园中性价比极高的清净去处。
2. 位置偏居东北角，多数旅行团不涉足，想避开人潮的独行客首选此地，尤其清晨和黄昏。
3. 建议从后湖东端进入，与[苏州街](/landmarks/summer-palace/suzhou-street)串联游览，后山一线清幽，与前山喧闹判若两园。
4. 若对无锡寄畅园感兴趣，不妨将两处"对照游"：同一种造园语言，在江南与京畿各自生长出不同的气质。
5. 园内游廊低矮、石径湿滑，雨后参观注意脚下；带孩子的游客在知鱼桥观鱼时务必看护好孩童。
6. 逛完谐趣园从园东小门出，即是东宫门方向，可接续[仁寿殿](/landmarks/summer-palace/renshou-dian)的行程。

## 常见问题

**Q：谐趣园需要买票吗？**

不需要。谐趣园包含在颐和园大门票内，免费进入，是全园性价比最高的景点之一。

**Q：谐趣园和无锡寄畅园哪个更值得看？**

各擅胜场：寄畅园是江南原生的私家园林，山水更苍古；谐趣园是皇家手笔的"北方版"，尺度更疏朗。时间允许，两处对照着看最有意思。

**Q：谐趣园为什么人少？**

位置偏在颐和园东北角，旅行团的大巴从北宫门、东宫门进出，很少绕到这里——这正是它最大的优点。

**Q：谐趣园什么时候去最美？**

夏看荷、冬看雪；行家里手更推荐雨天的涵远堂——隔水听雨打荷叶，才是这座园子名字里那个"趣"字。
