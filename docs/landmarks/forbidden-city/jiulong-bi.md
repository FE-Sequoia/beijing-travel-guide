# 九龙壁

![九龙壁](https://images.unsplash.com/photo-1746258537565-85e3b780cd18?w=1200&q=80)

> 九条巨龙腾跃于琉璃云海之间——而其中一条龙，竟是木头做的。这是紫禁城最华丽的障眼法。

## 简介

九龙壁位于故宫东北部**宁寿宫区的皇极门外**，是一座建于清乾隆三十七年（1772年）的**琉璃影壁（照壁）**。它背靠皇极门、面向宁寿宫广场，是进入宁寿宫区（今珍宝馆）的第一道视觉屏障——按中国传统建筑礼制，照壁设于大门之内或之外，用以遮挡视线、聚拢气运，九龙壁正是这一功能的最高规格体现。

壁身以**彩色琉璃**拼砌而成，长29.4米、高3.5米，九条巨龙在云涛海雾中腾挪翻涌，姿态无一雷同。它与**北京北海公园九龙壁**、**山西大同九龙壁**并称"中国三大九龙壁"，是故宫内唯一一座九龙壁，也是三座中最精巧细腻的一座。

## 建筑与景观

| 项目 | 数据 |
|:---|:---|
| **位置** | 宁寿宫区皇极门外（珍宝馆入口附近） |
| **建成年代** | 清乾隆三十七年（1772年） |
| **长度** | 29.4米 |
| **高度** | 3.5米（含须弥座与壁顶） |
| **材质** | 彩色琉璃构件拼砌（黄、蓝、白、紫、绿诸色） |
| **结构** | 庑殿式壁顶、琉璃斗拱、须弥座壁身 |
| **龙数** | 正面九条巨龙，加壁顶正脊行龙，共合"九五之尊"之数 |

九龙壁通体由预制琉璃构件拼合而成，壁顶为庑殿式，覆琉璃瓦、设斗拱，仿木构建筑屋顶的形制；壁身置于汉白玉须弥座之上，九条龙以高浮雕手法塑于壁面，龙身起伏、鳞甲毕现，衬以翻卷的云纹与澎湃的海水江崖纹。色彩以黄、蓝为主调，间杂白、紫、绿诸色，在日光下流光溢彩。

九龙的布局暗合"九五之尊"：正面九条龙居中者（第五条）为正龙，金黄盘龙、正面昂首，象征皇帝居中驭极；两侧八龙两两相对，升龙降龙交替，构成四对对称。若加上壁顶正脊上的行龙，龙的总数又合"九五"之数——数字的讲究无处不在。

## 历史故事

### 楠木"假龙"：瞒过乾隆的障眼法

九龙壁流传最广的传说，是其中**一条龙的腹部构件是楠木仿制的**。相传乾隆年间烧制九龙壁琉璃构件时，一块龙腹构件在出窑时不慎烧裂——琉璃构件一旦烧坏便无法补救，而重烧的工期已赶不上工期，欺君之罪是要杀头的。情急之下，一位老工匠想出绝招：选用上等楠木，依原样雕刻成龙腹构件的形状，再用油漆彩画成与琉璃完全一致的色泽纹理，拼回壁面。乾隆帝验收时，九条龙栩栩如生，竟未能察觉其中一块是"木龙"。直到多年后一次暴雨冲刷，油漆剥落，楠木本色才露出破绽——这则传说未必全真，但今天游客若贴近细看九龙壁东起第三条龙的腹部，据传仍可辨认出那块与琉璃质感略有不同的"补丁"。真假已不可考，但这个"匠人急智"的故事，让冰冷的琉璃壁多了一分人情温度。

### 三大九龙壁：从大同到北海再到紫禁城

中国现存的九龙壁以三座最为著名：**山西大同九龙壁**（明洪武二十五年，1392年，为明太祖第十三子代王府所建，是中国现存最早、最大的九龙壁）、**北京北海公园九龙壁**（清乾隆二十一年，1756年，双面九龙，两面各有九条龙）、**故宫九龙壁**（1772年，单面九龙，工艺最精）。三座九龙壁分属王府、皇家园林、皇宫三个层级，恰好构成一部"九龙壁等级史"——故宫这座虽最小、最晚，却因工艺之精、位置之尊而居三者之冠。

## 必看亮点

- **九条龙的姿态差异**：逐一辨认九条龙——居中正龙威严、两侧升龙降龙各异，无一雷同，是高浮雕琉璃工艺的教科书。
- **"木龙"传说处**：寻找东起第三条白龙腹部，传说那块楠木补丁就藏在此处，贴近细看别有意味。
- **九五之数的暗喻**：数一数正面九龙（九）与壁顶行龙，体会"九五之尊"在数字上的处处呼应。
- **海水江崖纹**：龙身下方的波涛纹饰雕工极细，云水翻涌的动感是整壁最见功力的部分。
- **琉璃色彩**：黄蓝紫白绿五色琉璃在阳光下的层次感，是清代官窑琉璃烧制技术的集中展示。

## 参观贴士

- **入珍宝馆必经**：九龙壁就在皇极门外，进入宁寿宫区（珍宝馆）时必然路过，不必专程寻找，但值得驻足5-10分钟细看。
- **上午光线最佳**：九龙壁坐南朝北，上午阳光斜照壁面时色彩最鲜亮、浮雕立体感最强；正午顶光下反而显得平淡。
- **别只看中间**：游客多聚集在壁前正中拍全景，走到两端侧视，龙的浮雕深度和透视效果更震撼。
- **联想三大九龙壁**：若行程中有北海公园，可对比观看北海双面九龙壁——单面与双面、王府与皇宫，两座壁互相对照，趣味倍增。
- **拍照提示**：壁长近30米，正面全景需退至广场南端；局部特写（龙头、龙爪、云纹）比全景更能体现琉璃工艺的精细。
