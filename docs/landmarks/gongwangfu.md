# 恭王府

![恭王府](https://images.unsplash.com/photo-BGZj9K41kOo?w=1200)

> "一座恭王府，半部清代史。"

## 简介

恭王府位于北京市西城区前海西街17号，是清代规模最大的一座王府，也是现存最完整的清代王府建筑群。恭王府始建于清乾隆年间，最初为和珅的宅邸，后成为恭亲王奕訢的府邸，见证了大清王朝从鼎盛走向衰落的历史进程。

## 历史沿革

### 和珅宅邸时期（1777-1799）

清乾隆四十一年（1776年），和珅开始在此营建宅邸。和珅是乾隆皇帝的宠臣，权倾朝野，其宅邸的奢华程度仅次于皇宫。

#### 建筑规模

- **占地面积**：约6万平方米
- **建筑面积**：约3万平方米
- **房屋数量**：约1000间
- **花园面积**：约2.8万平方米

### 庆王府时期（1799-1851）

嘉庆四年（1799年），和珅被赐死，家产被抄，宅邸被赐给庆郡王永璘。

### 恭王府时期（1851-1912）

咸丰元年（1851年），咸丰帝将这座宅邸赐给恭亲王奕訢，从此称为"恭王府"。恭亲王奕是道光帝第六子，洋务运动的主要领导者之一。

#### 恭亲王奕訢

奕訢是晚清政坛的核心人物，曾任总理衙门大臣、军机大臣等要职，主导了洋务运动。

### 近代变迁（1912-至今）

民国时期，恭王府先后成为辅仁大学女部、中国艺术研究院等机构的所在地。1982年被列为全国重点文物保护单位，2008年全面对外开放。

## 建筑布局

恭王府由府邸和花园两部分组成，整体布局呈"前府后园"的格局。

### 府邸部分

#### 银安殿

银安殿是恭王府的正殿，也是王府中最重要的建筑。殿内原设有亲王宝座，是举行重大礼仪活动的场所。

- **建筑形制**：重檐歇山顶
- **面阔**：五间
- **功能**：举行礼仪、接见宾客

#### 多福轩

多福轩是恭亲王日常起居和处理政务的地方。轩内悬挂着"多福"匾额，寓意多福多寿。

#### 乐道堂

乐道堂是恭亲王的书房，也是他处理政务、读书写字的地方。堂内收藏有大量珍贵文物。

### 花园部分

恭王府花园又名"萃锦园"，是王府的精华所在。

#### 西洋门

西洋门是花园的正门，采用汉白玉石拱门，具有典型的西洋建筑风格，是恭王府花园的标志性建筑。

#### 大戏楼

恭王府大戏楼是北京现存最完整的王府戏楼，可容纳200余人观戏。戏楼建筑精美，音响效果极佳。

#### 福字碑

福字碑是康熙皇帝的御笔，被誉为"天下第一福"。这个"福"字包含了"子、才、田、福、寿"五个字，寓意五福临门。

#### 滴翠岩

滴翠岩是花园中的假山，由太湖石堆砌而成。假山内有秘云洞，洞内有福字碑。

## 游览攻略

### 推荐路线

正门 → 银安殿 → 多福轩 → 乐道堂 → 西洋门 → 大戏楼 → 滴翠岩（福字碑） → 方塘水榭 → 邀月台 → 出口

正门 → 银安殿 → 多福轩 → 乐道堂 → 锡晋斋 → 后罩楼 → 西洋门 → 大戏楼 → 滴翠岩（福字碑） → 方塘水榭 → 邀月台 → 牡丹园 → 出口

### 实用贴士

1. **提前预约**：恭王府实行预约制，建议提前在官网或微信公众号预约
2. **最佳时间**：工作日人流较少，上午9点前入园最佳
3. **福字碑**：排队人很多，建议一入园先去排福字碑
4. **拍照位置**：西洋门、大戏楼前、方塘水榭都是绝佳拍照点
5. **讲解服务**：建议租用讲解器或请导游，了解更多历史故事

## 文化意义

恭王府是清代王府建筑的典范，也是研究清代政治、文化、建筑的重要实物资料。从和珅的奢靡到恭亲王的洋务，这座王府见证了大清王朝最后的风云变幻。

"一座恭王府，半部清代史"，这句话形象地概括了恭王府在中国历史上的重要地位。
