# 首都博物馆

> "了解北京历史文化的最佳窗口。"

## 简介

首都博物馆位于北京市西城区，是国家一级博物馆，也是北京市属的综合性博物馆。首都博物馆以北京历史、文化、艺术为主要展示内容，是了解北京历史文化的最佳窗口。

## 历史沿革

### 创建

首都博物馆创建于1981年，原址位于北京市东城区国子监街孔庙内。

### 新馆建设

2001年，首都博物馆新馆立项。2006年5月，新馆正式对外开放。新馆由著名建筑师设计，建筑面积6.3万平方米，是一座现代化的大型博物馆。

## 建筑特色

### 外观设计

首都博物馆新馆外观设计独特，融合了传统与现代元素：

- **青铜展馆**：椭圆形的青铜展馆，象征古代青铜器
- **玉石展馆**：矩形的玉石展馆，象征古代玉器
- **中庭**：两个展馆之间的玻璃中庭，象征历史与现代的对话

### 内部空间

博物馆内部空间宽敞明亮，展厅众多。

## 常设展览

### 北京历史文化展

这是首都博物馆的基本陈列，全面展示北京的历史文化。

#### 展览内容

| 展厅 | 内容 |
|:---|:---|
| 燕地青铜 | 北京出土的青铜器 |
| 玉器精品 | 北京出土的玉器 |
| 书法绘画 | 北京名家的书画 |
| 老北京民俗 | 老北京的生活习俗 |
| 京剧文化 | 京剧的历史与现状 |

### 老北京民俗展

这个展览生动展示了老北京的民俗风情。

#### 展示内容

- **年节习俗**：春节、元宵节等传统节日的习俗
- **婚嫁习俗**：老北京的婚嫁礼仪
- **生育习俗**：生子、满月、抓周等习俗
- **寿诞习俗**：过生日的习俗
- **丧葬习俗**：丧葬礼仪
- **商业习俗**：老北京的商业活动
- **娱乐习俗**：戏曲、曲艺、杂耍等

### 古代佛像艺术精品展

展示中国古代佛像艺术精品，包括石雕、木雕、铜铸等。

### 燕地青铜艺术精品展

展示北京出土的青铜器精品，包括商周青铜器、春秋战国青铜器等。

## 特色展览

### 京剧文化展

北京是京剧的发源地，京剧文化展展示了京剧的历史、流派、名家等。

#### 展览内容

- **京剧起源**：京剧的形成与发展
- **京剧流派**：四大名旦、四大须生等
- **京剧脸谱**：各种脸谱的含义
- **京剧服饰**：京剧的服装与道具

### 老北京戏楼展

展示老北京戏楼的历史与文化。
