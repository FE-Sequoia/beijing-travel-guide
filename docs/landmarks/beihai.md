# 北海公园

![北海公园](https://images.unsplash.com/photo-1508804185872-d7badad00f7d?w=1200)

> "让我们荡起双桨，小船儿推开波浪，海面倒映着美丽的白塔，四周环绕着绿树红墙。"
> —— 歌曲《让我们荡起双桨》

## 简介

北海公园位于北京市中心区，是中国现存最古老、最完整的皇家园林之一。公园始建于辽代，历经金、元、明、清几个朝代的扩建，至今已有近千年的历史。

北海以琼华岛为中心，岛上建有著名的白塔，四周环绕着碧波荡漾的湖水和亭台楼阁，是中国古典园林艺术的杰作。1961年被国务院公布为第一批全国重点文物保护单位。

## 主要景点

### 琼华岛与白塔

琼华岛是北海公园的中心，岛上建有著名的永安寺白塔，是北海的标志。

![北海白塔](https://images.unsplash.com/photo-1590549016683-3775f8b36b08?w=800)

| 项目 | 内容 |
|:---|:---|
| **白塔高度** | 35.9米 |
| **建造年代** | 清顺治八年（1651年） |
| **建筑风格** | 藏式喇嘛塔 |

登临白塔可俯瞰全园，远眺故宫、景山、中南海，视野开阔，令人心旷神怡。

### 九龙壁

位于北海公园北岸，是中国现存三座九龙壁中最精美的一座。壁上九条大龙，形态各异，栩栩如生。

### 五龙亭

位于北岸，由五座造型各异的亭子组成，是皇帝垂钓和观赏焰火的地方。

### 团城

位于北海公园南门内，是一座圆形城台，台上建有承光殿，供奉白玉佛。著名的渎山大玉海就在此处。

## 文化故事

### "让我们荡起双桨"

1955年，电影《祖国的花朵》在北海取景，插曲《让我们荡起双桨》唱遍大江南北，歌词中的"海面倒映着美丽的白塔，四周环绕着绿树红墙"成为北海最经典的写照。

### 渎山大玉海的传奇

团城上的渎山大玉海是元代的巨型酒器，由一整块墨玉雕刻而成，重约3.5吨。传说当年忽必烈曾用它盛酒，宴请群臣。明末被当作腌菜缸使用，清代才被重新发现。

## 游览攻略

### 推荐路线

**经典半日游（2-3小时）**：
南门（团城）→ 琼华岛 → 白塔 → 永安寺 → 沿湖漫步 → 五龙亭 → 九龙壁 → 北门出

### 划船游湖

北海公园是北京著名的划船胜地：

| 船型 | 价格 | 适合 |
|:---|:---:|:---|
| 手划船 | 60元/小时 | 情侣、朋友 |
| 脚踏船 | 80元/小时 | 家庭 |
| 电动船 | 100元/小时 | 省力游览 |

### 最佳拍照点

1. **白塔前**：经典角度，绿树红墙白塔
2. **五龙亭**：拍北海全景
3. **琼华岛东侧**：拍湖面倒影
