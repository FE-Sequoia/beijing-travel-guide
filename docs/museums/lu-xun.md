# 鲁迅博物馆

![鲁迅博物馆](https://images.unsplash.com/photo-1714296160734-9d6331b77cd1?w=1200)

> "横眉冷对千夫指，俯首甘为孺子牛。"
> —— 鲁迅《自嘲》

## 简介

北京鲁迅博物馆位于西城区阜成门内西三条21号，是新中国最早建立的"人物传记性博物馆"。博物馆成立于1956年10月19日——鲁迅逝世20周年之际。

博物馆由鲁迅故居和鲁迅生平陈列馆两部分组成，完整展示了鲁迅在北京生活、工作的历史。

## 鲁迅与北京

### 在京岁月（1912-1926）

鲁迅在北京生活、工作了14年，这是他一生中最重要的创作时期之一。

| 时期 | 经历 |
|:---|:---|
| 1912年 | 随中华民国临时政府教育部迁往北京 |
| 1923年 | 购得阜成门内西三条21号房产 |
| 1924年 | 搬入新居，完成《野草》《彷徨》等作品 |
| 1926年 | 离开北京南下 |

### 故居岁月

1924年5月至1926年8月，鲁迅在这座小四合院中居住，完成了：

- 小说集《彷徨》
- 散文诗集《野草》
- 杂文集《华盖集》《华盖集续编》

## 故居布局

鲁迅故居是一座典型的北京四合院，由鲁迅亲自设计改建。

### "老虎尾巴"

故居最著名的建筑是北房后面接出的一间平顶灰棚，被鲁迅戏称为"老虎尾巴"。这是鲁迅的工作室兼卧室，《野草》大部分篇章在此完成。

### 庭院

院内种植着鲁迅亲手栽种的树木：丁香、黄刺玫、榆叶梅等，至今依然茂盛。

## 文化故事

### 从周树人到鲁迅

鲁迅原名周树人，"鲁迅"这个笔名取自母亲鲁瑞的姓，寓意"愚鲁而迅速"。

### 《新青年》与《狂人日记》

1918年，鲁迅以"鲁迅"为笔名发表了中国现代文学史上第一篇白话小说《狂人日记》，发出了"救救孩子"的时代呐喊。

### "民族魂"

鲁迅逝世后，覆盖在他棺木上的旗帜上写着"民族魂"三个大字，被誉为"中国现代文学之父"。
