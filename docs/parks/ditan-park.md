# 地坛公园

> "明清祭地之所，银杏大道金黄。"

## 简介

地坛公园位于北京市东城区，是明清两代皇帝祭祀土地神的地方。地坛始建于明嘉靖九年（1530年），与天坛相对应，天坛祭天，地坛祭地。地坛公园以银杏大道闻名，每年秋季银杏叶黄，吸引大量游客。

## 历史沿革

### 始建

明嘉靖九年（1530年），明世宗朱厚熜决定实行天地分祀，在北京北郊建地坛，专祀土地神。

### 清代修缮

清代，地坛多次修缮，规模不断扩大。乾隆年间，地坛达到鼎盛。

### 民国荒废

民国时期，地坛逐渐荒废，成为乱葬岗。

### 现代公园

1957年，地坛辟为公园，对外开放。1984年，地坛被列为北京市文物保护单位。

## 建筑特色

### 方泽坛

方泽坛是地坛的核心建筑，是明清皇帝祭祀土地神的地方。

#### 布局

- **坛台**：方形，两层，象征"地方"
- **坛墙**：方形，四面
- **泽门**：坛南的正门

#### 与天坛对比

| 对比项 | 天坛 | 地坛 |
|:---|:---|:---|
| **形状** | 圆形（天圆） | 方形（地方） |
| **颜色** | 蓝色 | 黄色 |
| **层数** | 三层 | 两层 |
| **祭祀对象** | 天 | 地 |

### 皇祇室

皇祇室位于方泽坛南，是供奉土地神牌位的地方。

### 斋宫

斋宫是皇帝祭祀前斋戒的地方。

### 银杏大道

地坛公园以银杏大道闻名，种植了大量银杏树。

#### 银杏树

地坛公园的银杏树有近200株，是北京市区最大的银杏林之一。

#### 最佳观赏时间

每年10月下旬至11月中旬，银杏叶黄，景色迷人。

## 游览攻略

### 推荐路线

**经典路线（约1-2小时）**：

南门 → 方泽坛 → 皇祇室 → 银杏大道 → 斋宫 → 北门

**秋季路线（看银杏）**：

西门 → 银杏大道 → 方泽坛 → 皇祇室 → 南门

| 方式 | 站点 |

### 实用贴士

1. **银杏最佳时间**：10月下旬至11月中旬
2. **周末人多**：周末人极多，建议工作日

4. **皇祇室另收**：皇祇室另收5元
5. **结合游览**：可与雍和宫、国子监一起游览
6. **拍照建议**：早上8:00前或下午4:00后光线最佳

## 文化价值

地坛是全国重点文物保护单位，是中国古代"天圆地方"宇宙观的重要体现。

地坛与天坛相对应，体现了中国古代"天圆地方""天南地北"的宇宙观念。

地坛的银杏大道是北京市区最美的秋景之一，每年吸引大量游客前来观赏。
