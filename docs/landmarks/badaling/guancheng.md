# 关城与城门

![关城与城门](https://images.unsplash.com/photo-1640608486292-f52ac5d7cd76?w=1200&q=80)

> 东题"居庸外镇"，西题"北门锁钥"。两道门额，道尽这座关城六百年的分量。

## 简介

八达岭关城坐落于关沟北口的山谷之间，是八达岭长城防御体系的**咽喉与中枢**。长城城墙以关城为核心，向南北两翼蜿蜒入山，形成"两翼展开、中扼谷口"的防御格局——敌若越岭而来，先要面对这道横亘谷中的坚城；而守军凭城而守，又可策应两翼城墙，进退有据。

明代，关城是**长城沿线驻军指挥所**所在地：守城官员在此坐镇，统辖两侧城墙、敌楼与城外烽堠，调度兵马、查验关防、传递军情。南来北往的商旅使节、茶马互市的人流，也都要经此门洞验放。一座关城，既是军事堡垒，也是关沟古道上最重要的关卡。

关城始建于明弘治十八年（1505年），此后在嘉靖、隆庆、万历年间随北疆防务的加强而屡经加固，城垣包砖、门楼增修，终成今日规模。1950年代，关城得到系统整修，成为八达岭长城面向世界的第一张面孔。

如今，关城是游客登临八达岭长城南北两线的**出发点**：自北门出即上北线直奔好汉坡，自南门出则入南线缓步览胜。登城之前，先读懂这两道门额上的字，这段城墙才算没白爬。

### 瓮城形制：城门里的"口袋阵"

关城东西设门、城楼对峙，并在城门一侧附设**瓮城**，构成典型的瓮城形制。入城者须先进入瓮城这方"口袋"，再经门洞验放方可通行；一旦敌骑突入瓮城而不得入内城，守军便可合闭瓮门，从四面城墙上居高临下、矢石俱下——所谓"瓮中捉鳖"，正是这种城制的战场写照。

瓮城与主城门之间还有一重巧思：内外门洞并不正对，入城路线被迫折转，使冲门而来的骑兵无法借直线加速，锋芒顿挫于方寸之间。冷兵器时代的城防智慧，尽在这几道门洞的错位之中。

关城虽小，五脏俱全：城内旧有官署、营房、仓廒与马号，驻军起居、军需周转皆在其中完成，是一座功能完备的微型军城。如今建筑虽已不存，但站在城中环顾四围高墙，仍能感到那种"城中之城"的围合与森严。

### 两道门额的含义

- **"居庸外镇"**：刻于东门之上，意为八达岭是居庸关的外围屏障与前哨重镇。居庸关号称天下第一雄关，而其险要的前哨防线正设在八达岭——"居庸之险不在关城，而在八达岭"，此额即这层战略关系的官方注脚。
- **"北门锁钥"**：刻于西门之上，典出《宋史·寇准传》。寇准坐镇大名府，辽使问"相公望重，何以不在中书？"寇准答："主上以朝廷无事，北门锁钥，非准不可。"后人遂以"北门锁钥"比喻镇守国门的要害之地。八达岭扼京师北大门，此额正是明代对其战略地位的盖棺之论。

## 历史故事

### 兵家必争的关沟咽喉

关沟是穿越太行山北端、连接华北平原与塞外高原的孔道，八达岭正卡在沟道最北、最险处。自战国至明清，这里战事频仍，从来是兵家必争之地。

明代中叶，蒙古骑兵多次叩关南下。嘉靖二十九年（1550年）"庚戌之变"后，明廷痛定思痛，大规模加固京师北部长城防线，八达岭关城与两翼城墙正是在这一背景下不断修葺、臻于完备——砖石包墙、敌楼增设、瓮城完固，皆为彼时防务升级的产物。

清代以后，长城军事意义渐退，关城转为商旅往来的税卡与驿路节点，"北门锁钥"从战场用语变成历史典故。一道门额含义的变迁，本身就是一部微缩的边疆史。

### 庚子年的最后一瞥

1900年，八国联军兵临北京，慈禧太后携光绪帝仓皇西逃。御辇出德胜门、过居庸关，正是经八达岭关城向西而去。相传銮驾出城之际，太后曾于关城附近登石回望京师（详见[望京石](/landmarks/badaling/wangjingshi)）。那道她穿过的门洞，至今仍在，门额上的字也仍是当年的字。

### 关城见证"长城外交"

1958年，八达岭长城正式对外开放，成为新中国接待外宾的首选长城景区。数百位外国元首与政府首脑先后登临此地，关城门楼与北线长城见证了无数次"长城外交"的高光时刻。对世界而言，很多人对长城的第一印象，正是从这道门洞开始的。

门洞内外，于是叠印着两种历史：一种是金戈铁马的古代，一种是握手言欢的当代。同一道门，不同的穿行者，这就是八达岭关城的独特之处。

## 必看亮点

- **"居庸外镇"门额**：东门之上的明代原刻石匾，字迹沉雄，是关城的身份铭牌。
- **"北门锁钥"门额**：西门之上，与东门额遥相呼应，读懂它才算读懂八达岭的战略地位。
- **瓮城格局**：在城门与瓮城之间走上一圈，看错位门洞与四面高墙如何构成"口袋阵"。
- **登城起点**：关城是南北两线的分岔口，城楼上可远眺关沟群山，是规划线路的最佳立足点。
- **门洞光影**：厚重的城门洞内冬暖夏凉，逆光拍摄门洞尽头的山色，极具纵深感。
- **微型军城想象**：站在城内空地对照说明牌，想象昔日官署营房的布局，在脑中复原一座军城。
- **晨昏门楼**：清晨与黄昏时分的门楼剪影，是拍摄关城最好的光线。

## 参观贴士

- **先读门额再登城**：进出城门时抬头看两道明代门额，三分钟了解八达岭的战略定位，比埋头爬墙收获更大。
- **线路分岔**：关城北侧登北线（好汉坡方向，热门），南侧登南线（人少坡缓），两线均需原路返回关城。
- **顺游博物馆**：关城脚下即[中国长城博物馆](/landmarks/badaling/changcheng-museum)，免费开放，建议登城前先入内打底。
- **城上寻踪**：登上关城城墙后向东看，可找到[望京石与"天险"摩崖题刻](/landmarks/badaling/wangjingshi)，下城后顺路寻访。
- **错峰出入**：旺季关城检票口人流集中，开城早到或午后晚些时候入城体验更佳。
- **东西穿行**：关城可东西方向穿行，两个门额进出各看一次，顺路把瓮城也走了。
- **城墙短走**：不想登大线的游客，也可在关城城墙上短距离行走，俯瞰关沟谷地。
- **勿登野墙**：关城两侧均有正规登城通道，切勿翻越围栏攀爬未开放墙段。
