# 北京城市公园

> "皇家园林与现代公园，构成北京独特的城市绿色空间。"

北京的公园既有传承千年的皇家园林，也有现代规划的城市公园。从琼岛白塔到奥林匹克森林公园，从皇家祭坛到市民乐园，北京的公园展现了这座城市的历史底蕴和现代风貌。

## 皇家园林

### [北海公园](/parks/beihai-park)

现存最古老皇家园林，"让我们荡起双桨"。

- **门票**：联票20元
- **特色**：白塔、九龙壁、让我们荡起双桨
- **最佳季节**：夏季荷花

### [景山公园](/parks/jingshan-park)

俯瞰紫禁城最佳位置。

- **门票**：2元
- **特色**：万春亭、崇祯自缢处
- **最佳时间**：日出日落

### [中山公园](/parks/zhongshan-park)

明清社稷坛，郁金香名园。

- **门票**：3元
- **特色**：社稷坛、郁金香展、古柏
- **最佳季节**：春季

### [劳动人民文化宫](/parks/cultural-palace)

明清太庙，祭祖圣地。

- **门票**：2元
- **特色**：古柏森森、皇家祭祀建筑
- **位置**：天安门东侧

## 皇家祭坛

### [天坛公园](/parks/temple-heaven)

明清祭天圣地，世界文化遗产。

- **门票**：联票28元
- **特色**：祈年殿、回音壁、圜丘坛
- **建议时长**：3-4小时

### [地坛公园](/parks/ditan-park)

明清祭地之所，银杏大道。

- **门票**：2元
- **特色**：银杏大道、祭地坛
- **最佳季节**：秋季

### [日坛公园](/parks/ritan-park)

明清祭日之所。

- **门票**：免费
- **特色**：日坛、使馆区
- **特色**：安静清幽

### [月坛公园](/parks/yuetan-park)

明清祭月之所。

- **门票**：1元
- **特色**：月坛、古典园林
- **特色**：人少清静

## 现代公园

### [香山公园](/parks/xiangshan)

北京红叶最佳观赏地，秋季赏秋首选。

- **门票**：10元（旺季）
- **特色**：红叶、香炉峰、碧云寺
- **最佳季节**：秋季

### [北京植物园](/parks/botanical-garden)

亚洲最大植物园，万木争荣。

- **门票**：10元（旺季）
- **特色**：温室群、专类园、卧佛寺
- **最佳季节**：春季

### [北京动物园](/parks/beijing-zoo)

中国最大动物园，亲子游首选。

- **门票**：15元（旺季）
- **特色**：大熊猫、海洋馆、各种动物
- **最佳时间**：上午

### [奥林匹克森林公园](/parks/olympic-forest)

北京最大的城市公园。

- **门票**：免费
- **特色**：奥运森林公园、鸟巢、水立方
- **面积**：680公顷

### [温榆河公园](/parks/wenyuhe)

北京最大城市公园，城市绿肺。

- **门票**：免费
- **特色**：湿地、花海、露营
- **面积**：30平方公里

### [朝阳公园](/parks/chaoyang-park)

北京四环内最大城市公园。

- **门票**：5元
- **特色**：万人广场、游乐场
- **活动**：音乐节、庙会

### [玉渊潭公园](/parks/yuyuantan)

北京最大樱花观赏地。

- **门票**：2元
- **特色**：樱花、电视塔倒影
- **最佳季节**：春季

### [紫竹院公园](/parks/zizhuyuan)

竹景名园，免费开放。

- **门票**：免费
- **特色**：竹林、荷花、划船
- **特色**：人少景美

## 公园游览建议

### 按季节推荐

| 季节 | 推荐公园 | 看点 |
|:---|:---|:---|
| 春 | 玉渊潭、中山公园 | 樱花、郁金香 |
| 夏 | 北海、颐和园 | 荷花、避暑 |
| 秋 | 香山、地坛 | 红叶、银杏 |
| 冬 | 什刹海、北海 | 滑冰、雪景 |

### 按类型推荐

| 类型 | 推荐公园 |
|:---|:---|
| 皇家园林 | 北海、颐和园、圆明园 |
| 登高望远 | 景山、香山、奥林匹克塔 |
| 亲子游玩 | 动物园、科技馆、朝阳公园 |
| 休闲散步 | 什刹海、紫竹院、日坛 |
| 健身运动 | 奥林匹克森林公园、朝阳公园 |

### 注意事项

1. **门票**：大部分公园门票很便宜（2-10元），但皇家园林较贵
2. **时间**：大部分公园6:00-7:00开门，21:00左右关门
3. **晨练**：早上有很多晨练的老人，可以体验北京生活
4. **划船**：北海、颐和园、玉渊潭等可以划船
5. **赏花**：春季樱花、郁金香、夏季荷花、秋季菊花

---

**💡 提示**：北京公园是市民生活的重要场所，不妨放慢脚步，体验一下老北京的生活节奏。
