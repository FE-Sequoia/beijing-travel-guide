# 西洋楼建筑群

![西洋楼遗址](https://images.unsplash.com/photo-1555085634-7b3d72893f38?w=1200&q=80)

> 在东方园林的心脏地带，曾矗立着一座巴洛克式的幻境——它是乾隆皇帝眼中的"西洋奇器"，也是中西文明相遇的石头证词。

## 简介

西洋楼位于长春园北部狭长地带，是圆明园三园中**唯一的西式景区**，南北宽约百米，东西绵延约750米，占地约**6万平方米**。乾隆十二年至二十四年（1747-1759年）间，由意大利传教士郎世宁、法国传教士蒋友仁等西洋匠师设计监造，中国工匠施工建成。

景区以**巴洛克风格**为基调，又大量融入中式纹样与琉璃构件，是18世纪东西方建筑艺术罕见的大规模融合实例。全景区自西向东由谐奇趣、蓄水楼、万花阵、养雀笼、方外观、海晏堂、大水法、观水法、远瀛观、线法山、线法墙等十余组建筑与园林构成，以[大水法](/landmarks/yuanmingyuan/dashuifa)喷泉群为高潮。

1860年遭焚劫后，西洋楼因石构居多而残迹留存最多，成为今日圆明园遗址的核心展区。这片绵延的石砌废墟，也是中国人最熟悉的圆明园形象。

### 谐奇趣与蓄水楼

**谐奇趣**是西洋楼最早建成的建筑，楼前有海棠形喷水池，池中铜鹅、铜羊喷水，楼内陈设西洋乐器，是演奏欧洲音乐之处。其北的**蓄水楼**为二层楼式蓄水池，以铜板焊接盛水180余吨，靠水力机械将水提升至高处，再凭重力供给全区喷泉——它是整座西洋楼水法系统的"心脏"。

### 万花阵（黄花阵）

**万花阵**是一座仿欧式园林的**花园迷宫**，阵墙高约1.2米，以雕花砖墙盘成回环往复的迷阵，中心设圆形凉亭。

中秋之夜，皇帝坐于中心亭中，看宫女手持黄绸扎制的莲花灯在阵中穿行，先至亭前者受赏，故又名"黄花阵"。如今的万花阵是1989年按原样复建的，是西洋楼景区内难得的可以"走入其中"体验的景观。

### 养雀笼与方外观

**养雀笼**为饲养孔雀等珍禽之所，券门两侧设铜丝笼架，昔日孔雀开屏于石券之间。**方外观**是乾隆为容妃（民间传说中的"香妃"）所建的礼拜之处，建筑西向，内部曾刻有阿拉伯文《古兰经》经文，是清代民族融合的一段见证。

### 线法山与线法墙

景区最东端为**线法山**与**线法墙**。"线法"即西洋透视画法：线法墙上曾张挂描绘欧洲街市、田园的巨幅透视画，皇帝立于特定视点望去，画景与实景连成一片，如身临其境——堪称18世纪的"沉浸式"视觉装置。

## 历史故事

### 乾隆的"西洋玩具"

西洋楼的建造源于乾隆对欧洲水法的好奇。传教士蒋友仁献上的欧洲喷泉图样引起皇帝极大兴趣，遂命在长春园北垣辟地建造。乾隆常于午后驾临西洋楼，观赏大水法喷涌、走万花阵迷灯、在方外观小坐。

对他而言，西洋楼是"天朝上国"尽收天下奇巧的缩影——西方技艺在此被收纳为皇权的点缀。这种"中体西用"的姿态，恰是那个时代中西关系的微妙注脚。

### 铜版画的最后影像

乾隆年间，宫廷委托制作**《圆明园西洋楼铜版图》**二十幅，以欧洲铜版蚀刻技法精细记录全景区面貌。这套铜版画成为西洋楼最完整的图像档案——1860年焚毁之后，后人正是依靠它复原想象当年的盛景。

如今园内多处展板即取自这套铜版画，是游客对照废墟、还原历史的最佳依据。1970年代后，中国学者依据铜版画与文献逐步清理复原遗址，1989年复建万花阵，2000年代起遗址区全面开放。

## 必看亮点

1. **[大水法](/landmarks/yuanmingyuan/dashuifa)残柱阵**：西洋楼主景，圆明园的标志性画面
2. **[海晏堂](/landmarks/yuanmingyuan/haiyan-tang)残墙与喷水池遗址**：十二生肖兽首喷泉的原址
3. **万花阵迷宫**：复建的花园迷阵，可亲身穿行，中心凉亭可小憩
4. **远瀛观石门与雕花石屏**：西洋军旗甲胄纹样的石雕精品
5. **谐奇趣海棠形喷水池遗址**：西洋楼最早的建筑遗存
6. **铜版画对照展板**：每处遗址旁均设原貌图版，强烈建议逐一对照

## 参观贴士

- 西洋楼遗址区需在10元大门票之外**另购15元门票**，入口位于景区内三园交界处以北
- 建议自西向东游览：谐奇趣 → 万花阵 → 养雀笼 → 方外观 → 海晏堂 → 大水法 → 远瀛观 → 线法山，全程约2小时
- 遗址石构件均为文物，**严禁攀爬、坐卧、触摸**
- 建议租语音导览或请讲解员：废墟若无背景知识，观感大打折扣
- 清晨开园即入或黄昏时段游览，光线最佳且游客稀少
