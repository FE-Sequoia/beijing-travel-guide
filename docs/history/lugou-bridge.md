# 卢沟桥

![卢沟桥](https://images.unsplash.com/photo-1599571234909-29ed5d1321d6?w=1200)

> "卢沟晓月，燕京八景；七七事变，民族记忆。"

## 简介

卢沟桥位于北京市丰台区永定河上，是北京现存最古老的石造联拱桥，也是"燕京八景"之一的"卢沟晓月"所在地。1937年7月7日，卢沟桥事变（七七事变）在此爆发，标志着中国全面抗战的开始。

## 历史沿革

### 始建

卢沟桥始建于金大定二十九年（1189年），金世宗完颜雍下令修建，金明昌三年（1192年）建成，历时四年。桥名"卢沟"取自永定河古名"卢沟"。

### 历代修缮

- **元代**：马可波罗游历中国，在《马可波罗游记》中称赞卢沟桥为"世界上最好的、独一无二的桥"
- **明代**：多次修缮，加固桥身
- **清代**：康熙年间重修，乾隆年间再次修缮
- **现代**：1982年全面修缮，1997年修复抗日战争时期的弹痕

### 七七事变

1937年7月7日夜，日本侵略军在卢沟桥附近进行挑衅性军事演习，借口士兵失踪，要求进入宛平城搜查。遭到拒绝后，日军炮轰宛平城，中国守军奋起抵抗。这就是震惊中外的"卢沟桥事变"（七七事变），标志着中国全面抗战的开始。

## 建筑特色

### 桥身结构

卢沟桥是一座多孔石拱桥，建筑结构独特。

| 项目 | 数据 |
|:---|:---|
| **全长** | 266.5米 |
| **宽度** | 7.5米 |
| **桥孔** | 11孔 |
| **桥墩** | 10座 |
| **桥身** | 石结构，采用纵联式结构 |

### 石狮

卢沟桥最著名的景观是桥栏上的石狮。

#### 石狮数量

- 桥两侧共有望柱281根
- 每根望柱上雕有石狮1-5只不等
- 石狮总数约500只

#### 石狮形态

卢沟桥的石狮形态各异，无一雷同。有的大狮抱小狮，有的嬉戏玩耍，有的昂首远眺，形态生动，栩栩如生。

#### 数狮歌

民间流传着"卢沟桥的石狮数不清"的说法。有"数狮歌"云：
> "卢沟桥的石狮，大的抱小的，小的抱更小的，更小的抱最小的，最小的还有更小的。"

### 碑亭

桥东头有碑亭两座，内立康熙和乾隆皇帝的御碑。

- **康熙碑**：立于康熙年间，记述重修卢沟桥经过
- **乾隆碑**：立于乾隆年间，记述"卢沟晓月"御笔

## 卢沟晓月

"卢沟晓月"是金代燕京八景之一，也是乾隆皇帝钦定的"燕京八景"之一。

### 景观特色

每当日出之前，晨霭茫茫，站在卢沟桥上向西远眺，可以欣赏"卢沟晓月"的美景。

### 乾隆御笔

乾隆皇帝曾多次游览卢沟桥，题写"卢沟晓月"四字，并立碑刻石。现存碑亭内的"卢沟晓月"碑即为乾隆御笔。

## 七七事变与抗战

### 事变经过

1937年7月7日夜，日军在卢沟桥附近进行军事演习，借口士兵失踪，要求进入宛平城搜查。遭到中国守军拒绝后，日军炮轰宛平城，中国第29军奋起抵抗。

### 弹痕遗址

卢沟桥上的石栏、望柱上至今保留着当年的弹痕，是日军侵华的铁证。这些弹痕已被列为文物保护对象，提醒后人铭记历史。

### 历史意义

七七事变标志着中国全面抗战的开始，是中华民族由衰败走向复兴的转折点。

## 文化价值

卢沟桥是中国古代桥梁建筑的杰作，也是抗日战争的重要历史见证。1961年，卢沟桥被列为第一批全国重点文物保护单位。

卢沟桥的石狮雕刻艺术和桥梁建筑技术，代表了金代石刻艺术和建筑技术的最高水平。
