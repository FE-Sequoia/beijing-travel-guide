# 天坛公园

> "明清祭天圣地，世界文化遗产。"

## 简介

天坛公园位于北京市东城区，是明清两代皇帝"祭天""祈谷"的场所。天坛占地约273公顷，是中国现存最大的古代祭祀建筑群，1998年被联合国教科文组织列入《世界遗产名录》。
