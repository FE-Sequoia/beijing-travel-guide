# 北京宗教建筑

> "多元宗教和谐共存，见证北京包容开放的城市精神。"

北京作为千年帝都，历来是多元文化交融之地。佛教、道教、天主教、基督教、伊斯兰教五大宗教在北京都有悠久的历史，留下了众多珍贵的宗教建筑。

## 天主教堂

北京是中国天主教传播最早的城市之一。明万历年间，利玛窦来京传教，建立教堂。如今北京保存有天主教堂数十座，其中最著名的是"四大堂"。

### [南堂（宣武门教堂）](/religion/nantang)

北京最古老的天主教堂，始建于1605年。

- **风格**：巴洛克式
- **特色**：康熙御笔「万有真原」匾额
- **地址**：西城区前门西大街

### [北堂（西什库教堂）](/religion/beitang)

北京最大、最宏伟的天主教堂，哥特式建筑。

- **风格**：哥特式
- **特色**：高耸尖塔、彩色玻璃窗
- **地址**：西城区西什库大街

### [东堂（王府井教堂）](/religion/dongtang)

位于繁华王府井，罗马式建筑。

- **风格**：罗马式
- **特色**：新人拍婚纱照热门地
- **地址**：东城区王府井大街

### [西堂（西直门教堂）](/religion/xitang)

四大堂中规模最小，意大利式风格。

- **风格**：意大利式
- **特色**：老北京风情
- **地址**：西城区西直门内大街

## 佛教寺院

### [雍和宫](/religion/yonghegong)

北京最大的藏传佛教寺院，清代皇家寺院。

- **门票**：25元
- **特色**：檀香木弥勒大佛、乾隆出生地
- **地址**：东城区雍和宫大街

### [潭柘寺](/religion/tanzhe)

"先有潭柘寺，后有北京城"。

- **门票**：50元
- **特色**：帝王树、流杯亭、古刹名刹
- **地址**：门头沟区潭柘寺镇

### [戒台寺](/religion/jietai)

中国三大戒台之一。

- **门票**：45元
- **特色**：戒台、松、牡丹
- **地址**：门头沟区马鞍山

### [法源寺](/religion/fayuan)

北京最古老的名刹，中国佛学院所在地。

- **门票**：5元
- **特色**：丁香、古刹、佛学研究
- **地址**：西城区法源寺前街

### [广济寺](/religion/guangji)

中国佛教协会所在地。

- **门票**：免费
- **特色**：佛教活动中心、免费斋饭
- **地址**：西城区阜成门内大街

## 道教宫观

### [白云观](/religion/baiyunguan)

道教全真派祖庭，北京最大道观。

- **门票**：10元
- **特色**：丘处机遗蜕、摸石猴祈福
- **地址**：西城区白云观街

### [东岳庙](/religion/dongyuemiao)

北京最大的道教庙宇。

- **门票**：10元
- **特色**：七十六司、民俗博物馆
- **地址**：朝阳区朝阳门外大街

### [火神庙](/religion/huoshen)

什刹海边的道教庙宇。

- **门票**：免费
- **特色**：火神祭祀、后海风光
- **地址**：西城区地安门外大街

## 参观须知

### 着装要求

- 避免穿短裤、背心等过于暴露的服装
- 进入寺庙建议脱帽
- 女士进入清真寺需包头巾

### 行为规范

- 保持安静，不要大声喧哗
- 不要在宗教场所内吸烟、饮食
- 未经允许不要拍摄宗教仪式
- 不要触摸佛像、神像

### 祈福礼仪

- 佛教：双手合十，三鞠躬
- 道教：抱拳礼，三鞠躬
- 天主教/基督教：十字礼

## 文化意义

北京的宗教建筑体现了中国多元宗教和谐共存的传统。从佛教的"慈悲为怀"到道教的"道法自然"，从天主教的"博爱"到伊斯兰教的"和平"，各种宗教文化在北京交融发展，共同构成北京丰富的文化景观。

这些宗教建筑不仅是信仰的场所，也是艺术和建筑的瑰宝。无论是佛教的殿堂楼阁，还是天主教的哥特尖塔，都体现了人类对美的追求和对神圣的向往。
