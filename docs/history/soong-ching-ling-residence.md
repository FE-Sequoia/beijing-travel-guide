# 宋庆龄故居

![宋庆龄故居](https://images.unsplash.com/photo-1583977393611-885b3e5cc701?w=1200)

> "国之瑰宝，二十世纪中国最伟大的女性之一。"

## 简介

宋庆龄故居位于北京市西城区后海北沿46号，是一座典型的北京四合院。宋庆龄女士从1963年至1981年在此居住，度过了人生的最后18年。故居原为清代醇亲王府花园的一部分，环境优美，现已成为"宋庆龄同志故居"，展示着这位伟大女性的生平事迹。

## 历史沿革

### 醇亲王府花园

宋庆龄故居原为清代醇亲王府的花园。醇亲王是光绪帝的父亲、宣统帝的祖父，王府建于清代，花园精美典雅。

### 宋庆龄晚年

1963年，宋庆龄女士迁入此院，在此居住直至1981年逝世。这18年间，她继续从事中国福利事业和儿童工作，接待了众多国内外宾客。

#### 晚年成就

| 年份 | 事件 |
|:---|:---|
| 1963年 | 迁入后海北沿46号 |
| 1966年 | 创立中国福利会 |
| 1981年 | 逝世于此，享年88岁 |

## 故居布局

### 四合院结构

宋庆龄故居是一座典型的北京四合院，分为前院、中院和后院，环境清幽，花木扶疏。

#### 建筑布局

| 建筑 | 功能 | 特色 |
|:---|:---|:---|
| **大门** | 入口 | 传统朱漆大门 |
| **前院** | 接待区 | 客厅、接待室 |
| **中院** | 生活区 | 宋庆龄居室、书房 |
| **后院** | 花园 | 百年海棠、月季 |
| **东西厢房** | 展览室 | 生平事迹展 |

### 主要展室

#### 宋庆龄居室

宋庆龄的居室简朴而典雅，保留了她生前的原貌。

- **卧室**：简朴的卧室，摆放着她生前使用的家具
- **书房**：她工作和学习的地方，藏书丰富
- **会客室**：会见国内外宾客的场所

#### 生平事迹展

故居内设有宋庆龄生平事迹展，展示了这位伟大女性的一生。

## 花园景观

### 百年海棠

故居花园内有两棵百年西府海棠，是宋庆龄生前最喜爱的花木。每年春天，海棠花开，繁花似锦。

### 月季花

宋庆龄生前喜爱月季，花园内种植了各种月季，四季常开。

## 游览攻略

### 推荐路线

**精华路线（约1小时）**：

入口 → 前院（客厅） → 中院（宋庆龄居室、书房） → 后院（花园、百年海棠） → 东西厢房（生平展） → 出口

### 实用贴士

1. **提前预约**：建议提前在官网或电话预约
2. **最佳时间**：春季海棠花开时最美
3. **拍照位置**：百年海棠前、书房前都是绝佳拍照点
4. **讲解服务**：可以租用讲解器，了解更多历史
5. **周边游览**：参观完后可以到什刹海、恭王府游览

## 文化意义

宋庆龄是中国近现代史上杰出的女性政治家、社会活动家，被尊称为"国之瑰宝"。她的一生是为中国革命和建设事业奋斗的一生，她的精神永远激励着后人。

"永远和党在一起"，这是宋庆龄先生的遗愿，也是她一生的真实写照。
