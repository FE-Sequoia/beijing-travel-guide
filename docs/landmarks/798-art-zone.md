# 798艺术区

![798艺术区](https://images.unsplash.com/photo-AWr2a2EK_A8?w=1200)

> "老厂房里的艺术梦想，工业与时尚的完美碰撞。"

## 简介

798艺术区位于北京市朝阳区酒仙桥街道，由原北京798厂等电子工业老厂区改造而成，是中国当代艺术的重要展示平台。这里汇聚了画廊、艺术家工作室、设计公司、时尚店铺等，是北京最具国际影响力的艺术区。

## 历史沿革

### 工业时期（1950-1990年代）

798艺术区的前身是北京798厂等电子工业厂区，建于20世纪50年代。这里曾是中国电子工业的重要基地，生产过大量电子元器件。

#### 建筑风格

厂区的建筑多为包豪斯风格，由德国专家设计，是中国最早采用包豪斯风格的建筑群之一。

### 艺术转型（2000年代）

2002年，一批艺术家开始在这里租用废弃的厂房作为工作室。2006年，798艺术区正式命名为"北京798艺术区"。

#### 发展历程

| 时间 | 事件 |
|:---|:---|
| 2002年 | 第一批艺术家入驻 |
| 2004年 | 画廊和艺术机构大量进驻 |
| 2006年 | 正式命名"北京798艺术区" |
| 2007年 | 举办首届798艺术节 |
| 2010年 | 成为北京文化创意产业的重要基地 |

## 主要区域

### 画廊区

798艺术区内有众多国内外知名画廊，展示着各种当代艺术作品。

#### 知名画廊

| 画廊 | 特色 |
|:---|:---|
| **佩斯画廊** | 国际知名画廊，展示当代艺术 |
| **长征空间** | 中国当代艺术重要平台 |
| **唐人当代艺术中心** | 展示亚洲当代艺术 |
| **站台中国** | 关注中国当代艺术 |

### 艺术家工作室

798艺术区内有大量艺术家工作室，游客可以近距离接触艺术家的创作过程。

#### 工作室特色

- **雕塑工作室**：可以观看雕塑创作过程
- **绘画工作室**：可以观看油画、国画创作
- **装置艺术工作室**：可以了解装置艺术的创作理念

### 创意市集

798艺术区内经常举办创意市集，出售各种原创设计产品。

#### 市集特色

- **原创设计**：服装、饰品、家居用品等
- **手工艺品**：陶艺、木作、皮具等
- **艺术衍生品**：画册、明信片、文创产品等

## 游览攻略

### 推荐路线

**精华路线（约3小时）**：

798创意广场 → 画廊区 → 艺术家工作室 → 创意市集 → 火车头广场 → 包豪斯广场 → 涂鸦墙

**深度路线（约一天）**：

上午：画廊区 → 艺术家工作室
下午：创意市集 → 包豪斯广场 → 涂鸦墙 → 艺术书店
晚上：798内餐厅用餐，感受艺术区夜生活

### 实用贴士

1. **最佳时间**：周末人较多，建议工作日前往
2. **展览信息**：关注798官网，了解最新展览信息
3. **拍照打卡**：涂鸦墙、火车头、创意广场都是绝佳拍照点
4. **购物建议**：创意市集上的商品价格可以议价
5. **餐饮推荐**：798内有很多特色餐厅和咖啡馆

## 文化意义

798艺术区是中国当代艺术的发源地之一，也是中国文化创意产业的标杆。这里不仅展示了当代艺术的魅力，更体现了中国城市更新的成功案例。
