# 天坛

> "圜丘坛上望苍穹，祈谷殿中祷丰年。"
> —— 清代诗人

## 简介

天坛是明清两代皇帝"祭天""祈谷"的场所，位于北京市东城区永定门内大街东侧。天坛占地约273公顷，是中国现存最大的古代祭祀建筑群，也是世界上现存规模最大、最完美的古代祭天建筑群。

## 历史沿革

### 始建

明永乐十八年（1420年），明成祖朱棣建成北京天地坛，用于合祀天地。嘉靖九年（1530年），明世宗朱厚熜实行天地分祀，在南郊另建圜丘坛，专祀天；在北郊建方泽坛，专祀地。天地坛改称天坛。

### 改建

嘉靖二十四年（1545年），改建大享殿（即祈年殿），将原三色琉璃瓦屋顶改为统一青瓦。清乾隆年间，又将祈年殿、皇穹宇、圜丘坛等建筑全部改为蓝瓦金顶，形成今日面貌。

## 建筑布局

天坛分内坛和外坛，主要建筑集中在内坛。

### 主要建筑

#### 祈年殿

祈年殿是天坛的主体建筑，是一座直径32米、高38米的圆形三重檐建筑。殿内28根楠木巨柱象征天空星辰：

- 中央4根"龙井柱"象征四季
- 中圈12根"金柱"象征十二月
- 外圈12根"檐柱"象征十二时辰

殿顶三层蓝色琉璃瓦代表天，鎏金宝顶熠熠生辉。

#### 圜丘坛

圜丘坛是皇帝冬至祭天的场所，是一座三层汉白玉圆坛。坛面、台阶、栏杆所用石块均为九或九的倍数，象征"九重天"。站在坛心石上说话，声音会被放大，产生奇妙的回音效果。

#### 皇穹宇

皇穹宇是供奉祭天神牌的场所，以其著名的"回音壁"而闻名。围墙是磨砖对缝的圆形围墙，声波可以沿墙面传播，两人分站东西两侧，贴墙说话，声音清晰可闻。

#### 丹陛桥

连接圜丘坛和祈年殿的甬道，长360米、宽29.4米，比地面高出2.5米，称为"海墁大道"或"丹陛桥"，是天坛的"神路"。

## 文化故事

### 祭天礼仪

天坛祭天是明清两代最重要的国家典礼之一。每年冬至，皇帝要在圜丘坛举行祭天大典。仪式极其隆重：皇帝提前三天到斋宫斋戒，祭天当天身着礼服，在乐舞伴奏下，向皇天上帝献祭。

### 祈年殿的传说

传说祈年殿是按照"天圆地方"的理念设计的。殿内中央的"龙井柱"取材于云南深山老林，运送进京耗费巨大人力物力。民间传说，这些柱子是"神木"，有避邪镇宅的功效。

### 天心石

圜丘坛中心有一块圆形石板，称为"天心石"或"亿兆景从石"。据说站在上面呼唤，会得到上天回应，声音会被放大数倍。其实这是声波反射造成的声学现象。

## 游览攻略

### 推荐路线

**精华路线（约3小时）**：

南门 → 圜丘坛 → 皇穹宇（回音壁） → 丹陛桥 → 祈年殿 → 皇乾殿 → 北门

**深度路线（约4-5小时）**：

南门 → 昭亨门 → 圜丘坛 → 皇穹宇 → 回音壁 → 丹陛桥 → 祈年殿 → 皇乾殿 → 斋宫 → 神乐署 → 西门

### 实用贴士

1. **门票选择**：建议购买联票，否则看不到祈年殿等精华景点
2. **最佳时间**：早上7:30前入园可以避开人流，还能看到晨练的老人
3. **拍照位置**：祈年殿正面台阶处、圜丘坛中心都是绝佳拍照点
4. **回音壁**：现在为了保护文物，回音壁已被围栏隔开，回音效果不如从前
5. **附近美食**：南门涮肉（老北京铜锅涮肉）就在天坛南门对面，非常推荐

## 文化价值

1998年，天坛被联合国教科文组织列入《世界遗产名录》。世界遗产委员会评价：

> "天坛是建筑和景观设计之杰作，其布局体现了中国传统的宇宙观和天人合一的思想，是中国古代祭祀建筑的杰出代表。"

天坛的设计融合了中国古代"天圆地方""天人合一"的哲学思想，是中国古代建筑艺术的瑰宝。
