# 北海公园

![北海公园白塔](https://images.unsplash.com/photo-1768109027865-f5b57609fabd?w=1200&q=80)

> "让我们荡起双桨，小船儿推开波浪，海面倒映着美丽的白塔……"
> 一首唱了七十年的歌，唱的就是这里。

## 简介

北海公园位于北京市中心，故宫西北、景山以西，是中国现存**建园最早、保存最完整、历史最悠久的皇家园林之一**。它的历史可上溯至**辽代**（约公元10世纪），此后历经**辽、金、元、明、清五个朝代近千年**的营建与修缮，园址从未迁移、格局绵延不绝，堪称一部活着的中国皇家园林史。

整座园林以太液池（北海）为中心，以池中**琼华岛**及其山顶的**白塔**为视觉核心，环湖布置团城、北岸、东岸诸景。辽金时此处为帝王离宫别苑，元代忽必烈以琼华岛为中心营建大都、将此地圈入皇城；明清两代成为禁苑，称"西苑"。1925年北海正式辟为公园向公众开放，1961年被列为第一批全国重点文物保护单位。

登琼华岛而四望，南可望故宫角楼与中南海，北可眺景山万春亭，红墙金瓦与碧水白塔相映，是北京城中最具诗意的古典园林空间。

| 朝代 | 大事 |
|:---|:---|
| 辽代 | 始建离宫别苑，为园林之肇始 |
| 金代 | 扩建为皇家苑囿"太宁宫" |
| 元代 | 忽必烈以琼华岛为中心营建大都，圈入皇城 |
| 明清 | 成为皇家禁苑"西苑"，营建白塔、九龙壁、五龙亭诸景 |
| 1925年 | 正式辟为北海公园，向公众开放 |
| 1961年 | 列为第一批全国重点文物保护单位 |

## 推荐游览路线

### 🕐 经典游（2-3小时，精华速览）

南门进园，直奔核心，不走回头路：

> **南门** → **团城**（看白玉佛与渎山大玉海）→ 过永安桥登**琼华岛** → **永安寺** → 登**白塔**俯瞰全园 → 下山环岛 → 沿东岸北行 → **九龙壁** → **五龙亭** → **北门**出

这条路线串联了白塔、团城、九龙壁、五龙亭四大标志性景观，是初访者的首选。出北门即地铁6号线北海北站，交通极便。

### 🗓️ 深度游（约4小时，尽览全园）

上午走北岸人文精华，下午登岛览胜：

> 上午：**北门** → **静心斋**（乾隆小花园）→ **西天梵境**（金丝楠木大殿）→ **九龙壁** → **五龙亭**
> 下午：**团城** → **琼华岛**（永安寺、白塔、琼岛春阴碑、阅古楼）→ 沿东岸漫步濠濮间、画舫斋 → 南门或东门出

深度路线可从容品味北岸的园林与宗教建筑精华，再登岛俯瞰全城，适合时间充裕、喜爱古建的游客。

## 文化故事：让我们荡起双桨

1955年，长春电影制片厂拍摄了新中国第一部儿童题材影片**《祖国的花朵》**，片中孩子们在北海划船的镜头，正是取景于北海公园的太液池。影片主题曲**《让我们荡起双桨》**由乔羽作词、刘炽作曲，"海面倒映着美丽的白塔，四周环绕着绿树红墙"——歌词里写的正是琼华岛白塔与环湖的宫墙绿树。这首歌从此传唱大江南北，成为几代中国人的共同记忆。如今在北海划船，望着水中的白塔倒影，仍能体会到当年那份纯真与诗意。

## 实用贴士

1. **联票更划算**：联票20元已含琼华岛与团城，单买琼岛门票反而更贵，建议直接购联票。
2. **北门进最便**：地铁6号线北海北站紧邻北门，由此进园先游北岸诸景，再登琼华岛，动线顺畅。
3. **登塔看全景**：琼华岛白塔是俯瞰故宫、景山、中南海的最佳观景点，下午光线尤佳。
4. **预留划船时间**：若逢晴好天气，留一小时泛舟太液池，从水面回望白塔，是北海最美的视角。
5. **联游周边**：北海与景山公园仅一街之隔，可安排同日联游，登景山万春亭回望北海白塔。
