# 雍和宫

![雍和宫鎏金殿顶与红柱回廊](https://plus.unsplash.com/premium_photo-1693236418889-3e0d269c59ad?w=1200&q=80)

> "潜邸化禅林，行宫成梵宇——北京最大的藏传佛教寺院，清王朝民族团结与宗教交融的珍贵见证。"

## 简介

雍和宫位于北京市东城区雍和宫大街28号，坐北朝南，占地约6.6万平方米，是北京城区规模最大的藏传佛教寺院，也是清代内地规格最高的格鲁派（黄教）皇家寺院。整座寺院融汉、满、蒙、藏建筑艺术于一体，殿宇黄瓦红墙、格局严整，香火延续近三百年。

雍和宫的历史可分为清晰的三个阶段：

- **王府时期（1694年）**：清康熙三十三年（1694年）始建为"四爷府"，即皇四子胤禛的府邸；康熙四十八年胤禛晋封和硕雍亲王，府邸升格为"雍亲王府"。
- **行宫时期（1725—1735年）**：胤禛即位为雍正帝后，于雍正三年（1725年）将王府升为行宫，赐名"雍和宫"。雍正十三年（1735年）雍正帝驾崩，灵柩曾停于此，主要殿宇改覆黄色琉璃瓦，自此具备皇家宫殿规制。
- **寺院时期（1744年至今）**：乾隆九年（1744年）正式改为藏传佛教寺院（喇嘛庙），成为清廷管理藏传佛教事务的重镇，蒙藏高僧驻锡讲经，法事不绝。

相传乾隆皇帝即诞生于雍亲王府，雍和宫因此享有"龙潜福地"之誉。如今这里既是各族信众礼佛祈福的圣地，也是中外游人凭吊历史、欣赏建筑艺术的胜地。

## 祈福与请香贴士

![殿前香炉敬香](https://images.unsplash.com/photo-1766872697135-ac4aaee364ec?w=1200&q=80)

- **免费领香**：雍和宫为游客免费提供环保香，无需自带外香；心诚则灵，三支清香即可。
- **敬香礼仪**：左手持香举至眉心，躬身三拜后插入香炉；过殿门勿踩踏门槛。
- **殿内勿摄**：各殿佛像、唐卡一律禁止拍照；院落内可拍摄，但请避免直拍僧人与礼佛信众。
- **保持肃静**：宗教场所请低声缓行，勿以手指点佛像，尊重各民族宗教习俗。
- **法物请购**：寺内设有正规法物流通处，开光等事宜请咨询寺院工作人员，勿轻信门外招揽。
