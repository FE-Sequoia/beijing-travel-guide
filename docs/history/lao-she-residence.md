# 老舍故居

![老舍故居](https://images.unsplash.com/photo-1662792193082-9f02de0c12f0?w=1200)

> "人民艺术家，京味文学的代言人。"

## 简介

老舍故居位于北京市东城区灯市口西街丰富胡同19号，是一座典型的北京四合院。老舍先生从1950年至1966年在此居住，在这里度过了人生的最后16年，也是他创作生涯最辉煌的时期。《龙须沟》《茶馆》等不朽名作均诞生于此。

## 历史沿革

### 故居由来

1950年，老舍从美国讲学归来，购置了这座四合院。这座宅院成为他后半生的居所，也是他创作《龙须沟》《茶馆》等名作的地方。

### 晚年岁月

在这16年间，老舍先生创作了《龙须沟》《茶馆》《正红旗下》等重要作品，并荣获"人民艺术家"称号。

#### 创作年表

| 年份 | 作品 |
|:---|:---|
| 1950年 | 从美国归来，入住丰富胡同 |
| 1951年 | 创作《龙须沟》，获"人民艺术家"称号 |
| 1956年 | 创作《茶馆》 |
| 1966年 | 文革开始，8月24日逝世 |

## 故居布局

### 四合院结构

老舍故居是一座典型的老北京四合院，占地约500平方米。

#### 建筑布局

| 建筑 | 功能 | 特色 |

| **影壁** | 照壁 | 砖雕影壁 |
| **正房** | 第一展室 | 生平与创作历程展 |
| **东厢房** | 第二展室 | 生活用品、家庭照片 |

### 主要展室

#### 第一展室（正房）

展示老舍先生的生平与创作历程。

- **生平介绍**：从出生到逝世的生平事迹
- **创作历程**：各时期的代表作品
- **荣誉展示**：获得的荣誉和称号

#### 第二展室（东厢房）

展示老舍先生的生活用品和家庭照片。

- **生活用品**：眼镜、毛笔、茶具等
- **家庭照片**：与家人、友人的合影
- **剧照**：《茶馆》《龙须沟》首演照片

#### 第三展室（西厢房）

展示老舍先生的代表作品和手稿。

- **手稿**：《正红旗下》等作品原稿
- **著作**：各种版本的著作
- **藏书**：约2500册藏书

## 后院花园

### 太平花

后院种植着两棵太平花，是老舍先生亲手栽种的。每年春夏之交，太平花盛开，香气袭人。

### 其他花木

院内还种植着丁香、黄刺玫、榆叶梅等花木，都是老舍先生生前喜爱的植物。

## 文化故事

### "老舍"笔名的由来

老舍原名舒庆春，他把"舒"字拆开，取"舍"和"予"，寓意"舍弃自我"。这个笔名伴随他一生，成为京味文学的标志。

### 《茶馆》的诞生

1956年，老舍在故居完成了话剧《茶馆》的创作。这部剧被誉为"东方舞台上的奇迹"，至今仍是北京人艺的保留剧目。

### 人民艺术家

1951年，北京市人民政府授予老舍"人民艺术家"的荣誉称号，老舍成为新中国第一位获得这一称号的作家。
