# 雍和宫殿（大雄宝殿 / 银安殿）

![雍和宫殿黄瓦大殿](https://images.unsplash.com/photo-1669518608789-2366f3cd7a2f?w=1200&q=80)

> "昔日银安殿，今朝大雄宫——王府正堂之上，三世佛端坐，十八罗汉列班，须弥山铜铸于庭。"

## 简介

雍和宫殿是雍和宫的主殿，居全寺正中，相当于汉地寺院的大雄宝殿。其前身是雍亲王府的正殿"银安殿"——亲王举行大典、接见臣属的王府正堂，也是整座王府等级最高的建筑。乾隆九年（1744年）改寺后，正殿供奉三世佛，更名"雍和宫殿"，寺因殿名，殿以寺重。

殿内正中供奉三世佛，两侧十八罗汉肃然侍立，佛像背后为西方三圣；殿后东院落中更有一座明代铜铸须弥山，是佛教宇宙观的微缩模型，极为珍罕。

## 建筑与景观

| 项目 | 规格与内容 |
|:---|:---|
| 殿宇规制 | 面阔七间，重檐歇山黄琉璃瓦顶，原王府银安殿升格 |
| 正中主供 | 三世佛：现在佛释迦牟尼居中，过去佛燃灯佛居左，未来佛弥勒佛居右 |
| 两侧 | 十八罗汉，分列东西两厢，神态各异 |
| 佛像背后 | 西方三圣（阿弥陀佛、观世音菩萨、大势至菩萨） |
| 殿后东院 | 铜铸须弥山，明万历年间铸造，佛教宇宙模型 |

三世佛代表佛教的时间观：燃灯佛为过去佛，曾为释迦授记；释迦牟尼为现在佛，娑婆世界的教主；弥勒佛为未来佛，将于未来降世度人。三佛并列，寓意佛法贯通过去、现在、未来三世。

### "银安殿"名号由来

"银安殿"之名，源自金代皇宫正衙"永安殿"，取"永保平安"之意。清代亲王、郡王府邸的正殿皆沿称"银安殿"，是王府中举行册封、节庆、接见等重大典礼的场所，规制仅低于皇宫太和殿一等。雍亲王府的银安殿面阔七间、前出丹陛，正是亲王正殿的标准格局。王府改寺，银安殿不拆不毁，只将殿内的王座仪仗请出，换上三世金身——殿宇犹是故物，气象已属梵宫。

### 铜铸须弥山

须弥山位于雍和宫殿后的东院落，为明万历年间铸造的铜质模型，原为明代佛寺旧物，后移奉于此。山体分层，自下而上依次展现佛教宇宙观：

- **底层咸海**：浩瀚海水之中分布东胜神洲、南赡部洲、西牛贺洲、北俱卢洲四大部洲及八小部洲，人类即居于南赡部洲；
- **中层七金山**：七重金山环绕主峰，层层渐低；
- **中央主峰须弥山**：上宽下窄，巍然独立，日月绕山而行；
- **峰顶忉利天宫**：帝释天（忉利天主）所居，即民间所谓"三十三天"。

整座铜山将佛教"三千大千世界"的宇宙图景凝于一器，构思奇巧，是雍和宫少有人知却极珍贵的宝物。

## 历史故事

### 银安殿改覆黄瓦

雍正十三年（1735年）八月，雍正帝驾崩于圆明园，梓宫移奉雍和宫，停于银安殿。按清制，亲王殿宇用绿琉璃瓦，唯帝后可用黄瓦。为奉安梓宫，银安殿等主要殿宇一夜之间改覆黄琉璃瓦，王府规制就此升为宫殿规制。这片黄瓦，既是人子之孝，也是帝王之尊，至今覆盖殿顶近三百年。

### "雍和"之名

"雍和"二字，取"雍穆和睦、天下和洽"之意。从雍亲王府到雍和宫，再到寺院主殿，这个名字一路沿用，成为全寺之名。主殿以"雍和"为名，也寄寓着这座皇家寺院护国佑民、协和万邦的初衷。

### 须弥山的明代身世

这座铜铸须弥山并非雍和宫原配之物。它铸于明万历年间，原供奉于明代佛寺，是当时信众集资铸造的供器。清初辗转移奉雍和宫，置于主殿东院。一件明代供器进入清代皇家寺院，恰好见证了两朝更替、佛事不绝的历史——佛门法物不问朝代，只问因缘，这也是须弥山最动人的地方。

## 必看亮点

- **王府正殿规制**：面阔七间、黄瓦重檐，是雍亲王府银安殿的原构，等级在京城寺院主殿中无出其右。
- **三世佛并坐**：过去燃灯、现在释迦、未来弥勒，藏传佛教寺院的典型时间佛组合。
- **十八罗汉**：东西两厢十八尊，造型生动，为清代精品。
- **铜铸须弥山**：明万历铜铸佛教宇宙模型，位于殿后东院，切勿错过。
- **殿前香炉与经幡**：庭中古柏、铜炉、经幡与黄瓦大殿相映，是雍和宫最经典的画面。

## 参观贴士

- 瞻仰三世佛时，可按"左过去、中现在、右未来"的顺序辨认（以佛的方位为准）。
- 铜铸须弥山在殿后东院落，位置偏僻，多数游客径直略过，建议专程绕看。
- 殿内禁止拍照，殿外庭院可拍摄黄瓦红墙的经典角度。
- 殿前广场为信众敬香主要区域，礼佛高峰时段请有序等候、勿拥挤。
- 此处为寺院核心礼佛区，请保持安静，随喜信众功德。
