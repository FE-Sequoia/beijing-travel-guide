# 东岳庙

> "北京最大的道教庙宇，七十六司神秘莫测。"

## 简介

东岳庙位于北京市朝阳区，是北京最大的道教庙宇，也是正一派在华北地区最大的庙宇。东岳庙始建于元代延祐六年（1319年），供奉东岳泰山神天齐仁圣大帝，主司人间生死祸福。

## 历史沿革

### 始建

东岳庙始建于元延祐六年（1319年），由玄教大宗师张留孙及其弟子吴全节所建，供奉东岳泰山神。

### 明清发展

明代和清代，东岳庙多次扩建，规模不断扩大，成为北京最大的道教庙宇。

### 现代

1997年，东岳庙修复后对外开放。1999年，设立北京民俗博物馆。

## 建筑布局

东岳庙规模宏大，占地约6万平方米，有房屋近400间。

### 山门

山门是东岳庙的入口，门额上书"敕建东岳庙"。

### 瞻岱门

瞻岱门是东岳庙的第二道门，"岱"指泰山。

### 岱宗宝殿

岱宗宝殿是东岳庙的主殿，供奉东岳泰山神天齐仁圣大帝。

#### 东岳大帝

东岳大帝是道教尊神，掌管人间生死祸福，是阴间的最高统治者。

### 七十六司

七十六司是东岳庙的特色，是东岳大帝管辖的76个部门。

#### 各司职能

| 司 | 职能 |
|:---|:---|
| **生死司** | 掌管生死 |
| **福禄司** | 掌管福禄 |
| **善恶司** | 掌管善恶 |
| **因果司** | 掌管因果 |
| **速报司** | 掌管报应 |
| **地狱司** | 掌管地狱 |

七十六司塑像神态各异，非常逼真，体现了民间对阴间的想象。

### 碑林

东岳庙内有大量碑刻，是研究道教和民俗的重要资料。

## 北京民俗博物馆

东岳庙内设有北京民俗博物馆，展示北京的民俗文化。

### 展览内容

- **老北京生活**：老北京的衣食住行
- **老北京节庆**：春节、中秋等传统节日
- **老北京行业**：各行各业的民俗
- **老北京信仰**：民间信仰

## 春节庙会

东岳庙每年春节举办庙会，是北京著名的庙会之一。

### 庙会时间

每年春节期间，正月初一至十五。

### 庙会内容

- **祈福法会**：道教祈福仪式
- **民俗表演**：舞狮、高跷等
- **民间手工艺**：面人、糖人等
- **传统小吃**：各种传统小吃

## 文化价值

东岳庙是北京最大的道教庙宇，是研究道教和民俗的重要场所。

东岳庙的七十六司是民间信仰的重要体现，反映了民间对生死、善恶的认识。

东岳庙的春节庙会是北京传统民俗的重要组成部分。
