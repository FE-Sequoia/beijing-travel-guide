# 圆明园

> "万园之园，东方凡尔赛，如今只剩断壁残垣。"

## 简介

圆明园是清代著名的皇家园林，由圆明园、长春园、绮春园三园组成，统称"圆明三园"，占地350多公顷，其中水面面积约140公顷。圆明园以其宏大的地域规模、杰出的营造技艺、精美的建筑景群、丰富的文化收藏和博大精深的民族文化内涵而享誉于世，被誉为"一切造园艺术的典范"和"万园之园"。

## 历史沿革

### 始建与发展

圆明园始建于康熙四十八年（1709年），最初是康熙帝赐给皇四子胤禛（即后来的雍正帝）的花园。雍正即位后大规模扩建，乾隆时期达到鼎盛，历经康雍乾嘉道咸六朝150余年的营造。

### 辉煌时期

鼎盛时期的圆明园，拥有景点150余处，建筑面积20余万平方米。园中仿建了多处江南名园胜景，如杭州西湖十景、苏州狮子林等，还建有西洋楼等西式建筑群。

### 浩劫与毁灭

咸丰十年（1860年）10月，英法联军攻入北京，对圆明园进行了疯狂洗劫，随后纵火焚烧，大火三天三夜不熄。1900年八国联军再次洗劫，民国时期的石劫、木劫、土劫使这座园林彻底化为废墟。

### 遗址公园

新中国成立后，圆明园遗址得到有效保护。1976年成立圆明园管理处，1988年对外开放，成为全国重点文物保护单位和爱国主义教育基地。

## 园林布局

### 圆明园（主园）

圆明园是主园，占地最大，以水面为主，仿江南园林布局，有"四十景"之称。主要景点包括：

- **正大光明殿**：皇帝在园中处理政务的地方
- **九洲清晏**：皇帝的寝宫区
- **勤政亲贤**：皇帝日常办公之所
- **镂月开云**：牡丹台，康熙、乾隆、嘉庆三代皇帝在此共赏牡丹
- **天然图画**：仿西湖"平湖秋月"而建

### 长春园

长春园以西洋楼建筑群著称，是园中唯一的西式景区。

#### 西洋楼遗址

西洋楼占地约6万平方米，由谐奇趣、蓄水楼、万花阵、养雀笼、方外观、海晏堂、大水法、观水法、远瀛观等十余座西式建筑和园林组成。

**海晏堂**：西洋楼中最大的建筑，楼前有十二生肖兽首人身像喷泉，每昼夜十二时辰依次喷水，正午时十二生肖同时喷水，蔚为壮观。

**大水法**：欧式喷泉，主楼前有椭圆形喷水池，池中有铜鹿、铜狗等喷水装置。残存的大理石柱至今矗立，成为圆明园的象征。

### 绮春园

绮春园原是若干私园，乾隆后期并入圆明园，嘉庆年间大规模扩建。主要景点有：

- **正觉寺**：清代皇室喇嘛庙
- **鉴碧亭**：临水而建，是赏荷佳处
- **涵秋馆**：秋季赏景之所

## 文化故事

### 十二生肖兽首

海晏堂前的十二生肖兽首人身铜像，由意大利传教士郎世宁设计，法国人蒋友仁监修，是中西合璧的艺术精品。1860年英法联军劫掠后，兽首流散世界各地。

目前已有牛首、猴首、虎首、猪首、马首、鼠首、兔首回归祖国，龙首据说在台湾，蛇首、羊首、鸡首、狗首至今下落不明。

### 文源阁与《四库全书》

文源阁是圆明园中收藏《四库全书》的处所。《四库全书》是清代乾隆年间编纂的大型丛书，收录书籍3503种，79337卷，是中国历史上最大的一部丛书。文源阁于1860年被焚毁，《四库全书》部分被劫掠，部分被焚毁。

### 咸丰帝与圆明园

咸丰十年（1860年），英法联军逼近北京，咸丰帝携慈禧等人逃往热河避难。临行前，咸丰帝在圆明园度过了最后的时光。一年后，咸丰帝在热河病逝，终年仅30岁。

## 游览攻略

### 推荐路线

**精华路线（约3小时）**：

南门 → 十二生肖兽首展 → 绮春园 → 三园交界 → 长春园 → 西洋楼遗址（谐奇趣→万花阵→养雀笼→方外观→海晏堂→大水法→观水法→线法山→线法墙）→ 东门

**深度路线（约5-6小时）**：

南门 → 正觉寺 → 绮春园 → 三园交界 → 圆明园（九州景区→福海景区）→ 长春园 → 西洋楼遗址 → 全景模型 → 东门

### 实用贴士

1. **联票建议**：一定要购买含西洋楼遗址的联票，否则看不到大水法等标志性景观
2. **游览车**：园区较大，可乘电瓶车（10-15元/站）节省体力
3. **讲解服务**：建议请人工讲解或租用语音导览，了解历史背景更有意义
4. **最佳时间**：春秋两季，尤其是秋天银杏黄时
5. **心态准备**：圆明园是一个"看废墟"的地方，带着历史敬畏心来参观

## 历史意义

圆明园的毁灭是中华民族的奇耻大辱，也是人类文明史上的一场浩劫。它不仅是一座园林的毁灭，更是中华文明遭受重创的象征。

如今的圆明园遗址公园，以残垣断壁向世人诉说着那段屈辱的历史，成为爱国主义教育的重要基地。正如法国作家雨果所说：

> "有一天，两个强盗闯进了圆明园。一个抢劫，一个放火……在历史面前，这两个强盗一个叫法兰西，一个叫英吉利。"
