# 绮春园

![绮春园荷塘](https://images.unsplash.com/photo-1753087457645-a4f6c7c64e39?w=1200&q=80)

> "绮春"者，春色如绮。这里曾是太后太妃们的颐养之所，也是圆明园浩劫之后，唯一有古建筑幸存的地方。

## 简介

绮春园位于圆明三园东南部，是圆明园三园中成园最晚的一座。其地原为若干**私家赐园**（如大学士傅恒的赐园等），乾隆三十五年（1770年）前后陆续并入御园，定名"绮春园"；**嘉庆年间大规模扩建**，增建宫门与寝殿，成为皇太后、太妃们的居住颐养之地。

绮春园以水景见长，湖面曲折，荷塘连片，素有"**夏赏荷、秋观叶**"之美。更重要的是，园内的**正觉寺**是1860年英法联军焚园之后圆明园中**唯一幸存的古建筑群**——因它独立于园墙之外、驻有喇嘛看守，侥幸躲过了那场大火。

今天的绮春园区域是圆明园**南门（正门）入口**所在，绝大多数游客由此进园。正觉寺内如今设有马首回归展等陈列，是入园后的第一站看点。

## 建筑与景观

### 正觉寺

正觉寺位于绮春园西南部，始建于乾隆三十八年（1773年），是清代**皇室喇嘛庙**（藏传佛教寺院），供满洲、蒙古王公礼佛之用。寺院为汉式布局，中轴线依次有：

- 山门、天王殿
- 三圣殿（正殿）
- 最上楼（二层佛楼）
- 左右配殿、文殊亭等

1860年大火中，圆明园建筑几乎荡然无存，正觉寺因**地处园外、独立成院**而幸免于难，成为圆明园范围内唯一留存至今的古建筑群。2002年起经全面修缮，如今作为圆明园博物馆展厅使用，**马首铜像**即回归供奉于寺内文殊亭中。

### 鉴碧亭与天心水面

绮春园水网密布，荷塘众多，是圆明园最著名的**赏荷佳处**：

- **鉴碧亭**：临水而建的方亭，取"以水鉴碧"之意，夏日凭栏可赏满池红莲
- **天心水面**：位于湖心位置的临水敞轩，"天在心、水在面"，是静观荷风月色的绝佳处
- **凤麟洲**：湖中小岛，形如凤麟，岛上有殿，为太后颐养之所

每年6-8月，绮春园一带荷塘接天莲叶、映日荷花，是圆明园荷花节的中心展区。

### 涵秋馆

**涵秋馆**位于绮春园西北部，取"涵纳秋光"之意，是清代帝后秋季游赏之所。馆区依山面水，秋日层林尽染时最为动人。如今遗址上仍可见基址与山形，秋来银杏黄、枫叶红，别有一番萧散之美。

### 绮春园宫门

嘉庆扩建时增建的绮春园宫门，是太后驻跸时的正门，规制严整。如今的圆明园南门即设于绮春园区域，宫门遗址就在入口附近，入园时可留意遗址标识。

## 历史故事

### 太后们的"绮春岁月"

嘉庆朝以后，绮春园成为皇太后、太妃的颐养之地。道光年间，孝和睿皇后长期居此；咸丰朝的康慈皇贵太妃亦居绮春园。深宫妇人们在这座水木清华的园子里度过晚年——亭台之间，曾有多少无人知晓的宫廷往事。

1860年大火起时，园中内眷仓皇随咸丰帝北逃热河，此后再未归来。绮春园的太后时代，随那场大火一同终结。

### 正觉寺的"幸存之谜"

为何满园皆焚，独正觉寺无恙？因正觉寺虽属圆明园范围，却**筑有独立围墙、另设山门**，且常年有喇嘛住持看守。英法联军焚掠集中于园内主体区域，这座偏居一隅的小寺被忽略，竟成圆明园地面古建筑的唯一孑遗。

今天站在正觉寺红墙之内，墙外是百年废墟，墙内是劫后梵宇——一墙之隔，恍若两个时代。2019年回归的马首铜像安奉于此，更让这座劫后寺院成为"国宝回家"的象征之地。

## 必看亮点

1. **正觉寺古建筑群**：圆明园唯一幸存的清代建筑，红墙黄瓦与园中废墟形成强烈对比
2. **马首铜像回归展**：寺内文殊亭，目前唯一回归圆明园本址的兽首真品
3. **绮春园荷塘**：6-8月赏荷最佳处，鉴碧亭、天心水面均可凭栏观荷
4. **绮春园宫门遗址**：嘉庆朝扩建的园门基址，可辨识当年的规制
5. **涵秋馆秋色**：10月下旬至11月初，银杏与红枫交织的遗址秋景
6. **凤麟洲小岛**：湖心岛影，昔日太后颐养之所的所在

## 参观贴士

- 绮春园区域紧邻圆明园**南门**（地铁4号线圆明园站最近入口），建议作为进园第一站
- 正觉寺（含马首展）随大门票入内即可；周一展览可能调整，行前可致电010-62543673确认
- **赏荷季**（6月中旬-8月中旬）清晨入园最佳，上午9点后游客渐多
- 游览路线建议：南门 → 正觉寺（马首展）→ 鉴碧亭 → 天心水面 → 三园交界 → 福海或西洋楼
- 园内荷花节期间设有摄影点，荷塘边请勿踩踏、采摘
- 正觉寺红墙是园内最佳拍照背景之一，建议避开人流高峰拍摄
- 绮春园与[福海](/landmarks/yuanmingyuan/fuhai)、西洋楼同属南门入园后的顺路区域，一日可串联游完
