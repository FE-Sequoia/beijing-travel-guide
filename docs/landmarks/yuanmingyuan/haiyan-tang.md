# 海晏堂与十二生肖兽首

![海晏堂遗址](https://images.unsplash.com/photo-1598117407983-b3a39ee9747a?w=1200&q=80)

> "海晏"者，河清海晏，天下太平。十二尊兽首曾在这里报时，如今它们散落在世界各地，等待着最后的团圆。

## 简介

海晏堂是长春园[西洋楼建筑群](/landmarks/yuanmingyuan/xiyang-lou)中体量最大的建筑，建成于乾隆二十四年（1759年），取"河清海晏、国泰民安"之意命名。楼体坐东朝西，上下两层十一开间，红砖墙身镶嵌汉白玉石雕，是西洋楼中西洋形式与中国意趣融合的代表。

海晏堂之所以举世闻名，缘于楼前那组**十二生肖兽首人身铜像喷泉**：十二尊铜像分列楼前八字形水池两侧，按十二时辰依次喷水报时，正午时分十二兽首同时喷涌，蔚为奇观。

这组铜像由意大利传教士郎世宁设计、法国传教士蒋友仁监修水力、中国宫廷工匠铸造，是18世纪中西工艺合作的巅峰之作。1860年英法联军劫掠圆明园时，兽首被强行锯断掠走，从此流散海外。一个半世纪以来，兽首的每一次回归都牵动国人之心，它们已成为流失海外中国文物的象征。

## 建筑与景观

### 海晏堂主楼

海晏堂主楼为巴洛克风格的两层楼建筑，面阔十一间，是皇帝宴请外藩、观赏水法的场所。楼前设大型蓄水池（俗称"锡海"），以锡板焊接而成，可容水180余吨，为楼前喷泉群供水。

1860年后主楼焚毁，今存高大的砖石残墙与汉白玉台阶、石栏，仍可辨识当年的恢宏尺度——仅台阶就宽达数十米，拾级而上可想见楼阁之高。

### 十二生肖水力钟

楼前八字形叠落式喷水池两侧，各置六尊**兽首人身铜像**：

- 南岸：鼠、虎、龙、马、猴、狗
- 北岸：牛、兔、蛇、羊、鸡、猪

铜像人身着中式袍服，兽首以写实手法铸成，高约50厘米，重约3-4公斤，为宫廷精铸铜胎。每到一个时辰（两小时），对应兽首即口中喷水；**正午午时，十二兽首同时喷涌**，水柱直冲数尺，声震全园——这是一座以生肖报时的"水力钟"。

## 历史故事

### 三国工匠共铸一钟

十二生肖兽首的设计与制造堪称"国际合作"：意大利人郎世宁依据中国生肖提出造型方案，以西洋写实手法绘制图样；法国人蒋友仁解决供水与水力计时机关；最后由清宫造办处中国工匠以失蜡法铸铜、精细錾刻而成。

兽首的毛发、皱纹纤毫毕现，兼具西洋写实与中国神韵，至今被视为清代宫廷铸造工艺的最高水平。

### 流失与回归

1860年10月，英法联军冲入海晏堂，将十二尊兽首锯下掠走，自此流散四方。此后百余年间，兽首辗转于拍卖行与私人藏家之手。

2000年春，牛、虎、猴三首现身香港拍卖会，保利集团以重金购回，开启兽首回归之路；2013年法国皮诺家族将鼠、兔二首无偿捐赠中国；2019年11月，何鸿燊捐赠的**马首**正式回归圆明园，安奉于[绮春园正觉寺](/landmarks/yuanmingyuan/qichun-yuan)——这是第一尊回归圆明园本址的兽首。

龙首据传存于台湾；蛇、羊、鸡、狗四首至今下落不明。兽首能否全部团圆，仍是几代中国人的牵挂。

## 必看亮点

1. **海晏堂残墙与台阶**：西洋楼最大建筑的遗存，感受当年的宏阔尺度
2. **八字形喷水池遗址**：对照说明牌想象十二兽首依次报时的场景
3. **马首回归展（正觉寺内）**：目前唯一回归圆明园本址的兽首真品，必看
4. **锡海遗址**：楼后巨大的锡制蓄水池遗迹，喷泉系统的"心脏"
5. **兽首复制品与图片展**：园内设有兽首复原模型与历史照片，便于对照想象
6. **汉白玉栏杆雕花**：台阶残存石栏上中西合璧的纹样细节

## 参观贴士

- 海晏堂位于**西洋楼遗址区**内，需另购15元遗址区门票
- **看真品兽首**：马首藏于绮春园正觉寺展厅；鼠、兔首在中国国家博物馆；牛、虎、猴、猪首在保利艺术博物馆
- 建议先读展板上的兽首分布图，再参观残址，历史感更完整
- 游览顺序建议：方外观 → **海晏堂** → [大水法](/landmarks/yuanmingyuan/dashuifa)，沿西洋楼主线东行
- 请勿触摸遗址石构件与展品；展厅内禁用闪光灯
