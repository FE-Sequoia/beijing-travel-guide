# 人民英雄纪念碑

> "人民英雄永垂不朽！"
> —— 毛泽东

## 简介

人民英雄纪念碑位于天安门广场中央，是中华人民共和国政府为纪念中国近现代史上的革命烈士而建造的纪念性建筑。碑高37.94米，碑身正面镌刻着毛泽东题写的"人民英雄永垂不朽"八个大字。

## 历史沿革

### 提议修建

1949年9月30日，中国人民政治协商会议第一届全体会议决定，在首都北京建立人民英雄纪念碑，纪念在人民解放战争和人民革命中牺牲的人民英雄。

### 奠基

1949年9月30日下午，毛泽东主席率领全体政协委员在天安门广场举行奠基典礼，毛泽东亲自执锨铲土，为纪念碑奠基。

### 设计建造

纪念碑的设计方案经过全国征集，最终由著名建筑师梁思成、林徽因等人主持设计。1952年8月正式动工，1958年4月竣工，历时近6年。

## 建筑特色

### 碑身

人民英雄纪念碑由碑身、碑座、台基三部分组成。

| 项目 | 数据 |
|:---|:---|
| **总高度** | 37.94米 |
| **碑身高** | 14.7米 |
| **碑座高** | 13米 |
| **台基高** | 10.24米 |
| **碑身尺寸** | 长14.7米，宽2.9米，厚1米 |
| **重量** | 约1万吨 |

### 碑身正面

碑身正面（北面）镌刻着毛泽东题写的"人民英雄永垂不朽"八个鎏金大字。每个字高2米，宽1.5米，笔力雄健，气势磅礴。

### 碑身背面

碑身背面（南面）是毛泽东撰文、周恩来书写的碑文。碑文共150字，字体为楷书，庄严肃穆。

#### 碑文全文

> "三年以来，在人民解放战争和人民革命中牺牲的人民英雄们永垂不朽！
>
> 三十年以来，在人民解放战争和人民革命中牺牲的人民英雄们永垂不朽！
>
> 由此上溯到一千八百四十年，从那时起，为了反对内外敌人，争取民族独立和人民自由幸福，在历次斗争中牺牲的人民英雄们永垂不朽！"

### 浮雕

纪念碑下层须弥座束腰部镶嵌着十幅汉白玉浮雕，每幅高2米，总长约40米，生动地再现了中国人民一百多年来的革命斗争历程。

| 浮雕 | 内容 | 时间 |
|:---|:---|:---:|
| **销毁鸦片烟** | 虎门销烟 | 1839年 |
| **金田起义** | 太平天国运动 | 1851年 |
| **武昌起义** | 辛亥革命 | 1911年 |
| **五四运动** | 反帝反封建运动 | 1919年 |
| **五卅运动** | 反帝爱国运动 | 1925年 |
| **南昌起义** | 中国共产党武装起义 | 1927年 |
| **抗日游击战** | 敌后抗日斗争 | 1937-1945年 |
| **胜利渡长江** | 解放战争 | 1949年 |
| **支援前线** | 人民支援解放战争 | 1946-1949年 |
| **欢迎解放军** | 人民欢迎人民解放军 | 1949年 |

### 碑座装饰

碑座上下两层须弥座，上层小，下层大。座上刻有荷花、菊花、牡丹等花卉图案，以及云纹、龙纹等装饰。

### 建筑材料

纪念碑采用青岛浮山花岗岩建造，这种花岗岩质地坚硬，色泽美观，耐腐蚀。

## 设计团队

### 建筑师

- **梁思成**：著名建筑学家，主持总体设计
- **林徽因**：著名建筑师、诗人，参与设计

### 浮雕作者

十幅浮雕由刘开渠、滑田友、萧传玖、曾竹韶等著名雕塑家创作，是中国现代雕塑艺术的杰作。

3. **观赏浮雕**：浮雕生动，值得细细观赏

5. **结合游览**：可与天安门广场、故宫一起游览

## 文化意义

人民英雄纪念碑是中国人民革命精神的象征，是中华民族精神的丰碑。

正如碑文所说：
> "人民英雄永垂不朽！"

纪念碑铭记着为民族独立和人民解放而英勇牺牲的革命先烈，激励着后人继承先烈遗志，为实现中华民族伟大复兴而奋斗。
