# 望京石与历史遗迹

![望京石与历史遗迹](https://images.unsplash.com/photo-1715905601923-e84750c8d01d?w=1200&q=80)

> 一块巨石，一道摩崖，一条"人"字形铁路——八达岭脚下，藏着长城之外的另一部近代史。

## 简介

在八达岭[关城](/landmarks/badaling/guancheng)东侧的山崖之间，卧着一块天然巨石，名曰**望京石**。石身浑厚，登之可远眺东南方向——那里，是一百多里外的北京城。巨石周围的山崖上，还散落着"天险""地利"等摩崖题刻，字迹历经风雨而筋骨犹存，与巍峨的关城互为表里。

如果说关城与[敌楼](/landmarks/badaling/dilou)写的是明代边防，那么望京石与八达岭山脚的遗迹，写的则是晚清以降的近代中国：慈禧西逃的仓皇背影、詹天佑与中国人自建的第一条铁路，都发生在这片山谷之中。

登城归来路过此处，不妨放慢脚步——长城的故事到关城为止，而八达岭的故事，还一直延伸到了山脚下那条百年铁轨上。

### 望京石与摩崖题刻

**望京石**本身并无雕琢，胜在其位与其事：它恰在出关西行必经之路旁，登石回望，关山阻隔之外的京师方向隐约可辨。千百年来，戍卒商旅经此出塞，回望故土的最后一点高处，往往就是这块石头。

崖壁上的**"天险"**二字摩崖，则是对八达岭地势最精炼的概括——两山夹峙、一夫当关，古人把对这片山河的判断，直接刻进了石头里。与之相伴的"地利"等题刻，同属明清时期往来官员、文人的手笔，是题刻者的"到此一游"，也是后人的历史路标。

石与字都未经精细雕琢，粗粝中带着边关气质——比起城楼的规整，这些野性的刻痕更接近八达岭作为军事要塞的本来面目。

### 青龙桥"人"字形线路

山脚下的**青龙桥"人"字形线路**是另一处必看奇观：京张铁路穿越八达岭时坡度极陡，詹天佑在此设计出"人"字形折返线——列车北上至青龙桥后，改换机车方向，先向东北、再折向西北，如人字一撇一捺爬上山岭，以"距离换高度"化解了越岭难题。

爬坡时更有巧思：两台机车一前一后，一台牵引、一台助推，行至折返点再互换角色。这是中国铁路工程史上教科书级的一笔，也是百年前中国工程师向世界交出的答卷。

如今，"人"字形线路仍是老京张线（京包铁路）的一部分，青龙桥车站仍在运营——百年设计至今服役，本身就是对詹天佑最好的纪念。

## 历史故事

### 慈禧登石望京

1900年（庚子年），八国联军攻入北京，慈禧太后携光绪帝微服出逃，沿关沟古道西奔西安。相传銮驾行至八达岭关城之侧，太后登上这块巨石，回望烟尘中的京师，良久不语——望京石因此得名。

此说虽为民间流传，但另一重功能同样可信：此石地处要冲、视野开阔，历来是兵家登临瞭望京师方向、观察关沟动静的望所。一块石头，叠印着家国之痛与兵家之智。

耐人寻味的是，庚子之变后不久，詹天佑便率队来到同一片山谷勘测铁路——望京石目送的，正是一个旧时代仓皇谢幕、新时代踉跄登场的转折时刻。

### 詹天佑与"人"字形线路

1905年，京张铁路开工。这条从北京通往张家口的铁路，要穿越八达岭的崇山峻岭，英俄工程师曾断言中国人无力自建。总工程师**詹天佑**踏勘山势，在青龙桥创造性地铺设"人"字形折返线，并配合两台机车一拉一推翻越山岭，使线路成功越岭。

1909年，京张铁路提前两年建成通车，是中国第一条完全由中国人自行设计、施工的干线铁路。青龙桥车站旁，詹天佑铜像至今目送列车穿山而过；老站房、旧铁轨与"人"字形道岔原址犹存，是全国重点文物保护单位。

### 百年京张的对照

老京张铁路通车110年后的2019年，**京张高铁**建成通车，八达岭长城站深埋地下102米，恰从青龙桥"人"字形线路下方穿越——从时速35公里到350公里，从"人"字形折返到一洞穿山，两条铁路在山谷间完成了一场跨越百年的对话。詹天佑当年的夙愿，以他未必想象得到的方式圆了。

## 必看亮点

- **望京石**：登石东南远眺，体会古人"望京"的视角与心境。
- **"天险""地利"摩崖**：关城崖壁上的大字题刻，是八达岭地势的千年评语。
- **青龙桥"人"字形线路**：在八达岭脚下看铁路如何"撇捺登山"，詹天佑铜像与老站房就在现场。
- **詹天佑纪念馆**：免费开放，系统陈列京张铁路的勘测、建设与通车史料。
- **百年对照**：老京张与新京张同谷并存，是中国铁路史最直观的现场教材。
- **关沟古道遗韵**：望京石所在一线即昔日车马出塞的古道，缓步其间可想见当年情景。
- **题刻寻字**：除"天险"外，崖壁间还散有"地利"等题刻，值得放慢脚步细寻。

## 参观贴士

- **位置顺路**：望京石与摩崖题刻就在关城旁，下城后经[关城](/landmarks/badaling/guancheng)步行即达，无需专程绕路。
- **青龙桥顺游**：青龙桥及詹天佑纪念馆位于八达岭山脚，免费开放，可与长城组成一日行程；自驾或乘S2线市郊铁路沿线而来均可。
- **别错过题刻**："天险"摩崖就在路旁崖壁，抬头即见，许多人爬完长城竟未留意。
- **拍照时机**：崖壁题刻侧光时立体感最强，上午或傍晚拍摄效果最佳。
- **S2线车上看**：乘S2线市郊铁路往来八达岭，列车行驶在老京张线上，车窗外即是百年铁路遗迹，本身就是流动的风景。
- **护石勿刻**：望京石与摩崖题刻均为历史遗存，切勿刻画攀踏石面。
- **组合建议**："登城 → 望京石 → 博物馆"半天即可；加上青龙桥则需安排一整天。
- **带望远镜**：想看清山脊上的墩台残迹或"人"字形线路全貌，小型望远镜很有帮助。
- **行程收尾**：访罢望京石，可回到关城脚下的[中国长城博物馆](/landmarks/badaling/changcheng-museum)为一日之行收尾。
