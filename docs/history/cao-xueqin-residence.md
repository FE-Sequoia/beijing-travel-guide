## 简介

曹雪芹故居纪念馆位于北京市东城区广渠门内大街207号（蒜市口街16号），是伟大文学家曹雪芹晚年居住的地方。在这里，曹雪芹经历了从锦衣玉食到举家食粥的巨变，以"字字看来皆是血，十年辛苦不寻常"的精神，创作了中国古典小说的巅峰之作《红楼梦》。

## 历史沿革

### 曹家兴衰

曹雪芹（约1715-1763），名霑，字梦阮，号雪芹。其家族为清代内务府正白旗包衣出身，曾祖曹玺、祖父曹寅、父曹頫三代世袭江宁织造，家世显赫。康熙南巡时，曹家曾四次接驾，富贵至极。

#### 家道中落

| 时间 | 事件 |
|:---|:---|
| 1728年 | 曹頫获罪抄家，举家迁回北京 |
| 1730年代 | 曹雪芹少年时期，家道逐渐中落 |
| 1740年代 | 曹雪芹居住于蒜市口，开始创作《红楼梦》 |
| 1763年 | 曹雪芹病逝于北京西山 |

### 蒜市口故居

乾隆年间，曹雪芹家从南京迁回北京后，居于崇文门外蒜市口一带。据考证，广渠门内大街207号院即为曹雪芹晚年居住地之一。此处靠近崇文门，交通便利，周边商业繁荣，是当年普通旗人居住的区域。

## 故居布局

### 院落结构

曹雪芹故居是一座典型的老北京四合院，虽然规模不大，但布局规整，体现了清代普通旗人的居住特点。

#### 建筑布局

| 建筑 | 功能 | 特色 |
|:---|:---|:---|
| **大门** | 入口 | 传统如意门 |
| **影壁** | 照壁 | 砖雕影壁 |
| **正房** | 展室 | 曹雪芹生平展 |
| **东厢房** | 展室 | 《红楼梦》创作展 |
| **西厢房** | 展室 | 曹家兴衰展 |
| **后院** | 庭院 | 复原清代民居场景 |

### 主要展室

#### 曹雪芹生平展（正房）

展示曹雪芹从锦衣玉食的公子到举家食粥的寒士，最终在西山著书的人生历程。

- **江宁织造时期**：曹家显赫的江南岁月
- **北京落魄时期**：家道中落后的北京生活
- **西山著书时期**：创作《红楼梦》的艰苦岁月

#### 《红楼梦》创作展（东厢房）

展示《红楼梦》的创作背景、版本流传以及研究情况。

- **创作背景**：从生活到艺术的升华
- **早期抄本**：甲戌本、己卯本、庚辰本等珍贵抄本介绍
- **研究历史**：红学发展的百年历程

#### 曹家兴衰展（西厢房）

展示曹家从显赫到衰落的历史，以及这段经历对曹雪芹创作的影响。

## 《红楼梦》与北京

### 创作背景

《红楼梦》虽然以江宁织造曹家的生活为原型，但曹雪芹在北京的生活经历对创作产生了深远影响。书中描写的贵族生活、人情冷暖、世态炎凉，无不渗透着他在北京的生活感悟。

### 北京元素

| 书中描写 | 北京元素 |
|:---|:---|
| **荣国府、宁国府** | 北京王府建筑 |
| **大观园** | 北京园林艺术 |
| **节庆风俗** | 北京民俗风情 |
| **饮食服饰** | 清代北京生活 |

### "红楼"文化

《红楼梦》不仅是一部文学巨著，更是一部清代社会的百科全书。从建筑园林到饮食服饰，从诗词歌赋到医药养生，无所不包。

## 文化意义

曹雪芹是中国文学史上最伟大的作家之一，《红楼梦》与《三国演义》《水浒传》《西游记》并称中国四大古典名著，且被公认为艺术成就最高。

"开谈不说《红楼梦》，读尽诗书也枉然。"《红楼梦》不仅在文学上达到了巅峰，更深刻影响了中国的文化、艺术和思想。

### 红学传承

自《红楼梦》问世以来，研究《红楼梦》的学问被称为"红学"，成为一门专门的学科。从脂砚斋到俞平伯、周汝昌，一代代学者前赴后继，探寻这部巨著的秘密。
