# 颐和园

> "十里青山行画里，双飞白鸟似江南。"
> —— 明代诗人 王绂

## 简介

颐和园是中国清朝时期皇家园林，坐落在北京西郊，距城区15公里，占地约290公顷，与圆明园毗邻。它是以昆明湖、万寿山为基址，以杭州西湖为蓝本，汲取江南园林的设计手法而建成的一座大型山水园林，也是保存最完整的一座皇家行宫御苑，被誉为"皇家园林博物馆"。

## 历史沿革

### 始建

颐和园前身为清漪园，始建于乾隆十五年（1750年）。乾隆皇帝为庆祝其母崇庆皇太后六十大寿，在此修建清漪园，工程历时15年，耗银480万两。

### 被毁与重建

咸丰十年（1860年），英法联军入侵北京，清漪园被焚毁。光绪十二年（1886年），慈禧太后挪用海军军费重修此园，取"颐养冲和"之意，改名"颐和园"，作为晚年的颐养之所。

### 劫难与修复

光绪二十六年（1900年），八国联军入侵北京，颐和园再遭洗劫。解放后，政府多次对颐和园进行修缮，1961年被列为全国重点文物保护单位，1998年被列入《世界遗产名录》。

## 建筑布局

颐和园由万寿山和昆明湖组成，大致可分为宫廷区、前山区、后山区、湖区四大部分。

### 宫廷区

#### 仁寿殿

仁寿殿是颐和园的政治活动区，慈禧太后和光绪皇帝曾在此处理朝政。殿前陈列着龙、凤、麒麟、太平有象等铜兽，象征着皇家的威仪。

#### 德和园

德和园是清代三大戏楼之一，与紫禁城畅音阁、承德避暑山庄清音阁齐名。戏楼共三层，可同时演出上天入地的神话戏剧。

### 前山区

#### 佛香阁

佛香阁是颐和园的标志性建筑，高41米，八面三层四重檐，是颐和园全园的最高点。阁内供奉着一尊明代铜铸千手观音像。

#### 长廊

长廊全长728米，共273间，是中国古典园林中最长的游廊。廊内绘有14000余幅彩画，内容涵盖山水、花鸟、人物、故事等，被誉为"世界第一长廊"。

#### 排云殿

排云殿是慈禧太后举行寿庆活动的地方，位于万寿山前山中央。殿名取"神仙排云出，但见金银台"之意。

### 湖区

#### 昆明湖

昆明湖面积220公顷，占全园四分之三。湖中有南湖岛、藻鉴堂、治镜阁三座岛屿，象征蓬莱、方丈、瀛洲三座仙山。

#### 十七孔桥

十七孔桥连接东堤和南湖岛，长150米，宽8米，是园内最大的石桥。桥上有形态各异的石狮544只，比卢沟桥还多。冬至前后，夕阳穿过桥洞，形成"金光穿洞"的奇观。

#### 西堤六桥

仿杭州西湖苏堤而建，包括界湖桥、豳风桥、玉带桥、镜桥、练桥、柳桥六座桥，桥桥有景，景景不同。

### 后山区

#### 苏州街

苏州街又称"买卖街"，是仿照江南水街建造的皇家商业街。乾隆皇帝曾在此体验民间购物的乐趣。

#### 谐趣园

谐趣园仿无锡寄畅园而建，是颐和园中独立的江南园林，小巧精致，被称为"园中之园"。

## 文化故事

### 慈禧与海军军费

光绪年间，慈禧太后为修建颐和园，挪用海军军费约3000万两白银。这笔钱原本用于购买军舰、建设北洋水师。有说法认为，这笔钱若用于海军建设，甲午海战的结果或许会有所不同。

### 石舫的故事

清晏舫俗称"石舫"，位于昆明湖西北岸，是用巨石雕成的船形建筑。乾隆皇帝修建此舫，寓意"水能载舟，亦能覆舟"，告诫子孙要勤政爱民。

### 知鱼桥的典故

谐趣园内的知鱼桥取自《庄子·秋水》中庄子与惠子"濠梁之辩"的故事，体现了中国古典哲学中"知鱼之乐"的智慧。

## 游览攻略

### 推荐路线

**北宫门入园路线（约4-5小时）**：

北宫门 → 苏州街 → 须弥灵境 → 香岩宗印之阁 → 谐趣园 → 德和园 → 仁寿殿 → 玉澜堂 → 宜芸馆 → 乐寿堂 → 长廊 → 排云殿 → 佛香阁 → 石舫 → 乘船至南湖岛 → 十七孔桥 → 新建宫门

**东宫门入园路线（约4-5小时）**：

东宫门 → 仁寿殿 → 德和园 → 玉澜堂 → 乐寿堂 → 长廊 → 排云殿 → 佛香阁 → 石舫 → 乘船 → 南湖岛 → 十七孔桥 → 铜牛 → 文昌院 → 东宫门

### 实用贴士

1. **联票推荐**：佛香阁、德和园、文昌院是精华景点，建议购买联票
2. **游船体验**：昆明湖游船（大船40元，小船60-80元/小时）是不错的体验
3. **荷花观赏**：夏季（7-8月）昆明湖荷花盛开，非常壮观
4. **最佳拍摄点**：佛香阁全景（南湖岛）、十七孔桥日落（东堤）
5. **避开人流**：早上7:00-8:00或下午4:00后入园，避开旅行团
