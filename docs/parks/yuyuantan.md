# 玉渊潭公园

> "北京最大樱花观赏地，电视塔倒影如画。"

## 简介

玉渊潭公园位于北京市海淀区，是北京最大的樱花观赏地，也是北京著名的城市公园。公园占地约137公顷，其中水面约60公顷，是北京三环内最大的公园。

## 历史沿革

### 辽金时期

玉渊潭地区在辽金时期就是著名的风景区，"玉渊潭"之名始见于金代。

### 清代行宫

清代，乾隆皇帝在此修建行宫，是皇帝游幸之地。

### 现代公园

1960年，玉渊潭正式辟为公园，对外开放。

## 园区布局

玉渊潭公园分为东湖、西湖、南湖三个湖区。

### 东湖

东湖是玉渊潭公园的主要景区，有樱花园、湿地园等。

#### 樱花园

樱花园位于东湖湖畔，是北京最大的樱花园。

| 项目 | 数据 |
|:---|:---|
| **面积** | 约25公顷 |
| **樱花品种** | 近30个品种 |
| **樱花数量** | 近3000株 |
| **赏樱时间** | 3月下旬至4月中旬 |

#### 樱花品种

| 品种 | 花色 | 花期 |
|:---|:---:|:---:|
| **染井吉野** | 淡粉色 | 3月下旬 |
| **关山** | 粉红色 | 4月上旬 |
| **普贤象** | 淡粉色 | 4月中旬 |
| **松月** | 白色 | 4月中旬 |
| **大山樱** | 深粉色 | 4月上旬 |

### 西湖

西湖有游船码头，可以划船游湖。

### 南湖

南湖人较少，比较安静。

## 樱花节

玉渊潭公园每年春季举办樱花节，是北京著名的花事活动。

### 举办时间

每年3月下旬至4月中旬，根据花期调整。

### 活动内容

- **赏樱**：观赏各种樱花
- **摄影比赛**：樱花摄影比赛
- **文化活动**：樱花文化展览
- **美食**：樱花主题美食

### 门票

樱花节期间门票10元（平时2元）。

## 电视塔倒影

玉渊潭公园的标志性景观是中央电视塔的倒影。

### 最佳拍摄点

- **东湖湖畔**：可以拍到电视塔在湖中的倒影
- **傍晚时分**：夕阳西下，景色最美

### 拍摄建议

天气晴好时，湖面平静，倒影清晰，是拍摄的最佳时机。

## 游览攻略

### 推荐路线

**经典路线（约2-3小时）**：

南门 → 东湖樱花园 → 西湖游船 → 电视塔倒影 → 北门

**樱花节路线**：

西门 → 樱花园（赏樱） → 东湖湖畔 → 南门

### 交通方式

| 方式 | 站点 |
|:---|:---|
| 地铁 | 1号线/9号线军事博物馆站A口 |
| 公交 | 1路、32路、65路等玉渊潭南门站 |
| 步行 | 从军事博物馆步行约10分钟 |

### 实用贴士

1. **樱花最佳时间**：3月下旬至4月中旬
2. **避开人流**：早上7:00前或下午4:00后人较少
3. **樱花节门票**：樱花节期间门票10元
4. **划船体验**：可以划船游湖
5. **拍照建议**：下午3:00-5:00光线最佳
6. **结合游览**：可与军事博物馆、首博一起游览
