# 中国人民革命军事博物馆

> "了解中国军事历史的最佳场所。"

## 简介

中国人民革命军事博物馆（简称军博）位于北京市海淀区，是中国唯一的大型综合性军事博物馆，也是中国第一个综合类军事博物馆。军博筹建于1958年，1960年正式对外开放。

## 历史沿革

### 筹建

1958年，为庆祝中华人民共和国建国10周年，中共中央决定筹建军事博物馆。1958年9月，军事博物馆正式破土动工。

### 开放

1960年8月，军事博物馆正式对外开放，成为当时全国最大的博物馆。

### 改扩建

2012年至2017年，军事博物馆进行了大规模改扩建，2017年7月重新对外开放。新馆建筑面积从原先的6万平方米扩大到15.3万平方米。

## 建筑特色

### 外观

军事博物馆建筑气势恢宏，正门上方悬挂着毛泽东主席题写的"中国人民革命军事博物馆"匾额。

| 项目 | 数据 |
|:---|:---|
| **建筑面积** | 15.3万平方米 |
| **陈列面积** | 近6万平方米 |
| **主楼高度** | 94.7米 |
| **结构** | 塔座式建筑 |

### 主楼

主楼采用塔座式建筑，顶部是解放军军徽，高达94.7米，是北京的标志性建筑之一。

## 常设展览

### 中国共产党领导的革命战争陈列

这是军博的基本陈列，全面展示中国共产党领导人民军队进行的革命战争。

#### 展示内容

- **土地革命战争**：南昌起义、秋收起义、红军长征等
- **抗日战争**：敌后抗战、重大战役等
- **解放战争**：三大战役、渡江战役等
- **抗美援朝战争**：抗美援朝战争全过程

### 兵器陈列

兵器陈列是军博的特色展览，展示各时期兵器装备。

| 展区 | 内容 |
|:---|:---|
| **古代战争馆** | 中国古代兵器 |
| **近代战争馆** | 中国近代兵器 |
| **土地革命战争馆** | 红军时期的兵器 |
| **抗日战争馆** | 抗战时期的兵器 |
| **解放战争馆** | 解放战争时期的兵器 |
| **兵器馆** | 各种武器装备 |

### 大型兵器装备

军博广场上陈列着大型兵器装备，是军事爱好者的最爱。

| 装备 | 说明 |
|:---|:---|
| **轰-6轰炸机** | 中国第一代轰炸机 |
| **东风-1导弹** | 中国第一枚导弹 |
| **坦克** | 各型坦克 |
| **火炮** | 各型火炮 |
| **舰艇** | 导弹艇等 |

### 军事科技陈列

展示中国军事科技的发展历程，包括导弹、航天、核武器等。
