# 午门

![午门](https://images.unsplash.com/photo-1757510146664-48b437cddd1a?w=1200&q=80)

> 五凤翼然，九门洞开。这里是紫禁城的正门，也是帝国威仪的第一道门槛。

## 简介

午门是紫禁城的正门，位于北京城中轴线的南端，因坐北朝南、居中向阳而得名"午"。它是故宫四座城门中最为宏伟的一座，也是整座皇宫的礼仪门户——一切国家大典的仪仗从这里开始，一切凯旋献俘的盛典在这里落幕。

午门建成于明永乐十八年（1420年），清顺治四年（1647年）重修。整座建筑由城台和城楼组成，城台高12米，正中开三门，两侧各开一掖门，形成"明三暗五"的格局。城台上矗立着五座重檐亭式楼阁，中央主楼面阔九间，两侧各两座方亭以廊庑相连，形如凤凰展翅，故又称"五凤楼"。

今天，午门是故宫博物院的唯一入口。穿过那幽深的门洞，六百年的皇家气象便在眼前铺展开来。

## 建筑与景观

| 项目 | 数据 |
|:---|:---|
| **通高** | 37.95米（故宫最高的建筑之一） |
| **城台高度** | 12米 |
| **门洞格局** | 明三暗五（正面三门，背面加两掖门共五门） |
| **城楼形制** | 五座重檐攒尖顶楼阁，俗称"五凤楼" |
| **主楼面阔** | 九间 |
| **两翼** | 左右各建廊庑十三间，环抱如阙 |

午门的平面设计极具巧思：城台向两侧延伸，环抱成"凹"字形，形成三面合围的广场空间。这种形制源自唐宋以来宫门"双阙"的传统，既增强了防御功能，又营造出强烈的礼仪围合感——当百官步入这片广场仰望五凤楼时，皇权的压迫感油然而生。

"明三暗五"的门洞各有严格的等级规定：**正中门洞专供皇帝通行**；皇帝大婚时，皇后的凤轿可从中门入宫，一生仅此一次；殿试揭晓后，状元、榜眼、探花三人可从中门出宫，亦仅此一次。此外，宗室王公走东侧门，文武大臣走西侧门，两掖门仅在大型朝会时按文东武西开启。一道门洞，便是帝国等级的缩影。

## 历史故事

### "推出午门斩首"——一个流传百年的误传

"推出午门斩首"是戏曲小说中最经典的台词之一，但历史上，午门从不是刑场。明代午门确有一项令人闻之色变的刑罚——**廷杖**：大臣触怒皇帝，便被拖到午门外当众杖责。正德十四年（1519年），群臣谏阻明武宗南巡，一百四十六人受廷杖，当场打死十一人。廷杖之酷烈，让"午门"在士大夫心中蒙上了血腥色彩，久而久之便讹传为"斩首"之地。

### 献俘礼：帝国的高光时刻

午门最庄严的功能是"献俘礼"。每逢大军凯旋，皇帝登临午门城楼，文武百官侍立两侧，战俘被押至城下献于御前。清乾隆年间平定准噶尔、大小和卓之乱后，都曾在此举行盛大的献俘仪式。彼时五凤楼上鼓乐齐鸣，城下三军山呼万岁，那是帝国武功的巅峰展示。

## 必看亮点

- **五凤楼全景**：从午门广场南侧回望，五座楼阁翼然展开，是紫禁城最具仪式感的画面。
- **中门门洞**：留意正中门洞上方的城楼——那是只有皇帝、皇后（一次）和状元（一次）才能通行的"御道"所在。
- **凹字形广场**：感受三面城台环抱的空间张力，体会古代"阙"制建筑的遗韵。
- **门钉与铺首**：朱红大门上纵横各九路金色门钉（共81颗），是最高等级的标志。

## 参观贴士

- **最佳拍摄时间**：早晨8:30开馆时人最少，站在广场南端可拍到无人的午门全景；傍晚逆光下五凤楼的剪影也极为壮观。
- **避开高峰**：上午10点至下午2点游客最密集，入口排队可能超过一小时，尽量赶早或赶晚。
- **注意朝向**：午门是故宫唯一入口，只能从南侧进入；参观结束后无法从午门出（神武门或东华门出）。
- **门洞里的温度**：厚重的城门洞内冬暖夏凉，夏天穿越门洞时会有明显的凉意。
