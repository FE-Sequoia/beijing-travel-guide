# 火神庙

> "什刹海边的千年古庙，火神庇佑之地。"

## 简介

火神庙位于北京市西城区什刹海边，是北京历史最悠久的火神庙。火神庙始建于唐贞观六年（632年），距今已有1400余年历史。

## 历史沿革

### 始建

火神庙始建于唐贞观六年（632年），是北京最早的火神庙。

### 历代修缮

- **元代**：元世祖忽必烈曾到此进香
- **明代**：明万历年间重修
- **清代**：清乾隆年间重修，规模扩大
- **现代**：2008年修复后对外开放

## 建筑布局

### 山门

山门是火神庙的入口。

### 灵官殿

灵官殿供奉道教护法神王灵官。

### 火神殿

火神殿是火神庙的主殿，供奉火神。

#### 火神

火神是道教尊神，掌管火，是民间信仰的重要神祇。

### 慈航殿

慈航殿供奉慈航道人（观音菩萨的道教化身）。

### 真武殿

真武殿供奉真武大帝。

## 什刹海

火神庙位于什刹海边，可以欣赏什刹海的美景。

### 什刹海

什刹海是北京著名的风景区，由前海、后海、西海组成。

### 荷花市场

夏季，什刹海有荷花市场，非常热闹。

## 游览攻略

### 推荐路线

**寺庙参观（约30分钟）**：

山门 → 灵官殿 → 火神殿 → 慈航殿 → 真武殿 → 出庙

**结合什刹海路线**：

火神庙 → 什刹海 → 银锭桥 → 烟袋斜街 → 钟鼓楼
