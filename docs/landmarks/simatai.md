# 司马台长城

> "中国长城之最，原始风貌的绝佳体验。"

## 简介

司马台长城位于北京市密云区，距北京城区120公里，是万里长城中最险要的一段，被誉为"中国长城之最"。司马台长城以"惊、险、奇"著称，保留了明长城的原始风貌。

## 历史沿革

### 始建

司马台长城始建于明洪武年间（1368-1398年），由大将戚继光主持修建。戚继光在任蓟镇总兵期间，对长城进行了大规模整修和加固。

### 建筑特点

司马台长城充分体现了戚继光的军事防御思想，敌楼设计巧妙，城墙修建坚固。历经400余年风雨，司马台长城保存相对完整，保留了明长城的原始风貌。

## 建筑特色

### 惊

司马台长城依山势而建，最险处坡度达85度，几近垂直。登城如同登天梯，惊心动魄。

### 险

城墙建于悬崖峭壁之上，两侧都是深谷，走在这座城墙上需要极大勇气。

### 奇

司马台长城的敌楼设计独特，有单眼楼、双眼楼、三眼楼、四眼楼、五眼楼之分，形式多样，堪称长城敌楼博物馆。

## 游览区域

### 东段（推荐）

东段是司马台长城的精华所在，也是最险峻的部分。从1号敌楼至10号敌楼，沿途风景壮丽，惊险刺激。

#### 仙女楼

仙女楼是司马台长城的标志性建筑，位于海拔约500米的悬崖之上，造型优美，传说有仙女在此居住。

#### 望京楼

望京楼是司马台长城的最高点，海拔986米。天气晴好时，站在望京楼上可以远眺北京城，故名"望京"。

### 西段

西段相对平缓，适合体力一般的游客。可乘坐索道上山，游览部分敌楼后原路返回。

## 夜游长城

司马台长城是全国唯一开放夜游的长城。夜晚登城，提灯而行，别有一番风味。

### 夜游亮点

- **提灯夜游**：提灯漫步长城，体验古人守边关的感觉
- **星空观赏**：远离城市光污染，仰望星空
- **古北水镇**：夜游后可到古北水镇泡温泉

### 夜游时间

夜游长城开放时间为每年4月至10月，具体时间以景区公告为准。

## 游览攻略

### 推荐路线

**日游路线（约4-5小时）**：

乘索道至5号敌楼 → 游览东段至10号敌楼 → 返回5号敌楼 → 乘索道下山

**夜游路线（约2-3小时）**：

乘索道上山 → 游览部分敌楼 → 观星赏夜景 → 乘索道下山 → 古北水镇泡温泉

### 交通方式

| 方式 | 路线 | 时间 | 费用 |
|:---|:---|:---:|:---:|
| 旅游专线 | 东直门 → 司马台 | 约2.5小时 | 约50元 |
| 自驾 | 京承高速 → 司马台出口 | 约2小时 | - |
| 包车 | 市区往返 | 约2小时 | 600-800元 |

### 实用贴士

1. **体力要求**：司马台长城较险，需要较好体力
2. **夜游准备**：夜游需带外套，山顶风大
3. **索道时间**：注意索道末班车时间
4. **结合游玩**：建议与古北水镇一起游览
5. **安全注意**：部分路段较险，注意安全

## 古北水镇

司马台长城脚下的古北水镇是近年来新建的仿古建筑群，集观光、休闲、度假于一体。

### 水镇特色

- **仿古建筑**：仿照明清风格建造
- **温泉**：地下温泉资源丰富
- **夜景**：夜景灯光秀
- **美食**：北方特色小吃

### 推荐行程

**两天一夜**：
- Day1：下午抵达古北水镇 → 游览水镇 → 夜游司马台长城 → 温泉
- Day2：日游司马台长城 → 午餐后返程
