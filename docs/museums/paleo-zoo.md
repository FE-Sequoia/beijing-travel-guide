# 中国古动物馆

> "亚洲最大的古动物博物馆，恐龙迷的天堂。"

## 简介

中国古动物馆位于北京市西城区，是中国第一座以古生物化石为载体的国家级自然科学类专题博物馆，也是亚洲最大的古动物博物馆。博物馆隶属于中国科学院古脊椎动物与古人类研究所，收藏了大量珍贵的古生物化石。

## 历史沿革

### 创建

中国古动物馆创建于1994年，依托中国科学院古脊椎动物与古人类研究所丰富的馆藏建立。

### 发展

1995年，古动物馆正式对外开放。此后，博物馆不断扩大规模，增加展览内容。

## 建筑特色

### 外观

中国古动物馆建筑外形似一个巨大的恐龙蛋，寓意生命的起源。

### 内部

博物馆内部有多个展厅，布局合理，参观路线清晰。

## 常设展览

### 古脊椎动物展厅

这是古动物馆的核心展厅，展示了从鱼到人演化过程中的各种古脊椎动物。

#### 展览内容

| 展区 | 内容 |
|:---|:---|
| **古鱼类** | 各种古代鱼类化石 |
| **古两栖类** | 古代两栖动物化石 |
| **古爬行类** | 恐龙、翼龙等化石 |
| **古鸟类** | 古代鸟类化石 |
| **古哺乳类** | 古代哺乳动物化石 |
| **古人类** | 古人类化石 |

### 恐龙展厅

恐龙展厅是古动物馆最受欢迎的展厅。

#### 恐龙化石

| 恐龙 | 说明 |
|:---|:---|
| **许氏禄丰龙** | 中国第一具装架的恐龙化石 |
| **马门溪龙** | 中国最大的恐龙之一 |
| **沱江龙** | 剑龙类恐龙 |
| **霸王龙** | 肉食性恐龙 |
| **三角龙** | 草食性恐龙 |
| **鸭嘴龙** | 草食性恐龙 |

#### 恐龙装架

展厅内有多个恐龙装架，包括：

- **马门溪龙装架**：长26米，高8米
- **霸王龙装架**：威猛霸气
- **三角龙装架**：三只角非常醒目

### 古人类展厅

古人类展厅展示了人类演化的历程。

#### 展览内容

- **古猿**：人类的祖先
- **直立人**：北京猿人、元谋人、蓝田人等
- **早期智人**：尼安德特人等
- **晚期智人**：现代人的直接祖先

#### 北京猿人

展厅内有北京猿人的头盖骨模型（原件在抗战中失踪）。

### 特展厅

特展厅定期举办各种特展，包括：

- 新发现的化石展览
- 国际交流展览
- 科普教育展览

## 科研价值

中国古动物馆依托中国科学院古脊椎动物与古人类研究所，具有极高的科研价值。

### 科研成果

- **古鱼类研究**：世界领先
- **恐龙研究**：成果丰硕
- **古人类研究**：北京猿人等
- **古哺乳动物研究**：成果丰硕
