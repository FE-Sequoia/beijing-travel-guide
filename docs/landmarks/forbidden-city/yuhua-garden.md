# 御花园

![御花园](https://images.unsplash.com/photo-1681495388843-42440711c89d?w=1200&q=80)

> 红墙之内，一园收尽江南意。堆秀山的太湖石、钦安殿的玄武像、四百年的古柏藤萝——这是帝王的后花园。

## 简介

御花园位于紫禁城中轴线的最北端、坤宁宫之后，是故宫内廷的皇家花园，始建于明永乐十八年（1420年），与紫禁城同龄。它南北长约80米、东西宽约140米，占地约12000平方米——在偌大的紫禁城中，它只是一方小巧的天地，却以极高的密度汇聚了亭台楼阁、山石花木、铜像石刻二十余处景观，是中国古典皇家园林"小中见大"的典范。

与江南私家园林的自由曲折不同，御花园是**中轴对称**的皇家格局：中路以钦安殿为主殿，两侧建筑严格对称排布，在规整中点缀山石花木的灵动。这里是帝后茶余饭后的游憩之所，也是重阳节登高、七夕乞巧等节令活动的宫廷舞台。1925年故宫博物院成立后，御花园成为游客出神武门前最后的流连之地。

## 建筑与景观

| 项目 | 数据 |
|:---|:---|
| **占地面积** | 约12000平方米 |
| **始建** | 明永乐十八年（1420年） |
| **布局** | 中轴对称，以钦安殿为中心 |
| **建筑数量** | 亭台楼阁二十余座 |
| **主殿** | 钦安殿（供奉玄武大帝/真武大帝） |
| **主山** | 堆秀山（太湖石假山，山顶有御景亭） |
| **古树** | 四百年以上古柏、藤萝数十株 |

御花园的景观可分三路：

- **中路**：以**钦安殿**为核心，殿内供奉**玄武大帝**（真武大帝，北方之神），是紫禁城内最重要的道教神祇。殿为明代原构，盝顶形制，是故宫中少见的明代建筑遗存。殿前一对**连理柏**（两株古柏树干相交而生），被视为爱情与祥瑞的象征。
- **东路**：**堆秀山**是一座用太湖石堆叠而成的假山，山顶建有**御景亭**。每逢重阳节（农历九月初九），帝后登此亭远眺，取"登高避灾"之意——紫禁城内并无真山，堆秀山便是帝王"不出宫而登高"的替代品。山下有**摛藻堂**，为乾隆帝藏书之所。
- **西路**：**延辉阁**、**位育斋**等建筑错落分布，配以古柏、藤萝、铜鹿、铜鹤，营造出幽深静谧的园趣。园中有一株**紫藤萝**，据传植于明代，花开时满架紫云，是御花园最富诗意的春景。

## 历史故事

### 堆秀山：皇帝"不出宫的登高"

重阳节登高是中国延续千年的节俗，但深居紫禁城的帝后不可能出城登山。于是，御花园东北角这座用江南太湖石堆叠而成的假山——堆秀山，便成了皇家的"专属登山处"。山顶的御景亭是全园制高点，重阳之日，帝后登亭饮菊花酒、佩茱萸囊，遥望宫墙外的秋色。一座几米高的假山承载了"登高"的全部仪式——礼制大于山水，这正是皇家园林与文人园林的根本分野。

### 连理柏与紫藤萝：四百年的植物见证

御花园中最动人的不是建筑，而是那些活过四百年的植物。钦安殿前的两株古柏，树干在中部交缠相连，被称为"连理柏"——"在天愿作比翼鸟，在地愿为连理枝"，帝后常以此树寄托恩爱之愿。而园中那株明代紫藤，每到暮春便垂下串串紫花，据记载已历经明清两代、四百余年春秋，是紫禁城最年长的"居民"之一。建筑会焚毁重建，这些古木却是真正的永乐遗民。

## 必看亮点

- **堆秀山与御景亭**：太湖石假山叠石嶙峋，山顶小亭是御花园的视觉高点，也是重阳节"皇家登高"的现场。
- **钦安殿**：供奉玄武大帝的明代原构，盝顶形制在故宫中独一无二，殿内神像与陈设保存完好。
- **连理柏**：钦安殿前两株交缠的古柏，是御花园最具象征意味的"活文物"。
- **古藤萝**：暮春时节满架紫花，是紫禁城内最富诗意的植物景观。
- **对称中的灵动**：漫步全园，感受"皇家规制"与"园林野趣"如何在方寸之间取得平衡。

## 参观贴士

- **别赶路**：御花园是故宫之行的收尾，多数游客匆匆穿过直奔神武门。放慢脚步，这里恰恰是整个皇宫中最松弛、最有人情味的空间。
- **暮春与深秋最佳**：四五月紫藤花开、九十月秋叶金黄，是御花园最美的季节。
- **注意对称布局**：站在中轴线上左右对照，会发现两侧建筑严格镜像——这是皇家园林区别于江南私园的标志，值得有意识地体会。
- **出口选择**：出神武门（北门）可顺游景山公园登顶俯瞰紫禁城全景；从御花园向东可折往珍宝馆方向（宁寿宫区）。
- **铜像与石子路**：留意园中的铜鹿、铜鹤和脚下的石子拼花路面——"连生贵子""麒麟送子"等吉祥图案就铺在脚底。
