# 祈年殿

![祈年殿](https://images.unsplash.com/photo-1769950971259-d9dfb821c7b8?w=1200&q=80)

> "三重蓝瓦覆苍穹，二十八柱藏星斗。"

## 简介

祈年殿是**天坛的主体建筑**，也是整个北京城的标志性符号。它矗立于内坛北部一座三层汉白玉圆台（祈谷坛）之上，是一座圆形三重檐攒尖顶大殿：直径32米，通高38米，上覆三层蓝色琉璃瓦，顶立鎏金宝顶，在古柏与长空的映衬下，恰似一顶从天而降的华盖。

祈年殿的功能是**孟春祈谷**——每年农历正月上辛日，皇帝在此举行祈谷大典，向上帝祈求五谷丰登、国泰民安。与南部的[圜丘坛](/landmarks/temple-of-heaven/huanqiu-tan)（冬至祭天）一北一南，构成天坛祭祀体系的两大核心，中由[丹陛桥](/landmarks/temple-of-heaven/danbi-qiao)相连。

祈年殿始建于明永乐十八年（1420年），初名"大祀殿"，为矩形大殿，合祀天地。嘉靖二十四年（1545年）改建为圆形三重檐的"大享殿"，瓦作青、黄、绿三色，分别象征天、地、万物。清乾隆十六年（1751年）再次大修，将三色瓦统一为纯蓝琉璃瓦，并正式定名"祈年殿"——蓝色，正是天的颜色。

在中国人的集体记忆中，祈年殿早已超越建筑本身：它与长城、故宫并列，是"中国"二字最直观的视觉符号之一。那抹立于古柏之上的深蓝，既是古代帝王对天的敬畏，也是中式美学"以象尽意"的极致表达。

## 建筑与景观

![祈年殿蓝瓦金顶](https://images.unsplash.com/photo-1761160763579-69a51c6ddd33?w=1200&q=80)

祈年殿最精妙之处，在于**全殿不用大梁与长檩**，仅凭二十八根楠木巨柱和层层斗拱、枋桷支撑起三重穹顶，堪称中国木构建筑的巅峰之作。这二十八根柱子本身就是一部历法：

| 柱位 | 名称 | 数量 | 象征 |
|:---|:---|:---:|:---|
| 最中央 | 龙井柱 | 4根 | 春夏秋冬四季 |
| 中圈 | 金柱 | 12根 | 一年十二个月 |
| 外圈 | 檐柱 | 12根 | 一天十二个时辰 |
| 合计 | — | 28根 | 天上二十八星宿 |

| 项目 | 规格 |
|:---|:---|
| 殿体直径 | 32米 |
| 通高（含台基宝顶） | 38米 |
| 屋顶形制 | 圆形三重檐攒尖顶 |
| 瓦作 | 三层纯蓝琉璃瓦 |
| 宝顶 | 鎏金宝顶 |
| 殿基 | 三层汉白玉圆台（祈谷坛），高6米 |
| 结构特点 | 无大梁长檩，以柱、枋、斗拱承重 |

殿内穹顶中央饰有九龙藻井，金龙盘绕，精美绝伦；地面中心铺一方大理石，天然纹样形似龙凤，与穹顶藻井上下呼应。殿北的皇乾殿则是平时供奉祈谷神牌之处。

殿前的祈谷坛同样暗藏"九"数：三层圆台每层台阶九级，栏板数目皆为九的倍数，与南部的[圜丘坛](/landmarks/temple-of-heaven/huanqiu-tan)同一数理体系。从坛下仰望，三层蓝瓦逐层收分，宝顶直指苍穹，整座建筑仿佛一个向上收束的祈愿手势。

## 历史故事

### 雷火焚殿与重建

清光绪十五年（1889年）八月二十四日，一场罕见的雷雨袭击北京，**祈年殿被雷电击中，烈焰冲天，整座大殿连同二十八根楠木巨柱尽数焚毁**。相传殿中珍贵的金丝楠木燃烧时异香弥漫，数里之外皆可闻到。

光绪帝下旨按原样重建。由于原图纸档案残缺，工匠们全凭记忆与实测，耗时数年（1890年开工，1896年全部竣工），将祈年殿分毫不差地恢复原貌。今日所见的祈年殿，即这次重建的遗构——它既是明代礼制的延续，也是晚清匠作技艺的见证。

### 嘉靖改制定圆形

明初天地合祀，大祀殿是方形大殿。嘉靖帝笃信道教、讲究礼制，力排众议推行天地分祀，并把大殿改为圆形三重檐——"天圆"之意从此凝固在建筑形制之中。这场"大礼议"之后的礼制改革，意外成就了天坛今日的经典轮廓。

### 深山楠木万里来

二十八根承重巨柱均为珍贵的金丝楠木，其中尤以中央四根龙井柱最为难得。明清时期优质楠木产于川、黔、滇深山密林之中，采伐一根大木往往"入山一千，出山五百"——伐木工半数殒于途。巨木经水路辗转数年方能抵京，故民间有"一根楠木一两金"之说。祈年殿的庄严背后，是万里山林与无数人力。

## 最佳游览季节

- **春季**（3-5月）：坛内丁香、二月兰盛开，蓝瓦与花影相映
- **秋季**（9-11月）：古柏苍翠、天空澄澈，是拍摄蓝瓦金顶的黄金期
- **雪后**：白雪覆坛、蓝瓦如洗，祈年殿呈现出最接近古画的气质

## 必看亮点

1. **二十八柱历法密码**：数清四季、十二月、十二时辰、二十八宿的象征体系，这是读懂中国古代时间观的一把钥匙。
2. **三层蓝瓦与鎏金宝顶**：天坛区别于紫禁城黄瓦的核心视觉符号，晴天时蓝瓦与天空几乎同色。
3. **九龙藻井与龙凤石**：殿内仰观穹顶藻井，俯察地面龙凤纹大理石，上下成趣。
4. **祈谷坛三层圆台**：环行台基可平视殿身细节，也是拍摄祈年殿全景的最佳高度。
5. **祈年门与皇乾殿**：殿前祈年门为明代原构，殿后皇乾殿供奉神牌，别错过。

## 参观贴士

1. **联票方可入内院**：祈年殿区域需持联票（28元）或另购景点票，仅凭10元门票只能远观。
2. **最佳拍摄时间**：清晨8:00开门即入，游客最少；逆光时蓝瓦呈深宝石色，顺光时则清亮。
3. **绕殿三圈看细节**：从台基外圈、台基上、殿门处三个层次观赏，空间感受截然不同。
4. **顺路游皇乾殿**：祈年殿北的皇乾殿陈列祭祀礼仪复原场景，常被游客忽略。
5. **向北出园**：看完祈年殿可经天坛北门出，衔接地铁或前往雍和宫方向。
6. **殿内勿错过数字**：数柱、数瓦当、看藻井，带着"二十八宿"的线索进殿，收获远超走马观花。
