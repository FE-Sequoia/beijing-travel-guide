# 雍和宫

> "北京最大的藏传佛教寺院，香火最旺的皇家寺院。"

## 简介

雍和宫位于北京市东城区雍和宫大街，是北京最大的藏传佛教寺院，也是清朝中后期全国规格最高的佛教寺院。雍和宫原为清雍正帝即位前的府邸，乾隆九年（1744年）改为藏传佛教寺院。

## 历史沿革

### 王府时期

清康熙三十三年（1694年），康熙帝在此建造府邸，赐予皇四子胤禛（即后来的雍正帝）。胤禛即位前在这里居住了近20年。

### 雍亲王府

胤禛被封为雍亲王，府邸称"雍亲王府"。1722年康熙帝驾崩，胤禛即位，年号雍正。

### 行宫时期

雍正帝即位后，雍亲王府改为行宫，称"雍和宫"。雍正帝经常来此礼佛、休息。

### 改为寺院

乾隆九年（1744年），乾隆帝为纪念其父雍正帝，将雍和宫改为藏传佛教寺院，成为清政府管理全国藏传佛教事务的中心。

## 建筑布局

雍和宫共有五进院落，主要建筑沿中轴线分布。

### 昭泰门

昭泰门是雍和宫的山门，门前有牌楼三座，东西两侧有钟楼、鼓楼。

### 天王殿

天王殿是雍和宫的第一重大殿，供奉弥勒佛和四大天王。

### 雍和宫殿

雍和宫殿是雍和宫的主殿，原为雍亲王府的正殿。殿内供奉三世佛：释迦牟尼、燃灯佛、弥勒佛。

### 永佑殿

永佑殿是雍和宫的第三重大殿，供奉无量寿佛。这里曾是雍正帝的书房和寝宫。

### 法轮殿

法轮殿是雍和宫的重要殿堂，是举行法会的场所。殿内供奉宗喀巴大师像。

#### 五百罗汉山

法轮殿内有著名的"五百罗汉山"，用紫檀木雕刻而成，工艺精湛。

### 万福阁

万福阁是雍和宫最宏伟的建筑，是供奉弥勒大佛的地方。

#### 弥勒大佛

万福阁内的弥勒大佛是雍和宫的镇寺之宝：

| 项目 | 数据 |
|:---|:---|
| **高度** | 18米（地上）+ 8米（地下） |
| **材质** | 整根白檀香木 |
| **重量** | 约100吨 |
| **地位** | 独木大佛，世界之最 |

这尊大佛由一整根白檀香木雕刻而成，是雍和宫最珍贵的文物。

## 藏传佛教

### 格鲁派

雍和宫属于藏传佛教格鲁派（黄教），是宗喀巴大师创立的教派。

### 活佛转世

雍和宫设有"金本巴瓶"，用于掣签认定蒙藏地区活佛的转世灵童。

### 法会

雍和宫每年举行多次法会，其中农历正月初一的祈福法会和正月二十九的金刚舞法会最为隆重。

昭泰门 → 天王殿 → 雍和宫殿 → 永佑殿 → 法轮殿 → 万福阁 → 绥成殿 → 出宫

**重点参观**：
1. 万福阁的弥勒大佛（必看）
2. 法轮殿的五百罗汉山
3. 永佑殿的无量寿佛

## 文化价值

雍和宫是全国重点文物保护单位，是汉族地区佛教全国重点寺院。

雍和宫是北京最大的藏传佛教寺院，也是清政府管理全国藏传佛教事务的中心，在历史上发挥了重要作用。

雍和宫的弥勒大佛是木雕艺术的杰作，是中国佛教艺术的瑰宝。
