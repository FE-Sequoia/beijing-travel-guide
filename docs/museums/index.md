# 北京博物馆

> "文明的殿堂，知识的海洋。"

北京拥有全国最丰富的博物馆资源，从综合性的国家博物馆到专题性的行业博物馆，从展示中华文明的古代文物到展现现代科技的前沿展览，北京的博物馆涵盖各个领域，是了解中国历史和文化的最佳窗口。

## 综合性博物馆

### [中国国家博物馆](/museums/national-museum)

世界上单体建筑面积最大的博物馆，中华文明的总汇。

- **门票**：免费（需预约）
- **看点**：古代中国、复兴之路、后母戊鼎、四羊方尊
- **建议时长**：半天至一天

### [首都博物馆](/museums/capital-museum)

展示北京历史文化的窗口。

- **门票**：免费（需预约）
- **看点**：北京通史、老北京民俗、燕地青铜器
- **建议时长**：3-4小时

### [故宫博物院](/museums/palace-museum)

明清皇宫，文物收藏巨擘。

- **门票**：60元
- **看点**：180万件藏品、钟表馆、珍宝馆
- **建议时长**：半天至一天

## 科技馆

### [中国科学技术馆](/museums/science-center)

国家级综合性科技馆，亲子游首选。

- **门票**：主展厅30元
- **看点**：华夏之光、探索与发现、特效影院
- **建议时长**：半天至一天

### [北京天文馆](/museums/planetarium)

探索宇宙奥秘，天象节目精彩。

- **门票**：展厅10元，剧场45元
- **看点**：天象厅、宇宙剧场、陨石展览
- **建议时长**：2-3小时

### [中国古动物馆](/museums/paleo-zoo)

亚洲最大的古动物博物馆。

- **门票**：20元
- **看点**：恐龙化石、古人类化石、黄河象
- **建议时长**：2小时

## 专题博物馆

### [军事博物馆](/museums/military)

展示中国军事历史和武器装备。

- **门票**：免费（需预约）
- **看点**：武器装备、革命历史、抗美援朝
- **建议时长**：3-4小时

### [自然博物馆](/museums/natural-history)

探索自然奥秘，恐龙世界精彩。

- **门票**：免费（需预约）
- **看点**：恐龙化石、人体奥秘、非洲动物
- **建议时长**：3-4小时

### [鲁迅博物馆](/museums/lu-xun)

纪念鲁迅生平与创作。

- **门票**：免费
- **看点**：鲁迅故居、手稿、生平展览
- **建议时长**：1-2小时

### [老舍纪念馆](/museums/laoshe)

纪念老舍生平与创作。

- **门票**：免费
- **看点**：老舍故居、手稿、生平展览
- **建议时长**：1小时

### 参观建议

1. **提前预约**：热门博物馆需提前1-7天预约
2. **避开高峰**：周二至周四人相对较少
3. **租借讲解**：建议租借语音导览或请人工讲解
4. **合理安排**：一天最多安排2个博物馆
5. **注意闭馆**：大部分博物馆周一闭馆

### 亲子推荐

| 年龄段 | 推荐博物馆 |
|:---|:---|
| 3-6岁 | 自然博物馆、科技馆儿童科学乐园 |
| 6-12岁 | 科技馆、天文馆、古动物馆 |
| 12岁以上 | 国博、首博、军事博物馆 |
