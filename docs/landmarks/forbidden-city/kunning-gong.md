# 坤宁宫

![坤宁宫](https://images.unsplash.com/photo-1551101509-f6f2cbae3604?w=1200&q=80)

> 明代皇后的正宫，清代祭神的萨满圣殿，大婚洞房的龙凤红烛——一座坤宁宫，三层身份的交叠。

## 简介

坤宁宫是内廷后三宫的最后一座，位于交泰殿之北，与乾清宫相对，"坤"为地、"宁"为安，取《道德经》"地得一以宁"之意，是明代**皇后的正宫**。然而，今天游客看到的坤宁宫，早已不是明代皇后起居的模样——清顺治十二年（1655年），清廷仿沈阳清宁宫的规制对坤宁宫进行了彻底改造，使其成为紫禁城内**萨满教祭祀的固定场所**，每日宰猪祭神，烟火不绝。

更为人津津乐道的是它的第三重身份：坤宁宫**东暖阁**是清代皇帝大婚的**洞房**。康熙、同治、光绪三位皇帝的大婚均在此度过新婚之夜，龙凤花烛、百子帐幔，是紫禁城中最具人间烟火气的角落。

坤宁宫面阔九间，重檐庑殿顶，体量与乾清宫相仿，是内廷最大的宫殿之一。

## 建筑与景观

| 项目 | 数据 |
|:---|:---|
| **面阔** | 九间（明代原制，清代改造后内部重新分隔） |
| **进深** | 五间 |
| **屋顶** | 重檐庑殿顶 |
| **正门** | 偏于东侧（满族"口袋房"习俗，门不开在正中） |
| **西四间** | 萨满祭祀场所，设煮肉大锅、神杆、神位 |
| **东暖阁** | 皇帝大婚洞房，龙凤喜床、百子帐幔 |
| **殿后** | 坤宁门，通往御花园 |

坤宁宫最显著的外观特征是**正门偏于东侧**而非居中——这是满族传统"口袋房"的布局习俗，是清代改造留下的最直观的民族印记。殿内空间被重新分隔：

- **西侧四间**：辟为萨满祭神之所。室内靠窗设连排大锅，每日宰杀两口活猪，在神位前煮"祭神肉"（白水煮猪肉，不加盐酱）。祭祀时，萨满（满洲巫祝）摇铃击鼓、诵唱神歌，皇帝或亲自主祭、或遣官代祭。殿外竖立一根**索伦杆**（神杆），杆顶置斗盛碎肉喂乌鸦，是满族"神鹊救祖"传说的礼仪遗存。
- **东暖阁**：大婚洞房。室内墙壁髹红漆、绘龙凤，设龙凤喜床，床上铺百子被，帐挂百子图。新婚帝后在此共居数日，然后皇帝返回养心殿，皇后移居东西六宫。

## 历史故事

### 大婚洞房：帝王的"三天新婚"

清代皇帝大婚是举国盛典，而洞房就设在坤宁宫东暖阁。新婚之夜，帝后行"合卺礼"（交杯酒），共食"子孙饽饽"（半生饺子，取"生子"谐音），在龙凤花烛下度过新婚之夜。按清制，帝后在坤宁宫同住不过三日，之后皇帝即返回养心殿，皇后移居自己的寝宫——这桩婚姻从一开始就是国家典礼而非私人生活。清代在坤宁宫大婚的皇帝仅三位：**康熙**（11岁大婚）、**同治**（17岁）、**光绪**（18岁）。溥仪大婚时清朝已亡，但仍在坤宁宫东暖阁举行了仪式。今天游客透过东暖阁的窗棂，仍能看到那铺着百子被的龙凤喜床——红烛已熄，龙凤犹在。

### 每日两口猪：紫禁城里的萨满祭祀

清代坤宁宫最令人意外的功能是它的"日常"：每天清晨和傍晚，这里都要举行萨满祭祀，**每日宰杀两口活猪**，在殿内大锅中白水煮祭。祭神肉不加任何调料，祭祀后由王公大臣分食——被皇帝"赏肉"是极大的荣耀，但那白水煮猪肉的滋味可想而知，不少大臣以各种借口推辞"赏肉"。这套祭祀从清初一直延续到清末，近三百年间坤宁宫的锅里几乎日日肉香不绝，殿外的神杆上天天有乌鸦啄食碎肉。一座皇后寝宫变成了萨满圣殿，是满族入关后"国语骑射"祖制在紫禁城最顽固的坚守。

## 必看亮点

- **东暖阁洞房**：透过殿东侧的窗棂窥看龙凤喜床与百子帐幔，是紫禁城最"有人间温度"的景观。
- **偏置的正门**：注意坤宁宫大门不在正中而偏东——这是满族口袋房习俗在紫禁城的直接体现。
- **殿外索伦杆**：殿前竖立的神杆（索伦杆）是满族萨满祭祀的标志物，杆顶饲鸦的习俗源自"神鹊救祖"传说。
- **西四间祭神格局**：殿内西侧的连排大锅、神位陈设，是清代萨满祭祀原貌的珍贵留存。
- **坤宁门**：殿后坤宁门是通往御花园的门户，门额"坤宁门"三字值得留意。

## 参观贴士

- **东暖阁在东侧**：洞房陈设在坤宁宫东端，从殿外东侧窗口可见，别只在殿前正面停留。
- **与乾清宫对照**：乾清（天）与坤宁（地）一前一后、一乾一坤，是故宫"天地对应"设计的核心一对。明代帝居乾清、后居坤宁；清代乾清转为礼仪殿、坤宁转为祭神殿——两宫的功能变迁折射了满汉文化的交融与角力。
- **别急着去御花园**：坤宁宫是后三宫的终点、御花园的起点，在这里稍作停留，感受从"朝政空间"到"皇家园林"的过渡。
- **了解满族习俗**：口袋房门、索伦杆、白水煮肉——坤宁宫的种种"不合汉制"之处，正是理解清代"满汉双轨"统治的实物教材。
