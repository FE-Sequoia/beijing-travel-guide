# 紫竹院公园

> "竹影婆娑，人少景美的城市公园。"

## 简介

紫竹院公园位于北京市海淀区，是一座以竹为景的城市公园。公园占地约47公顷，其中水面约15公顷，是北京著名的竹景名园。

## 历史沿革

### 始建

紫竹院公园所在地原是一片洼地，清代在此修建行宫，种植竹子。

### 现代公园

1953年，辟为公园，取名"紫竹院"。此后不断扩大规模，增加竹种。

## 竹景

### 竹林

紫竹院公园种植了各种竹子，是北京著名的竹景公园。

| 竹种 | 特点 |
|:---|:---|
| **紫竹** | 竹秆紫色，公园名由此而来 |
| **斑竹** | 竹秆有斑点 |
| **苦竹** | 竹秆有苦味 |
| **早园竹** | 生长迅速 |
| **毛竹** | 竹秆粗壮 |

### 竹文化

紫竹院公园以竹文化为主题，有竹文化展览、竹工艺品展示等。

## 主要景点

### 荷花渡

夏季，荷花渡荷花盛开，是赏荷的好地方。

### 问月楼

问月楼是公园的主要建筑，可以登楼远眺。

### 澄碧山房

澄碧山房是公园的水上建筑，环境清幽。

### 八宜轩

八宜轩是公园的亭子，"八宜"指"春宜花，夏宜风，秋宜月，冬宜雪，昼宜棋，夜宜琴，晴宜书，雨宜酒"。

## 游船

公园内有水面，可以划船、脚踏船。

## 游览攻略

### 推荐路线

**经典路线（约1-2小时）**：

南门 → 荷花渡 → 问月楼 → 澄碧山房 → 八宜轩 → 竹林 → 北门

### 实用贴士

1. **免费入园**：公园免费开放
2. **人少清静**：游客较少，比较清静
3. **夏季荷花**：夏季荷花盛开
4. **竹景优美**：竹影婆娑，非常优美
5. **划船体验**：可以划船游湖
6. **结合游览**：可与动物园、国家图书馆一起游览
