# 故宫博物院

![故宫](https://images.unsplash.com/photo-1599661046289-e31897846e41?w=1200)

> "金碧辉煌紫禁城，红墙宫里万重门。"
> —— 唐代诗人 王建

## 简介

故宫博物院，旧称紫禁城，是明清两代的皇家宫殿，位于北京中轴线的中心。它是世界上现存规模最大、保存最为完整的木质结构古建筑群之一，被誉为世界五大宫之首（北京故宫、法国凡尔赛宫、英国白金汉宫、美国白宫、俄罗斯克里姆林宫）。

## 历史沿革

### 营建历史

明永乐四年（1406年），明成祖朱棣下诏营建北京宫殿。永乐十八年（1420年）建成，正式迁都北京。故宫占地面积72万平方米，建筑面积约15万平方米，有大小宫殿七十多座，房屋九千余间。

### 朝代更迭

- **明朝（1420-1644）**：14位皇帝在此居住
- **清朝（1644-1911）**：10位皇帝在此居住
- **民国时期（1912-1924）**：溥仪退位后仍居宫中
- **故宫博物院（1925至今）**：1925年10月10日正式成立

## 建筑布局

故宫按照"前朝后寝"的制度布局，分为外朝和内廷两部分：

### 外朝（前朝）

以太和殿、中和殿、保和殿为中心，是皇帝举行大典、处理朝政的地方。

| 宫殿 | 功能 | 看点 |
|:---|:---|:---|
| **太和殿** | 举行大典 | 金銮宝座、龙纹装饰 |
| **中和殿** | 休息准备 | 皇帝大典前小憩 |
| **保和殿** | 宴饮科举 | 清代殿试场所 |

### 内廷（后寝）

以乾清宫、交泰殿、坤宁宫为中心，是皇帝和后妃居住的地方。

| 宫殿 | 功能 | 看点 |
|:---|:---|:---|
| **乾清宫** | 皇帝寝宫 | "正大光明"匾额、秘密立储 |
| **交泰殿** | 皇后起居 | 存放玉玺 |
| **坤宁宫** | 皇后寝宫 | 萨满祭祀 |
| **御花园** | 皇家园林 | 堆秀山、摛藻堂 |

## 必看珍宝

### 钟表馆

奉先殿钟表馆收藏中外钟表200余件，有英国、法国、瑞士、日本及中国清宫造办处制造的各式钟表，体现了18-20世纪世界钟表制造的顶尖水平。

### 珍宝馆

宁寿宫珍宝馆展示清代宫廷制作及收藏的珍贵文物，包括金器、银器、玉器、宝石、珍珠、翡翠、珊瑚等。

### 石鼓馆

收藏被誉为"石刻之祖"的十面先秦石鼓，是中国最早的石刻文字之一。

## 文化故事

### "正大光明"匾额

乾清宫"正大光明"匾额是顺治皇帝御笔亲书。自雍正皇帝开始，这块匾额背后成为秘密立储诏书的存放处。皇帝生前将继承人名字写好，密封藏于匾后，驾崩后当众取出宣读。

### 九龙壁

九龙壁位于皇极门外，是一座长29.4米、高3.5米的琉璃照壁。壁上九条龙栩栩如生，形态各异，展现了高超的工艺水平。传说当年烧制时有一块构件不慎烧坏，工匠用楠木雕制补上，瞒过了乾隆皇帝的检查。

### 珍妃井

珍妃井位于东北角贞顺门内。1900年八国联军攻入北京时，慈禧太后命人将光绪皇帝宠爱的珍妃推入此井溺死。这口井成为宫廷斗争悲剧的见证。

## 游览攻略

### 推荐路线

**半日游路线（约4小时）**：

午门 → 太和门 → 太和殿 → 中和殿 → 保和殿 → 乾清门 → 乾清宫 → 交泰殿 → 坤宁宫 → 御花园 → 神武门

**一日游路线（约6-8小时）**：

午门 → 武英殿（陶瓷馆） → 太和门 → 太和殿 → 中和殿 → 保和殿 → 钟表馆 → 珍宝馆 → 东六宫 → 乾清门 → 乾清宫 → 交泰殿 → 坤宁宫 → 御花园 → 西六宫 → 神武门

### 实用贴士

1. **提前预约**：必须提前在官网或微信预约，现场不售票
2. **避开人流**：上午10点前或下午3点后入场，避开旅行团高峰
3. **穿着舒适**：需要大量步行，建议穿舒适鞋子
4. **讲解服务**：可租借语音导览（20元）或请人工讲解（150-350元）
5. **拍摄建议**：清晨和傍晚光线最佳，红墙黄瓦非常适合拍照
