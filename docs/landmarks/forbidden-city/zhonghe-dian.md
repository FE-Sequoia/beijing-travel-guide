# 中和殿

![中和殿](https://images.unsplash.com/photo-1590829495436-6231c80f6a6a?w=1200&q=80)

> 三殿之中，方亭独立。它是大典前的一息静默，也是"中庸"之道在建筑上的化身。

## 简介

中和殿是外朝三大殿的中间一座，位于太和殿与保和殿之间，建成于明永乐十八年（1420年），初名"华盖殿"，嘉靖年间改名"中极殿"，清顺治二年（1645年）定名"中和殿"。其名取自《礼记·中庸》"喜怒哀乐之未发谓之中，发而皆中节谓之和"，寓意帝王行事当持守中正、致达和谐。

在三大殿的序列中，中和殿体量最小，形制却最独特：它是三殿中唯一的**方形攒尖顶**建筑，外观玲珑如亭，与前后两座恢宏的长方形大殿形成鲜明的节奏对比。这座小殿并非摆设——它是皇帝赴太和殿举行大典前的休憩之所，也是国家农事与祭祀礼仪的准备空间。

## 建筑与景观

| 项目 | 数据 |
|:---|:---|
| **平面形制** | 正方形，面阔、进深各三间 |
| **屋顶** | 单檐四角攒尖顶，鎏金宝顶 |
| **建筑面积** | 约580平方米 |
| **殿内匾额** | "允执厥中"（乾隆御笔） |
| **殿内陈设** | 两座肩舆（皇帝乘轿）、乾隆御制《中和殿记》屏风 |

中和殿是一座边长约24米的正方形建筑，四面开门，四面设窗，四面各设三交六椀菱花槅扇门。这种四面通透的形制在紫禁城的大殿中绝无仅有，既便于皇帝出入，也暗合"四通八达、无往不中"的寓意。

殿顶为**单檐四角攒尖顶**，上覆铜胎鎏金宝顶，在阳光下金光灿灿，是三大殿天际线上一个精巧的句读。殿内高悬"允执厥中"匾额，语出《尚书·大禹谟》，告诫帝王治国须持守中道。殿中放置两座皇帝赴典礼时乘坐的**肩舆**（御轿），以及一面刻有乾隆御制《中和殿记》的屏风。

## 历史故事

### 大典前的"静默时刻"

每逢太和殿举行登基、册封、朝贺等大典礼，皇帝需先乘舆至中和殿暂驻。在这里，礼部官员会最后一次核对手续流程，皇帝稍事休息、整理仪容，待时辰一到，才起驾前往太和殿升座。这座小殿承担的是"大典前最后一间准备室"的功能——外面是万人山呼的太和殿广场，里面是片刻的宁静，一墙之隔，便是从"人"到"天子"的转换。

### 阅视祝版与亲耕农具

中和殿还承担着两项颇具象征意义的礼仪：每逢皇帝赴天坛、地坛、太庙祭祀前，要在此**阅视祝版**（祭文），确认无误后方能举行大典，以示对天地祖宗的恭敬。每年春季皇帝行"亲耕礼"（在先农坛象征性扶犁耕田）之前，也要在中和殿**检视种子与农具**。堂堂天子在一座小殿里郑重其事地查看谷种和犁具——这套仪式的意义不在农事本身，而在于向天下昭示"以农为本"的治国姿态。

## 必看亮点

- **攒尖宝顶**：从太和殿背后回望，鎏金宝顶是三大殿天际线上最亮眼的一笔，与太和殿的重檐庑殿顶形成大小对比。
- **"允执厥中"匾额**：殿内正中高悬，是理解中和殿命名哲学与帝王治国理念的钥匙。
- **四面通透的形制**：绕殿一周，欣赏四面相同的菱花槅扇门——这种"四方如一"的设计正是"中"的视觉化表达。
- **肩舆与屏风**：透过殿门可见两座御轿和乾隆御记屏风，是难得一见的清宫礼仪实物。

## 参观贴士

- **别匆匆走过**：中和殿夹在两座大殿之间，游客往往一掠而过。它恰恰是三大殿节奏中不可或缺的"呼吸"，值得驻足细看。
- **最佳机位**：站在太和殿与中和殿之间的月台上，可同时框住两座殿宇，大小对比一目了然。
- **外观为主**：与太和殿一样，中和殿内部不开放入内，只能隔门观赏。
- **留意名字含义**：把"太和""中和""保和"三个名字连起来读，正是《周易》"保合太和，乃利贞"的完整哲学——三大殿的命名是一套连贯的治国理念。
