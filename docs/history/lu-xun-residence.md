# 鲁迅故居

> "新文化运动的旗手，民族魂的栖息地。"

## 简介

鲁迅故居位于北京市西城区阜成门内西三条21号，是鲁迅先生1924年至1926年在北京居住的地方。这是一座典型的北京四合院，鲁迅在这里完成了《华盖集》《华盖集续编》《野草》等作品。

## 鲁迅简介

### 生平

鲁迅（1881-1936），原名周树人，字豫才，浙江绍兴人。中国现代文学的奠基人，伟大的文学家、思想家、革命家。

- **1881年**：生于浙江绍兴
- **1902年**：留学日本
- **1909年**：回国任教
- **1918年**：发表《狂人日记》，首次使用"鲁迅"笔名
- **1924年**：购得阜成门内西三条21号
- **1926年**：离开北京南下
- **1936年**：病逝于上海

## 故居建筑

### 四合院

鲁迅故居是一座典型的北京小四合院，坐北朝南。

| 项目 | 描述 |
|:---|:---|
| **占地面积** | 约400平方米 |
| **建筑格局** | 小型四合院 |
| **正房** | 鲁迅卧室兼书房 |
| **南房** | 会客室和藏书室 |
| **东西厢房** | 厨房和杂物间 |

### 鲁迅手植白丁香

院内有一棵白丁香树，是鲁迅1924年手植，至今已有近百年历史。

### "老虎尾巴"

正房后面接出一间平顶小屋，鲁迅称之为"老虎尾巴"，是他的书房和卧室。在这里，鲁迅完成了大量作品。

## 展览内容

故居内设有鲁迅生平展览，包括：

- **生平照片**：鲁迅各个时期的照片
- **手稿书信**：鲁迅的手稿和书信
- **生活用品**：鲁迅使用过的物品
- **藏书**：鲁迅的藏书
- **作品展示**：鲁迅的作品和翻译作品

## 鲁迅在北京

### 北京岁月

鲁迅在北京生活了14年（1912-1926），这是他创作生涯的重要时期。

| 时间 | 经历 |
|:---|:---|
| **1912年** | 任教育部部员 |
| **1918年** | 发表《狂人日记》 |
| **1920年** | 任北大讲师，讲授中国小说史 |
| **1923年** | 与周作人决裂，搬出八道湾 |
| **1924年** | 购得阜成门内西三条21号 |
| **1926年** | 离开北京南下 |

### 创作高峰

在北京期间，鲁迅创作了大量作品，包括小说、散文、杂文等。

## 文化价值

鲁迅故居是鲁迅在北京的最后居所，是他创作生涯的重要场所。

鲁迅是中国现代文学的奠基人，他的作品深刻揭露了旧社会的黑暗，唤醒了民众的意识。参观鲁迅故居，可以了解这位伟大文学家的生平事迹和创作历程。
