# 北京话剧地图

> "方寸舞台，演尽人生百态。"

北京是中国话剧的发源地和中心，拥有众多历史悠久、风格各异的剧院。从人艺的经典到先锋实验，从传统戏曲到现代话剧，北京的舞台永远充满活力。

## 主要剧院

### 首都剧场（北京人艺）

![首都剧场](https://images.unsplash.com/photo-2S_13rwpjKY?w=1200)

北京人艺是中国最著名的话剧院团，成立于1952年，被誉为"中国话剧的殿堂"。首都剧场是北京人艺的固定演出场所，位于王府井商圈。

#### 基本信息

| 项目 | 内容 |
|:---|:---|
| **地址** | 北京市东城区王府井大街22号 |
| **交通** | 地铁1号线王府井站A口 |
| **座位数** | 约900个 |
| **票价** | 80-680元 |
| **官网** | [www.bjry.com](http://www.bjry.com) |

#### 特色

- **经典传承**：北京人艺的看家戏《茶馆》《雷雨》等常年上演
- **戏骨云集**：濮存昕、冯远征、何冰等著名演员常驻
- **京味文化**：以京味话剧著称，台词韵味十足

### 天桥艺术中心

天桥艺术中心是北京最新、最现代化的艺术演出中心，拥有大、中、小三个剧场，是北京话剧演出的重要场所。

#### 基本信息

| 项目 | 内容 |
|:---|:---|
| **地址** | 北京市西城区天桥南大街9号 |
| **交通** | 地铁8号线天桥站A口 |
| **座位数** | 大剧场1600个，中剧场1000个，小剧场400个 |
| **票价** | 80-1080元 |
| **官网** | [www.tqpac.com](http://www.tqpac.com) |

#### 特色

- **设施一流**：拥有北京最先进的舞台设施
- **剧目丰富**：涵盖话剧、音乐剧、舞剧等多种类型
- **先锋实验**：经常上演先锋实验剧目

### 选座建议

| 位置 | 体验 | 价格 |
|:---|:---|:---|
| **前排中央** | 最佳观演位置，演员表情清晰可见 | 最贵 |
| **中排中央** | 视野开阔，性价比高 | 中等 |
| **后排中央** | 视野较好，价格实惠 | 较低 |
| **侧边座位** | 可能有遮挡，但价格便宜 | 最便宜 |

### 观剧礼仪

1. **提前到场**：建议提前30分钟到场，以免错过开场
2. **关闭手机**：演出期间请将手机调至静音或关机
3. **禁止拍照**：演出期间禁止拍照和录像
4. **保持安静**：演出期间请保持安静，不要大声说话
5. **鼓掌时机**：演出结束后可以鼓掌，谢幕时可以起身鼓掌
