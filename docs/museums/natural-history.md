# 北京自然博物馆

> "探索自然奥秘，恐龙世界精彩。"

## 简介

北京自然博物馆位于北京市东城区，是中国唯一的国家级大型自然博物馆。博物馆以生物进化为主线，展示了生物的起源、演化以及人与自然的关系。自然博物馆以恐龙化石闻名，是亲子游的热门目的地。

## 历史沿革

### 创建

北京自然博物馆前身是中央自然博物馆筹备处，筹建于1951年。1959年，新馆建成，1962年正式命名为"北京自然博物馆"。

### 发展

1980年代，博物馆进行了大规模改造，增加了恐龙展厅。2000年代，博物馆再次改造，增加了"恐龙公园"等新展览。

## 建筑特色

### 外观

北京自然博物馆建筑呈对称布局，正面有高大的柱廊，庄重大气。

### 内部

博物馆内部有多个展厅，布局合理，参观路线清晰。

## 常设展览

### 古爬行动物厅（恐龙世界）

这是自然博物馆最热门的展厅，展示了各种恐龙化石。

#### 恐龙化石

| 恐龙 | 说明 |
|:---|:---|
| **马门溪龙** | 中国最大的恐龙之一，长26米 |
| **许氏禄丰龙** | 中国第一具装架的恐龙化石 |
| **沱江龙** | 剑龙类恐龙 |
| **霸王龙** | 肉食性恐龙 |
| **三角龙** | 草食性恐龙 |

#### 恐龙公园

展厅内复原了恐龙生活的场景，有电子恐龙模型，可以动、可以叫，深受小朋友喜爱。

### 古哺乳动物厅

展示古代哺乳动物化石，包括：

- **黄河象**：中国最大的古象化石
- **猛犸象**：史前巨兽
- **披毛犀**：冰河时期的犀牛
- **剑齿虎**：史前猫科动物

### 无脊椎动物厅

展示各种无脊椎动物，包括昆虫、贝类、珊瑚等。

### 植物世界厅

展示植物的起源和演化，从藻类到被子植物。

### 人之由来厅

展示人类的起源和演化，从猿人到现代人。

#### 人类进化

- **古猿**：人类的祖先
- **直立人**：北京猿人、元谋人等
- **智人**：现代人的直接祖先

### 动物陈列厅

展示各种动物标本，包括：

- **哺乳动物**：狮子、老虎、熊等
- **鸟类**：各种鸟类标本
- **爬行动物**：蛇、龟、鳄鱼等
- **两栖动物**：蛙、蟾蜍等
- **鱼类**：各种鱼类标本

### 神奇的非洲厅

这是自然博物馆的特色展厅，展示了非洲的自然环境和动物。

#### 展厅特色

- **非洲草原**：复原非洲草原场景
- **非洲动物**：狮子、大象、长颈鹿、斑马等
- **非洲部落**：非洲原始部落文化

## 游览攻略

### 预约须知

1. **提前预约**：必须提前在官网或微信预约
2. **实名制**：需携带身份证
3. **周一闭馆**：注意开放时间
4. **票源紧张**：尤其节假日，需提前抢

### 推荐路线

**经典路线（约3-4小时）**：

恐龙世界 → 古哺乳动物厅 → 神奇的非洲 → 人之由来 → 动物陈列 → 植物世界 → 无脊椎动物

**亲子路线**：

恐龙世界（重点） → 神奇的非洲 → 动物陈列 → 人之由来

## 教育意义

北京自然博物馆是国家级科普教育基地，承担着向青少年普及自然科学知识的重要使命。

博物馆通过丰富的展品和生动的展览，让小朋友们了解生命的起源和演化，激发对自然科学的兴趣。
