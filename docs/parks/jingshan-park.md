# 景山公园

> "京华览胜第一处，俯瞰紫禁城的最佳位置。"

## 简介

景山公园位于北京内城南北中轴线的中心，南接故宫神武门，北对鼓楼，是明、清两代的皇家御苑。景山原名"万岁山"，是元、明、清三代的皇家园林。公园中心的景山是人工堆砌而成，是北京内城的制高点，也是俯瞰紫禁城的最佳位置。

## 历史沿革

### 元代

景山所在位置在元代为大都城内的小丘，名"青山"。元世祖忽必烈曾在此堆土成山，作为元大都的"镇山"。

### 明代

明永乐年间，为营建紫禁城，在此堆放挖护城河的泥土和拆除元宫殿的渣土，堆积成山，名"万岁山"，又称"煤山"。

### 清代

清顺治十二年（1655年），改名"景山"。乾隆年间大规模扩建，形成今日规模。

### 崇祯自缢

明崇祯十七年（1644年），李自成农民军攻入北京，崇祯皇帝朱由检仓皇出逃，在煤山东麓的一棵槐树下自缢身亡，明朝灭亡。

## 园林布局

景山公园主要景点沿中轴线分布，包括山门、绮望楼、五方亭、寿皇殿等。

### 五方亭

景山五峰之上，各建一亭，合称"五方亭"。

| 亭子 | 位置 | 特色 |
|:---|:---|:---|
| **万春亭** | 主峰（中峰） | 最高最大，供奉佛像 |
| **观妙亭** | 东一峰 | 供奉铜佛 |
| **周赏亭** | 东二峰 | 供奉关帝 |
| **富览亭** | 西一峰 | 供奉铜佛 |
| **辑芳亭** | 西二峰 | 供奉财神 |

**万春亭**是景山的制高点，海拔88.35米，是俯瞰紫禁城的最佳位置。

### 寿皇殿

寿皇殿位于景山北麓，是供奉清代皇帝先祖神像的地方。建筑形制仿太庙，规模宏大。

### 绮望楼

绮望楼位于景山南麓，是登山的入口。楼名取自"登高远望"之意。

### 崇祯自缢处

位于景山东麓，原有一棵老槐树，是崇祯皇帝自缢的地方。现在的槐树是后来补种的，原树在文革中被砍。

## 文化故事

### 崇祯皇帝殉国

1644年3月19日，李自成率领的农民军攻入北京。崇祯皇帝在皇宫中鸣钟召集百官，竟无人上朝。绝望之下，崇祯手持宝剑杀死了自己的女儿昭仁公主（长平公主被砍断一臂未死），又逼迫皇后周氏自尽。

随后，崇祯带着太监王承恩出东华门，直奔朝阳门，欲逃往成国公朱纯臣处避难，但未能进入。又奔安定门，城门已被封锁。走投无路之下，只得返回皇宫。

凌晨时分，崇祯带着王承恩登上煤山，在一棵槐树下自缢身亡，时年34岁。王承恩也在旁自缢殉主。

崇祯死前在衣襟上留下遗书：
> "朕自登极十七年，虽朕薄德匪躬，上干天咎，然皆诸臣误朕。朕死无面目见祖宗于地下，自去冠冕，以发覆面。任贼分裂朕尸，勿伤百姓一人。"

李自成入城后，发现了崇祯的尸体，以门板装殓，停放在东华门外。三天后，才草草安葬。

### 老槐树的故事

崇祯自缢的那棵老槐树，后来被称为"罪槐"。清代，这棵树被铁链锁住，以示对明朝灭亡的惩戒。

这棵老槐树一直存活到1960年代，在文革中被砍。现在的槐树是后来补种的，树龄仅几十年。

### 景山的镇山之说

传说元世祖忽必烈营建大都时，命人堆土成山，作为"镇山"，以镇压北京城的风水。明永乐年间扩建紫禁城，又将渣土堆于此山，使其更加高大。

### 煤山名称的由来

明代称景山为"煤山"，有人说是因为山中埋藏煤炭，以备不时之需；也有人说是因为山上堆满渣土，形似煤堆。后一种说法更为可信。

## 游览攻略

### 最佳游览时间

| 时间 | 看点 |
|:---|:---|
| **日出** | 看太阳从紫禁城升起 |
| **日落** | 看夕阳中的紫禁城 |
| **夜景** | 看紫禁城灯光（仅周五、周六及节假日亮灯） |
| **雪天** | 看"紫禁城雪景" |

### 推荐路线

**经典路线（约1-1.5小时）**：

南门 → 绮望楼 → 登山 → 万春亭（俯瞰紫禁城） → 观妙亭 → 周赏亭 → 富览亭 → 辑芳亭 → 下山 → 寿皇殿 → 崇祯自缢处 → 东门

### 实用贴士

1. **最佳拍摄点**：万春亭正前方，可俯瞰整个紫禁城
2. **日出日落**：如想看日出日落，需提前查看时间，公园开门/关门时间随季节调整
3. **避开人流**：早上7点前或下午4点后，避开旅行团
4. **门票便宜**：门票仅2元，是北京最便宜的景点之一
5. **周边游览**：南门紧邻故宫北门，可安排在一起

### 与故宫的关系

景山与故宫紧密相连，南门正对故宫神武门。游览完故宫后，从神武门出，直接进入景山公园，登上万春亭俯瞰紫禁城全景，是很多游客的必选项目。

## 文化价值

景山不仅是一座公园，更是明王朝灭亡的见证。崇祯皇帝殉国的故事，至今仍让人唏嘘。

同时，景山也是北京城的风水宝地。它位于北京内城的中轴线上，与故宫、天安门、正阳门、永定门连成一线，是北京城规划的重要组成部分。
