# 香山公园

![香山公园](https://images.unsplash.com/photo-UN1Y_14Heo0?w=1200)

> "香山红叶红满天，层林尽染醉秋烟。"

## 简介

香山公园位于北京市海淀区西郊，是一座具有皇家园林特色的山林公园。香山以其秀美的自然风光和丰富的历史文化闻名，尤以秋季红叶著称。香山公园占地面积约160公顷，最高峰海拔557米，是北京市民登高赏秋的首选之地。

## 历史沿革

### 金代始建

香山在金代就已经是皇家园林。金世宗在此修建了香山寺，开启了香山的皇家园林历史。

### 明清时期

明清时期，香山得到了大规模的营建。乾隆皇帝在此修建了静宜园，使香山成为皇家园林的巅峰之作。

#### 重要事件

| 时间 | 事件 |
|:---|:---|
| 1186年 | 金世宗修建香山寺 |
| 1745年 | 乾隆皇帝修建静宜园 |
| 1860年 | 英法联军焚毁静宜园 |
| 1956年 | 香山公园正式开放 |
| 1993年 | 香山公园被评为AAAA级景区 |

## 主要景点

### 香山红叶

香山红叶是香山公园最著名的景观，每年秋季吸引大量游客前来观赏。

#### 红叶观赏

| 树种 | 特点 | 最佳观赏时间 |
|:---|:---|:---|
| **黄栌** | 香山红叶的主体 | 10月中旬-11月上旬 |
| **枫树** | 叶色鲜红 | 10月中旬-11月上旬 |
| **银杏** | 叶色金黄 | 10月中旬-11月上旬 |
| **火炬树** | 叶色火红 | 10月中旬-11月上旬 |

#### 最佳观赏地点

- **香炉峰**：香山最高峰，俯瞰红叶全景
- **玉华岫**：黄栌最为集中的地方
- **森玉笏**：红叶与古建筑相映成趣
- **流憩亭**：俯瞰香山红叶的最佳位置

### 香炉峰

香炉峰是香山的最高峰，海拔557米，因形似香炉而得名。

#### 登顶路线

| 路线 | 长度 | 难度 | 用时 |
|:---|:---|:---|:---|
| **北线** | 约2.5公里 | 中等 | 约1.5小时 |
| **南线** | 约2公里 | 较易 | 约1小时 |
| **中线** | 约2.2公里 | 中等 | 约1小时 |

### 碧云寺

碧云寺位于香山东麓，是一座历史悠久的佛教寺院。

#### 主要建筑

- **山门**：碧云寺的正门
- **天王殿**：供奉四大天王
- **大雄宝殿**：主殿，供奉释迦牟尼
- **罗汉堂**：供奉五百罗汉
- **金刚宝座塔**：明代建筑，是碧云寺的标志性建筑

### 卧佛寺

卧佛寺位于香山东麓，因寺内供奉一尊巨大的卧佛而得名。

#### 主要看点

- **卧佛**：长5米余的铜铸卧佛
- **千年古树**：寺内有多棵千年古树
- **古建筑**：保存完好的古建筑群

## 游览攻略

### 推荐路线

**红叶观赏路线（约4小时）**：

东门 → 勤政殿 → 玉华岫 → 森玉笏 → 香雾窟 → 香炉峰 → 碧云寺 → 卧佛寺 → 西门

**登山路线（约3小时）**：

北门 → 香炉峰 → 香雾窟 → 森玉笏 → 玉华岫 → 勤政殿 → 东门

### 实用贴士

1. **红叶季**：10月中旬至11月上旬是最佳观赏期，但人很多，建议早去
2. **登山准备**：穿舒适的鞋子，带足饮水
3. **拍照位置**：香炉峰、玉华岫、森玉笏都是绝佳拍照点
4. **交通建议**：红叶季建议乘坐公共交通，自驾停车困难
5. **周边游览**：可以与碧云寺、卧佛寺一起游览

### 交通方式

| 方式 | 路线 |
|:---|:---|
| **地铁** | 西郊线香山站，步行约10分钟 |
