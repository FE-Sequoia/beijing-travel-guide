# 苏州街（买卖街）

![苏州街](https://images.unsplash.com/photo-1566455123644-5c94561acf9b?w=1200&q=80)

> "一条买卖街，半部帝王心。"
> —— 紫禁城外的人间烟火，皇家深宫里的"过家家"

## 简介

苏州街位于颐和园后湖（后溪河）两岸，又称"买卖街"，是乾隆年间仿江南水乡市肆而建的一条皇家商业水街。它全长约300米，两岸鳞次栉比地排列着60余家店铺：茶楼、酒肆、药铺、绸缎庄、钱庄、帽店、糕点铺、古玩店……青瓦白墙、招幌高悬，小桥流水穿街而过，恍如把姑苏阊门外的十里繁华整体搬进了皇家禁苑。

这条街的诞生源于乾隆的江南情结。乾隆六下江南，对苏州水乡的市井风情念念不忘，回京后便命人在后湖两岸仿建水街，让无法出宫的帝后们也能"足不出园逛苏州"。

咸丰十年（1860年）苏州街随清漪园一同毁于英法联军大火，此后荒废百余年。1986年按原样重建，1990年正式对外开放，是颐和园中最年轻的"古迹"。

| 项目 | 内容 |
|:---|:---|
| **全长** | 约300米，沿后溪河两岸展开 |
| **店铺** | 60余家 |
| **格局** | 前店后河，跨河拱桥相连，码头临水，招幌高悬 |
| **风格** | 江南水街风貌，青瓦白墙，尺度刻意缩小以求亲切 |
| **游览方式** | 沿河岸步行或乘摇橹船穿行水街 |

苏州街的妙处在于"小"与"真"：店铺门脸刻意做得比实物略小，沿河一字排开，临水设码头，跨河架拱桥，岸上行人摩肩，水中画舫往来，江南水街的格局一应俱全。

店铺虽为复建，招幌、匾额皆按清宫档案复原，行走其间，耳边是叫卖声与丝竹声，恍惚回到二百年前的姑苏城外。主要店铺类型包括：

| 业态 | 代表店铺 |
|:---|:---|
| 茶楼酒肆 | 各式茶馆、酒楼，可临水品茗 |
| 绸缎百货 | 云锦斋等绸缎庄 |
| 钱庄当铺 | 品泉斋等钱庄 |
| 糕点小吃 | 漱芳斋等点心铺 |
| 医药文玩 | 药铺、古玩铺、字画铺 |

## 历史故事

### 太监宫女扮"店员"

苏州街建成时，乾隆定下规矩：街上的"掌柜""伙计""茶博士"全部由太监宫女扮演，商品则是内务府预备的奇珍杂货。每年皇帝携后妃驾临"逛街"，太监们便扯开嗓子吆喝，宫女们扮作买办讨价还价，一时市声鼎沸。

据说乾隆最爱在绸缎庄"购物"，由太监扮的掌柜故意抬价，再由宫女扮的"顾客"砍价，最终以"御笔钦定"成交，君臣尽欢。深宫高墙之内，这是一场心照不宣的"人间烟火"cosplay——帝王要的不是买卖，而是那份寻常百姓才有的逛街之乐。

### 灰烬中的重生

1860年英法联军的大火烧了三天三夜，后湖两岸的买卖街化为焦土，此后一百二十年里只余断壁残垣与河床荒草。1986年，园林部门依据清宫《颐和园图》和档案记载，按原址、原制、原貌重建苏州街，连店铺的字号招幌都逐一考证复原。

1990年开放当日，许多老北京赶来"认亲"——他们小时候听祖辈讲过的买卖街，终于从故事里走了回来。如今街上的店员换成了真营业员，卖着茶叶、糕点和文创，那份市井气，竟真的续上了。

### 后溪河的水系密码

苏州街不是凭空造景，它依托的后溪河（后湖）是一条精心设计的人工水道：引昆明湖水东流，绕万寿山北麓蜿蜒而过，既解决了后山的排水，又造出了"水随山转、街因水成"的江南意境。重建时考古清理出乾隆朝的河底石基与驳岸遗址，成为复原的坐标依据——可以说，今天的苏州街是踩着两百多年前的地基重新站起来的。

## 必看亮点

1. **水街格局**：两岸店铺夹河而立，小桥流水穿行其间，是北方园林中罕见的江南水街实景。
2. **招幌匾额**：留意各家店铺的老式招幌与匾额，云锦斋、品泉斋、漱芳斋……一块匾就是一门营生。
3. **摇橹船穿街**：乘摇橹船从后湖穿过苏州街，船娘摇桨、两岸吆喝，最接近当年帝后游街的体验（需另购票）。
4. **临水戏台**：街中小戏台偶有民俗表演，评弹、京韵大鼓轮番上阵，逛累了正好歇脚听曲。
5. **北宫门起点**：苏州街紧邻北宫门，从后门进园的游客第一眼便是它，"未见湖山，先闻市声"的开场别有设计。

## 拍照机位

1. **跨河拱桥上**：站在小拱桥上顺河拍，两岸店铺招幌层层叠叠，最出"姑苏繁华图"的感觉。
2. **临水码头**：低机位拍河面倒影与画舫，雨后或清晨水面平静时效果最佳。
3. **招幌特写**：各色布招幌随风招展，长焦压缩后颇有古意。

## 参观贴士

1. 苏州街为园中园，需持联票或另购10元门票；街内购物消费另计，价格与普通景区相仿。
2. 建议上午早些抵达，店铺开门、游人尚少，拍照最有"包场"感。
3. 街全长仅300米，步行15分钟即走完，若乘摇橹船放慢节奏体验更佳。
4. 从北宫门入园后先逛苏州街，再上山经四大部洲至[佛香阁](/landmarks/summer-palace/foxiang-ge)，是[北宫门线](/landmarks/summer-palace/)的标准开场。
5. 后湖一带还有[谐趣园](/landmarks/summer-palace/xiequ-yuan)（东端）和须弥灵境遗址（中段），可与苏州街串联成"后山半日游"，人少清净，与前山的喧闹恍如两园。
6. 街内小吃可以尝鲜，但正餐建议留到园外的海淀老字号，性价比更高。

## 常见问题

**Q：苏州街需要单独购票吗？**

苏州街属园中园，单独购票10元；持颐和园联票可直接进入。

**Q：苏州街里的东西是真的在卖吗？**

是真的开门营业：茶叶、糕点、文创、丝绸都可实买，也有临水茶馆可坐——当然，价格按景区标准。

**Q：苏州街是乾隆时期的原物吗？**

不是。原街1860年被英法联军焚毁，荒废百余年后于1986年按清宫图档原址重建，1990年开放，是颐和园最年轻的"古迹"。

**Q：苏州街适合带孩子吗？**

很适合。小吃、叫卖、摇橹船、民俗表演，孩子全程有得看有得玩，比殿宇陈列更有吸引力。
