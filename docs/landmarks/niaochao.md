# 鸟巢与水立方

![鸟巢](https://images.unsplash.com/photo-rLCooDyRYqA?w=1200)

> "钢铁编织的鸟巢，见证了一个民族的腾飞。"

## 简介

国家体育场"鸟巢"和国家游泳中心"水立方"位于北京奥林匹克公园中心区，是2008年北京奥运会的标志性建筑。鸟巢以其独特的钢结构外观闻名世界，水立方则以气泡状的ETFE膜结构成为建筑奇观。

## 鸟巢（国家体育场）

### 建筑特色

鸟巢由瑞士建筑师赫尔佐格和德梅隆与中国建筑师合作设计，是世界上规模最大、技术含量最高的钢结构建筑之一。

#### 建筑参数

| 项目 | 数据 |
|:---|:---|
| **占地面积** | 21公顷 |
| **建筑面积** | 25.8万平方米 |
| **座位数** | 91000个 |
| **用钢量** | 4.2万吨 |
| **高度** | 69.2米 |

#### 设计灵感

鸟巢的外观像一个巨大的鸟巢，由24根钢柱和大量钢结构编织而成。这种设计既体现了中国传统文化中"天圆地方"的理念，又具有现代建筑的前卫感。

### 主要功能

鸟巢不仅是体育场馆，还是大型演出、展览等活动的举办地。

#### 主要活动

- **体育赛事**：中超、亚冠等足球比赛
- **演唱会**：国内外知名歌手演唱会
- **旅游参观**：日常对外开放参观

## 水立方（国家游泳中心）

### 建筑特色

水立方由澳大利亚PTW建筑事务所设计，外观像一池碧蓝的清水，由3000多个ETFE膜结构气泡组成。

#### 建筑参数

| 项目 | 数据 |
|:---|:---|
| **占地面积** | 6.29公顷 |
| **建筑面积** | 8万平方米 |
| **座位数** | 17000个 |
| **膜结构数量** | 3607个 |
| **高度** | 30米 |

#### 设计灵感

水立方的设计灵感来源于"水分子"，外观由无数个"气泡"组成，象征着水的生命力和灵动性。

### 主要功能

水立方目前主要作为游泳馆和旅游观光点对外开放。

#### 主要设施

- **比赛泳池**：国际标准游泳池
- **嬉水乐园**：适合家庭游玩的水上乐园
- **冰壶场地**：冬季开放冰壶体验

## 奥林匹克公园

### 奥林匹克塔

奥林匹克塔是北京的新地标，高246.8米，塔顶设有观景台，可以俯瞰整个奥林匹克公园。

### 奥林匹克森林公园

奥林匹克森林公园位于鸟巢和水立方的北侧，是北京最大的城市公园之一。

## 游览攻略

### 推荐路线

**精华路线（约2小时）**：

地铁8号线奥体中心站 → 鸟巢外观 → 鸟巢内部参观 → 水立方外观 → 水立方内部参观 → 奥林匹克公园散步

**深度路线（约半天）**：

上午：鸟巢 → 水立方 → 奥林匹克塔
下午：奥林匹克森林公园 → 奥林匹克公园
晚上：夜景灯光秀

### 实用贴士

1. **最佳时间**：傍晚时分和夜晚，灯光亮起后景色最美
2. **夜景拍摄**：建议携带三脚架，夜景拍摄效果更佳
3. **门票购买**：可以购买联票，更划算
4. **交通便利**：地铁8号线直达，非常方便
5. **周边餐饮**：奥林匹克公园周边有大量餐厅和购物中心

## 文化意义

鸟巢和水立方是2008年北京奥运会的象征，也是中国改革开放成就的标志。它们不仅代表了中国的建筑水平，更代表了中国的国际形象。
