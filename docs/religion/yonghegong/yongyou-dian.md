# 永佑殿

![永佑殿红墙殿宇](https://images.unsplash.com/photo-1590301729964-23833732ee04?w=1200&q=80)

> "龙寝旧地，梓宫曾停——三尊白檀佛像静默垂目，东西配殿里，藏医与历算的学问仍在流传。"

## 简介

永佑殿是雍和宫中轴线上的第四进大殿，"永佑"取"永久佑护"之意。这里原是雍亲王府的内寝正殿，即雍亲王胤禛日常起居的寝殿；胤禛即位后，此处成为行宫内殿。雍正十三年（1735年）雍正帝驾崩，梓宫自圆明园移奉雍和宫，即停放于此殿，随后殿宇改覆黄瓦。一座寝殿，由起居而停灵，由停灵而奉佛，身世之跌宕，全寺无出其右。

改寺后，永佑殿内供奉三尊白檀木雕佛像：中为无量寿佛（阿弥陀佛的报身相），东为药师佛，西为狮吼佛。殿前东西配殿，则分别是藏传佛教四大扎仓中的医药殿与数学殿，是清代藏医学与历算学的研习重镇。

## 建筑与景观

| 项目 | 规格与内容 |
|:---|:---|
| 殿宇规制 | 面阔五间，单檐歇山黄琉璃瓦顶，原王府内寝正殿 |
| 正中主供 | 无量寿佛（白檀木雕），主长寿之法 |
| 东侧 | 药师佛（白檀木雕），主消灾祛病、延寿安康 |
| 西侧 | 狮吼佛（白檀木雕），骑狮作吼，以降伏毒龙、护持正法 |
| 东配殿 | 原医药殿（曼巴扎仓），研习藏医藏药 |
| 西配殿 | 原数学殿（丁科尔扎仓），研习历法、数学与天文 |

三尊佛像均以整块白檀木雕成，木质温润，刀法圆熟，历二百余年仍香气隐约。无量寿佛居中，寓意福寿绵长；药师佛手托药钵，疗愈众生身心之疾；狮吼佛则是观音菩萨的忿怒化身，传说以狮子吼声降伏兴风作浪的毒龙，护一方安宁。

### 四大扎仓中的两所学府

雍和宫改寺后仿照拉萨三大寺设立扎仓（学院），共四所，分设于中路殿宇及两侧配殿：

| 扎仓 | 汉名 | 研习内容 |
|:---|:---|:---|
| 参尼扎仓 | 显宗殿 | 显宗经论、因明学、佛学哲学 |
| 居巴扎仓 | 密宗殿 | 密宗仪轨、坛城修持、灌顶传承 |
| 曼巴扎仓 | 医药殿（东配殿） | 藏医《四部医典》、药材炮制与诊疗 |
| 丁科尔扎仓 | 数学殿（西配殿） | 时轮历算、数学、天文推步 |

医药殿所传藏医，以《四部医典》为根本，兼习药浴、针灸与藏药配伍，学僧兼通医理与药理；数学殿所传时轮历法，是蒙藏地区沿用数百年的历法体系，学僧须精通天文测算与数理推演，每年编制颁行历书。清代蒙古、藏地学僧来京，多在此深造考核，学成者返回本寺任教——永佑殿前的两座配殿，曾是名副其实的"皇家佛学院"，也是藏医药与历算学向内地传播的枢纽。

## 历史故事

### 梓宫停灵与黄瓦之变

雍正十三年（1735年）八月二十三日，雍正帝驾崩。梓宫先奉于乾清宫，随后移至雍和宫，停放于这座昔日的寝殿之中。乾隆帝即位后，为奉父灵，将永佑殿及中路主要殿宇的绿瓦尽改黄瓦。同年十一月，梓宫由此启行，奉安河北易县泰陵。寝殿变灵堂、绿瓦变黄瓦——永佑殿是雍正—乾隆之际那段国丧岁月最直接的见证者。

### 寝殿缘何奉"寿"

停灵之地改奉无量寿佛，并非偶然。藏传佛教视生死为轮回，无量寿佛（阿弥陀佛）接引往生、赐予长寿，药师佛疗疾延寿，狮吼佛降魔护法——三尊佛的组合，既是对先帝的追荐超拔，也是对在世者的福寿祝祷。"永佑"之名，恰是这份心意的写照。

### 章嘉国师与雍和宫的规制

雍和宫改寺后的殿堂布局、佛像供设与学经制度，多出自三世章嘉呼图克图若必多吉的擘画。章嘉国师是乾隆帝的佛法上师，自幼与弘历（乾隆）同窗共读，一生辅佐清廷处理蒙藏宗教事务。正是在他的主持规划下，雍和宫仿拉萨寺院设立四大扎仓、安设显密诸佛，使一座王府行宫蜕变为制度完备的格鲁派大寺。今日永佑殿的白檀三佛与东西配殿的学经格局，仍可见当年规划之遗意。

## 必看亮点

- **雍正寝殿与停灵处**：王府内寝、帝王梓宫、佛寺正殿三重身份叠加，全寺历史感最浓的一殿。
- **三尊白檀木雕佛像**：无量寿佛、药师佛、狮吼佛，均为整木雕刻，近三百年仍保存完好。
- **医药殿与数学殿**：东西配殿即藏传佛教四大扎仓中的两所，藏医与历算的传承之地。
- **黄瓦规制**：殿顶黄琉璃瓦即雍正停灵时所改，是"因丧升格"的实物印记。
- **长寿祈福**：无量寿佛主长寿之法，是许多信众专程叩拜之处。

## 参观贴士

- 三尊白檀佛像木质温润，细观刀工，可辨与铜铸佛像的不同质感。
- 东西配殿常设藏医药、历算相关展陈，时间允许可入内浏览。
- 为长辈祈福者，可在无量寿佛前随喜供养、默祷安康。
- 此殿游客较主殿少，适宜静立片刻，感受寝殿旧地的肃穆氛围。
- 殿内禁止拍照，请勿触摸佛像与供器。
