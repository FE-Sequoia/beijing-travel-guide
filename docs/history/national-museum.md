## 简介

中国国家博物馆位于天安门广场东侧，是世界上建筑面积最大的博物馆，也是中国最高历史文化艺术殿堂。馆藏文物超过140万件，系统展示了中国从远古到近现代的历史发展进程。

## 历史沿革

### 前身溯源

| 时间 | 事件 |
|:---|:---|
| 1912年 | 国立历史博物馆筹备处成立 |
| 1926年 | 正式开馆，馆址在故宫的午门和端门 |
| 1949年 | 新中国成立后更名为中国历史博物馆 |
| 1959年 | 迁入新馆（现址） |
| 2003年 | 中国历史博物馆和中国革命博物馆合并组建中国国家博物馆 |
| 2011年 | 新馆建成开放 |

### 中国革命博物馆

中国革命博物馆的前身是1950年成立的中央革命博物馆筹备处，主要收藏中国共产党领导中国革命的历史文物。

### 两馆合并

2003年2月28日，中国历史博物馆和中国革命博物馆合并组建中国国家博物馆。

### 改扩建工程

2007年至2011年，国博进行了历时近4年的改扩建工程。新馆建筑面积近20万平方米，展厅面积6.5万平方米，成为世界上单体建筑面积最大的博物馆。

## 镇馆之宝

### 后母戊鼎

商代后期铸造，是世界现存最大、最重的青铜器，重达832.84公斤。

![后母戊鼎](https://images.unsplash.com/photo-1566127444979-b3d2b654e3d7?w=800)

### 四羊方尊

商代晚期青铜礼器，尊身四角各塑一羊，被誉为"臻于极致的青铜典范"。

### 金缕玉衣

西汉时期的帝王葬服，用2008片玉片以金丝编缀而成，全长1.88米。

## 文化故事

### 后母戊鼎的故事

后母戊鼎1939年出土于河南安阳，当时正值抗日战争时期。当地村民为保护国宝，将其重新埋藏，直至抗战胜利后才取出。
