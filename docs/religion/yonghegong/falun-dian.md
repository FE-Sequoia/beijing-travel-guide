# 法轮殿

![法轮殿藏式殿宇](https://images.unsplash.com/photo-1782915763074-87f3c2c62fc1?w=1200&q=80)

> "殿顶五塔镏金，殿中罗汉成山——藏汉合璧的法轮殿，是雍和宫法事最盛之处，也是宗喀巴大师在京城的精神道场。"

## 简介

法轮殿是雍和宫僧众诵经礼佛、举行大型法事的主殿，也是全寺藏式风格最鲜明的建筑。殿名取"法轮常转"之意——佛法如轮，辗转不息，摧破烦恼，普利有情。在雍亲王府时期，此处相传为福晋（雍亲王嫡福晋，即后来的孝敬宪皇后）的内寝院落；改寺之后扩建为大经堂，成为蒙藏学僧云集、法会经声不断的佛学中心。

殿内正中供奉藏传佛教格鲁派创始人宗喀巴大师的巨型铜像，大师背后陈设举世闻名的"五百罗汉山"——以整块紫檀木雕成山峦，五百尊金属罗汉散布其间，与万福阁白檀弥勒、金丝楠木佛龛并称"雍和宫三绝"。

## 建筑与景观

| 项目 | 规格与内容 |
|:---|:---|
| 平面形制 | 十字形平面，藏汉合璧：汉式殿堂之上加藏式平顶与天窗 |
| 屋顶 | 黄琉璃瓦屋面，设五座镏金藏式小宝塔 |
| 殿内主供 | 宗喀巴大师铜像，高约6.1米，鎏金庄严 |
| 大师背后 | 五百罗汉山（紫檀木雕山 + 五百尊罗汉） |
| 殿内陈设 | 坛城（曼荼罗）、经卷、法器等 |

法轮殿外观为汉式大殿，殿顶却突起藏式平顶天窗，天窗之上矗立五座镏金小塔——五塔象征须弥山五形与四大部洲，也对应格鲁派尊崇的五方佛。汉地的大屋顶与藏式的金塔共处一殿，正是雍和宫"汉藏一家"最直观的建筑宣言。

### 宗喀巴大师像

宗喀巴（1357—1419）是藏传佛教格鲁派（黄教）的创始人，第一世达赖喇嘛与第一世班禅额尔德尼皆出其门下。殿内铜像高约6.1米，大师头戴黄色尖顶僧帽，双手结转法轮印，左右各出一茎莲花，分托经箧与宝剑——寓意智慧如剑断惑、经教如灯照世。像前设有讲经法座，昔日大法会期间，蒙藏高僧即在此升座说法。

### 五百罗汉山

五百罗汉山陈设于宗喀巴大师像背后，以一整块紫檀木（一说红木）雕成普陀洛伽山景：峰峦起伏、洞壑幽深，松柏亭塔错落其间；五百尊罗汉以金、银、铜、铁、锡五金铸成，或坐或立，或降龙或伏虎，或诵经或禅定，散布于山石之间，无一雷同。此山为乾隆年间雕造，工程浩大，是清代木雕与铸造工艺的巅峰之作。20世纪初庚子之变及此后动荡中，部分罗汉散佚损毁，现存已不足五百之数，更觉弥足珍贵。

### 坛城（曼荼罗）

殿内陈设有坛城（梵文 Mandala，曼荼罗），即密宗观修用的宇宙坛场模型：方形坛城之外设圆形护轮，中央主尊安住，四方四佛、八方菩萨各依方位排列，象征佛国净土的完整秩序。僧人修法时观想坛城，即是"入佛国土"的修持法门。坛城结构严谨、毫厘不差，是藏传佛教建筑与数理观念的浓缩。

### "雍和宫三绝"

法轮殿的五百罗汉山，与万福阁白檀木大佛、照佛楼檀香木佛龛并称"雍和宫三绝"：

| 宝物 | 所在 | 特色 |
|:---|:---|:---|
| 五百罗汉山 | 法轮殿 | 整块紫檀木雕山景 + 五金铸造五百罗汉 |
| 白檀木大佛 | 万福阁 | 26米独木弥勒立像，吉尼斯世界纪录 |
| 檀香木佛龛 | 照佛楼 | 整块檀香木雕佛龛，透雕九龙盘旋 |

三绝皆为"一木一器"的极致之作，集中在寺院最北三进院落，建议连贯参观。

## 历史故事

### 福晋内寝变经堂

相传法轮殿一带原为雍亲王府福晋的起居内寝。1744年改寺，王府最私密的内院被改建为最开放的经堂——僧侣日诵、大法会、跳布扎（打鬼）法事皆在此举行。从闺阁到法堂，空间的转身，恰是雍和宫"舍家为寺"的缩影。

### 罗汉山的沧桑

五百罗汉山雕成之后，曾完好陈列两百余年。清末民初战乱频仍，罗汉屡遭散失，山石亦有残损。所幸主体紫檀山景与大部分罗汉仍存，今日所见虽不足五百之数，其规模与工艺依然冠绝京华。

### 岁末"打鬼"与正月祈愿

法轮殿是雍和宫法事活动的中心。清代定制，每年岁末寺院举行"打鬼"（跳布扎，即藏传佛教羌姆法舞）法会：僧人头戴护法神面具，身着彩衣，在鼓钹诵经声中起舞，寓意驱除邪魔、迎祥纳福，届时王公大臣、蒙藏僧俗云集观礼，为京师一大盛况。正月新春又有祈福法会，诵经祝祷国泰民安。这一传统在今天的雍和宫仍以不同形式延续，法轮殿的经声梵呗从未真正断绝。

## 必看亮点

- **藏汉合璧建筑**：十字形平面、汉顶藏窗、五座镏金小塔，全寺建筑融合度最高的一殿。
- **宗喀巴大师铜像**：高约6.1米的鎏金巨像，京城供奉宗喀巴最重要的道场。
- **五百罗汉山**："雍和宫三绝"之一，紫檀雕山与五金罗汉的旷世组合。
- **坛城陈设**：立体曼荼罗展现佛教宇宙观，密法艺术的精华。
- **法会现场**：若有缘逢大法会，可闻经声梵呗，感受活态的宗教传承。

## 参观贴士

- 罗汉山体量庞大、细节繁复，建议预留时间逐区细观罗汉姿态。
- 殿内光线较暗且禁止拍照，可入殿后先静立片刻待眼睛适应。
- 瞻仰宗喀巴大师像时，留意其尖顶黄帽与肩头莲经、莲剑的标识。
- 殿为诵经重地，请绕行礼让僧人，勿阻挡法座与经堂通道。
