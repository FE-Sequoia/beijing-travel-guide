# 北京名胜古迹

> "千年帝都，古迹如云。"

北京作为六朝古都，拥有丰富的历史文化遗产。从皇家宫殿到万里长城，从祭天圣地到园林杰作，每一处古迹都诉说着中华文明的辉煌。

## 皇家建筑

### [故宫博物院](/landmarks/forbidden-city)

世界最大宫殿建筑群，明清两代皇宫，世界文化遗产。

- **门票**：旺季60元，淡季40元
- **建议时长**：半天至一天
- **看点**：太和殿、乾清宫、御花园、珍宝馆

### [天坛](/landmarks/temple-of-heaven)

明清皇帝祭天祈谷之地，世界文化遗产。

- **门票**：联票28元
- **建议时长**：3-4小时
- **看点**：祈年殿、回音壁、圜丘坛

### [颐和园](/landmarks/summer-palace)

中国古典园林巅峰之作，世界文化遗产。

- **门票**：联票60元
- **建议时长**：半天至一天
- **看点**：佛香阁、长廊、十七孔桥、昆明湖

### [圆明园](/landmarks/yuanmingyuan)

"万园之园"遗址，爱国主义教育基地。

- **门票**：联票25元
- **建议时长**：半天至一天
- **看点**：西洋楼遗址、大水法、十二生肖

### [北海公园](/landmarks/beihai)

现存最古老皇家园林，白塔琼岛。

- **门票**：联票20元
- **建议时长**：2-3小时
- **看点**：白塔、九龙壁、让我们荡起双桨

### [景山公园](/landmarks/jingshan)

北京内城制高点，俯瞰紫禁城最佳位置。

- **门票**：2元
- **建议时长**：1-2小时
- **看点**：万春亭、崇祯自缢处

## 长城

### [八达岭长城](/landmarks/badaling)

万里长城精华段，最早开放的长城。

- **门票**：40元
- **建议时长**：3-4小时
- **看点**：北八楼、好汉坡、关城

### [慕田峪长城](/landmarks/mutianyu)

植被覆盖率最高，风景秀丽。

- **门票**：40元
- **建议时长**：4-5小时
- **看点**：滑道、缆车、野长城

### [司马台长城](/landmarks/simatai)

保存最原始，夜景独特。

- **门票**：40元
- **建议时长**：半天
- **看点**：夜游长城、古北水镇

### [居庸关](/landmarks/juyongguan)

"天下第一雄关"，云台石刻。

- **门票**：35元
- **建议时长**：2-3小时
- **看点**：云台、关城、云台石刻

## 古城遗迹

### [明城墙遗址](/landmarks/city-wall)

北京仅存的一段明城墙。

- **门票**：10元
- **看点**：东南角楼、城墙遗址

### [前门大街](/landmarks/qianmen)

老北京商业街，百年老字号。

- **门票**：免费
- **看点**：正阳门、铛铛车、老字号

### [钟鼓楼](/landmarks/drum-tower)

古代报时中心，老北京地标。

- **门票**：联票30元
- **看点**：鼓楼、钟楼、击鼓表演

## 游览建议

### 经典三日游

| 天数 | 上午 | 下午 | 晚上 |
|:---:|:---|:---|:---|
| Day1 | 天安门广场 | 故宫博物院 | 王府井 |
| Day2 | 八达岭长城 | 明十三陵 | 后海酒吧街 |
| Day3 | 天坛 | 颐和园 | 前门大街 |

### 门票预约须知

1. **故宫**：必须提前7天预约，现场不售票
2. **国博**：提前1-7天预约，周一闭馆
3. **长城**：建议提前预约，旺季限流
4. **颐和园/天坛**：可现场购票，但旺季建议预约

### 注意事项

- 穿舒适的鞋子，每天步行量很大
- 带足饮水，景区内水较贵
- 注意防晒，夏季北京紫外线强
- 保管好证件，很多景点需刷身份证
