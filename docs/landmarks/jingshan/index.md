# 景山公园

![景山公园俯瞰故宫](https://images.unsplash.com/photo-1736237174975-0be4f327f35d?w=1200&q=80)

> "会当凌绝顶，一览紫禁小。"——站在北京内城制高点，六百年的皇家气象尽收眼底。

## 简介

景山公园位于北京故宫北侧，地处北京内城的几何中心，是元、明、清三代的皇家御苑。公园占地23公顷，园内主峰海拔88.7米，是北京内城的制高点。这里曾是明代"万岁山"、清代"景山"，见证了中国历史上最惊心动魄的一幕——1644年明思宗朱由检在此自缢殉国。

如今的景山公园，既是市民晨练、赏花的日常去处，也是游客俯瞰紫禁城全景的绝佳观景点。万春亭上的日出日落，五方亭间的四季轮转，让这座有着近千年历史的皇家园林，在当代依然焕发着独特的魅力。无论是摄影爱好者追逐的"京城天际线"，还是历史爱好者凭吊的崇祯殉国处，景山都以其独特的位置与厚重的历史，成为北京旅行不可错过的一站。

## 推荐游览路线

### 经典半日路线（约2.5小时）

**南门进 → 崇祯殉国处 → 万春亭 → 五方亭 → 西门出**

1. **南门进**：从景山公园南门入园，迎面便是皇家园林的气派格局。
2. **崇祯殉国处**（东侧）：沿东侧步道前行，参观"明思宗殉国处"石碑与老槐树遗址，感受历史的沉重。
3. **万春亭**：沿主路登顶，在景山最高点俯瞰故宫全景、北海白塔、钟鼓楼，是全程的高潮。
4. **五方亭**：从山顶向西，依次游览周赏亭、观妙亭、辑芳亭、富览亭，欣赏五方亭的建筑之美。
5. **西门出**：从西门离园，可顺路前往北海公园或什刹海继续游览。

**替代路线：东门出**

若想前往北大红楼、中国美术馆一带，可从东门出园，途经绮望楼外观后继续向东。

## "京城天际线"拍照点

景山万春亭是北京最具标志性的城市天际线拍摄点之一。站在海拔88.7米的制高点，向南可拍摄故宫全景中轴线，向北可远眺钟鼓楼与奥林匹克塔，向西可见北海白塔与国家大剧院，向东则能捕捉到央视大楼（CCTV总部）的现代轮廓。

### 最佳机位

| 方向 | 拍摄内容 | 最佳时间 |
|------|----------|----------|
| **南** | 故宫全景中轴线 | 日出后1小时 / 日落前1小时 |
| **北** | 钟鼓楼、奥林匹克塔 | 傍晚蓝调时刻 |
| **西** | 北海白塔、国家大剧院 | 日落时分 |
| **东** | 央视大楼天际线 | 夜间灯光亮起后 |

> **摄影提示**：建议携带广角镜头（16-35mm）和三脚架。节假日游客众多，清晨6:30开园时前往可获得最佳机位。

### 春季·牡丹

![景山牡丹](https://images.unsplash.com/photo-1754119431062-b818396bc36a?w=1200&q=80)

景山公园是北京最著名的牡丹观赏地之一，每年4月中旬至5月中旬举办"景山牡丹节"，园内种植牡丹、芍药万余株，涵盖姚黄、魏紫、赵粉、豆绿等传统名品。五方亭下的牡丹园，更是花开时节游人如织。

### 冬季·雪

![雪后景山](https://images.unsplash.com/photo-1773734313426-2bbf04d368d1?w=1200&q=80)

雪后的景山，是摄影爱好者的天堂。当皑皑白雪覆盖在黄色琉璃瓦上，紫禁城在白雪的映衬下更显庄严壮丽。万春亭上的雪景，是北京冬季最经典的画面之一。

## 历史小知识

- **金代**：此处为金中都的"瑶屿行宫"，是皇家游幸之所。
- **元代**：成为元大都的中心，称"青山"。
- **明代**：永乐年间修建紫禁城时，将拆除元故宫的渣土堆积于此，形成"万岁山"，又称"镇山""堆煤山"。
- **清代**：顺治年间改名为"景山"，取"仰瞻高山，景行行止"之意。
- **民国**：1928年正式开放为公园。
- **现代**：现为全国重点文物保护单位，国家AAAA级旅游景区。
