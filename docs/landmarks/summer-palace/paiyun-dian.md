# 排云殿

![排云殿](https://images.unsplash.com/photo-1704852285252-cf86631cbe22?w=1200&q=80)

> "神仙排云出，但见金银台。"
> —— 郭璞《游仙诗》中的仙境，慈禧万寿节的舞台

## 简介

排云殿位于万寿山前山中央，背倚[佛香阁](/landmarks/summer-palace/foxiang-ge)，面朝昆明湖，是颐和园中轴线上的核心殿宇，也是慈禧太后举行万寿庆典、接受朝贺的场所。

殿名取自晋代郭璞《游仙诗》"神仙排云出，但见金银台"之句，寓意此处如仙山琼阁，云雾缭绕间神仙排云而出——既写山势之高，也暗合慈禧自比"老佛爷"的心思。

这里的前身是大报恩延寿寺，始建于乾隆十五年（1750年），为乾隆庆祝母亲六十大寿所建。1860年寺院毁于英法联军之手，光绪年间慈禧重建时，将寺院改为殿堂，命名"排云殿"。此后每年十月初十慈禧寿辰，光绪帝率王公百官在此行三跪九叩大礼，是晚清宫廷最隆重的典礼现场。

| 项目 | 内容 |

| **殿宇规制** | 面阔五间，重檐歇山顶，黄琉璃瓦，前后出廊 |

| **殿前陈设** | 铜狮、铜缸、铜鹿、铜鹤、铜鼎，月台设铜鼎香炉 |
| **殿内陈设** | 慈禧御座、宝座屏风、掌扇宫灯、"复殿留云"匾 |
| **功能** | 慈禧万寿节受贺、朝会大典之所 |

排云殿建筑群沿山势层层递升，以排云门为起点，经芳辉殿、紫霄殿夹峙两翼，主殿排云殿踞于高台之上，其后德辉殿紧接登山磴道直通佛香阁。

殿前月台宽阔，可陈设仪仗卤簿，容纳百官朝贺。殿内陈设为光绪朝原状：正中设慈禧御座，宝座后立屏风，两侧掌扇、宫灯俱全，一派皇家威仪。

## 历史故事

### 十月初十的万寿盛典

慈禧生于道光十五年十月初十（1835年11月29日），此后每逢此日便是清宫最盛大的节日。颐和园重建后，万寿节庆典移师排云殿：天未亮，光绪帝已率群臣在排云门外候驾，吉时一到，慈禧升座，百官山呼"万寿无疆"，三跪九叩，乐舞齐奏。

据档案记载，庆典前数日，从东宫门到排云殿一路扎彩棚、悬宫灯，昆明湖上龙舟竞渡，极尽铺张。彼时国库空虚、外患频仍，排云殿前的每一次叩拜，都是晚清"量中华之物力，结与国之欢心"的注脚。

### 六十大寿的遗憾

1894年（甲午年）本是慈禧六十大寿，朝廷早早在排云殿筹备庆典，挪用海军军费修缮园林、置办戏装。然而这一年甲午战争爆发，北洋水师节节败退，群臣弹劾，慈禧被迫下旨"庆典从简"，万寿节改在紫禁城内草草举行。

排云殿前张好的彩棚黯然撤去，两年后《马关条约》签订，割地赔款。一座殿宇的庆典兴衰，恰是一个帝国的黄昏缩影。

### 从佛寺到朝堂

排云殿的前身大报恩延寿寺，本是乾隆为母亲祈福的佛门净地，香火鼎盛近百年。慈禧将其改为朝贺殿堂，佛堂变朝堂，颂经声换成山呼声——建筑的功能转换，折射出这座园林从"帝王孝亲"到"太后专权"的气质之变。有趣的是，寺内部分佛像并未移走，至今山巅智慧海仍供着琉璃佛像，佛与权在同一座山上共处了一百多年。

## 必看亮点

1. **中轴线气象**：排云门至佛香阁的层层台地，是颐和园最见皇家气派的轴线，殿前回望昆明湖，湖山一览无余。
2. **万寿庆典陈列**：殿内复原慈禧受贺场景，御座、屏风、贡品清单一目了然，比空殿更有故事。
3. **殿前铜兽**：铜狮、铜缸、铜鹤、铜鹿列置月台，多为乾隆朝清漪园旧物，躲过1860年大火，弥足珍贵。
4. **芳辉殿与紫霄殿**：两翼配殿现为宫廷生活展室，展示慈禧起居用品，细节处见奢华。
5. **登山起点**：从排云殿后的德辉殿拾级而上，二十分钟即可登顶[佛香阁](/landmarks/summer-palace/foxiang-ge)，是体力游的最佳路线。

## 拍照机位

1. **月台回望**：站在排云殿月台南缘俯拍，昆明湖与十七孔桥尽收镜头，前景是铜兽与汉白玉栏杆。
2. **中轴仰拍**：从排云门内仰拍，殿宇、飞檐、佛香阁层层叠起，气势十足。
3. **铜兽特写**：清晨侧光下，月台铜鹤铜鹿的金属质感最动人。

## 参观贴士

1. 排云殿需持联票或另购票参观，属园中园范围，与佛香阁同票通用。
2. 殿内禁止拍照，但殿前月台和铜兽区可以尽情拍，清晨侧光时铜器质感最佳。
3. 中轴线台阶陡峭，老人小孩建议缓行；不想爬山者可只参观殿内陈列，登山段量力而行。
4. 排云殿与[长廊](/landmarks/summer-palace/changlang)仅一墙之隔，从长廊中部的排云门进入最为顺路。
5. 逛完排云殿沿中轴登顶佛香阁，再下山至[石舫](/landmarks/summer-palace/shifang)，是[北宫门线](/landmarks/summer-palace/)最经典的山水连游。
6. 想感受"万寿节"氛围，可留意每年十月初十前后的园方纪念活动。

## 常见问题

**Q：排云殿和佛香阁是什么关系？**

二者在同一条中轴线上：排云殿是山脚的朝贺正殿，[佛香阁](/landmarks/summer-palace/foxiang-ge)是山巅的宗教地标，一殿一阁，一俗一圣，拾级可连游。

**Q：排云殿现在做什么用？**

殿内辟有慈禧万寿庆典陈列，复原了御座、屏风与受贺场景，比空殿更有看头。

**Q：万寿节是哪一天？**

农历十月初十，慈禧的生日。这一天是晚清宫廷最隆重的节日，王公百官要在排云殿行三跪九叩大礼。

**Q：从排云殿爬到佛香阁要多久？**

经德辉殿登山磴道，约20-30分钟；台阶陡峭，量力缓行，回望湖景则越爬越值。
