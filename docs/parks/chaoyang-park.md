# 朝阳公园

> "四环内最大的城市公园，音乐节的举办地。"

## 简介

朝阳公园位于北京市朝阳区，是北京市四环内最大的城市公园。公园占地约290公顷，其中水面约68公顷，是北京市民休闲娱乐的重要场所。朝阳公园也是北京著名的音乐节举办地。

## 历史沿革

### 水碓公园

朝阳公园前身是水碓公园，1984年开始建设，1987年建成开放。

### 改名朝阳公园

1992年，水碓公园改名"朝阳公园"，并进行大规模扩建。

### 现代发展

2000年后，朝阳公园不断完善设施，增加游乐项目，成为北京市民喜爱的城市公园。

## 园区布局

朝阳公园分为多个景区。

### 中心岛

中心岛是公园的核心景区，有万人广场、世纪喷泉等。

#### 万人广场

万人广场是朝阳公园最大的广场，可容纳数万人。这里是音乐节的主要举办地。

#### 世纪喷泉

世纪喷泉是朝阳公园的标志性景观，喷泉水柱最高可达数十米。

### 沙滩排球场

沙滩排球场是北京沙滩排球的训练基地，有专业的沙滩排球场。

### 游乐园

公园内有游乐园，有各种游乐设施。

| 项目 | 说明 |
|:---|:---|
| **过山车** | 刺激的过山车 |
| **旋转木马** | 经典的旋转木马 |
| **碰碰车** | 好玩的碰碰车 |
| **摩天轮** | 浪漫的摩天轮 |

### 水上游乐

公园内有大片水面，可以划船、脚踏船等。

## 音乐节

朝阳公园是北京著名的音乐节举办地。

### 草莓音乐节

草莓音乐节是国内著名的音乐节品牌，每年在朝阳公园举办。

#### 举办时间

每年5月左右，具体时间以官方公告为准。

#### 参演艺人

邀请国内外知名乐队和歌手参演。

### 其他音乐节

朝阳公园还举办其他各种音乐节和演出活动。

## 游览攻略

### 推荐路线

**经典路线（约2-3小时）**：

南门 → 万人广场 → 世纪喷泉 → 游乐园 → 沙滩排球场 → 水上游乐 → 北门

**音乐节路线**：

万人广场 → 舞台区 → 美食区 → 休息区

6. **夜间活动**：夜间有灯光秀等活动
