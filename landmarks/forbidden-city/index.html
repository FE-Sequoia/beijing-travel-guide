故宫博物院 ​

"一座紫禁城，半部中国史。" 六百年红墙黄瓦间，二十四位帝王的家国往事仍在回响。

简介 ​

故宫，旧称紫禁城，是明清两代二十四位皇帝的皇家宫殿，也是世界上现存规模最大、保存最完整的木质结构古建筑群。它始建于明永乐四年（1406年），历时十四年建成，此后近五百年间始终是中国的权力中心。1925年，紫禁城辟为故宫博物院，昔日禁地终向世人敞开大门。

故宫占地72万平方米，建筑面积约15万平方米，传说有房屋九千九百九十九间半（实测约八千七百余间）。整座宫殿沿北京城中轴线南北展开，分为"外朝"与"内廷"两大部分：外朝以太和殿、中和殿、保和殿三大殿为中心，是举行国家大典之所；内廷以乾清宫、交泰殿、坤宁宫后三宫为中心，是帝后居住之地。左右对称、前朝后寝的布局，处处体现着"天人合一"与皇权至上的理念。

1987年，故宫被列入《世界文化遗产名录》。如今，它收藏文物超过186万件，是中国最大的古代文化艺术博物馆。

子景点导览 ​

外朝三大殿 ​

午门 — 故宫正门，俗称"五凤楼"。高37.95米的城楼翼然展开，五个门洞暗藏玄机。这里是"推出午门斩首"误传的源头，也是皇帝举行献俘大礼的威仪之地。

太和殿 — 俗称"金銮殿"，中国现存最大的木结构大殿。明清两代登基、大婚、册封、命将出征等最隆重的国家大典均在此举行，殿内金漆龙椅与蟠龙藻井是皇权的终极象征。

中和殿 — 三大殿中唯一的方形攒尖顶建筑，玲珑如亭。皇帝赴太和殿大典前在此小憩，也在此阅视祭天的祝版与亲耕的种子农具，取"允执厥中"之意。

保和殿 — 清代殿试的固定考场，天子亲自主持的最高等级科举考试在此举行。殿后那块重达250吨的"云龙阶石"，是故宫最大的石雕，堪称工程奇迹。

内廷后三宫 ​

乾清宫 — 内廷之首，明代皇帝的寝宫与日常理政之所。殿内高悬的"正大光明"匾额背后，曾藏着决定帝国命运的秘密立储诏书。

交泰殿 — 乾清、坤宁之间的方形小殿，皇后生辰在此受贺，清代二十五方御用宝玺珍藏于此。殿内的铜壶滴漏与大自鸣钟，是紫禁城的时间之心。

坤宁宫 — 明代皇后的正宫，清代改为萨满祭祀场所，每日宰猪祭神。东暖阁则是皇帝大婚的洞房，康熙、同治、光绪三位皇帝在此度过新婚之夜。

园林与专馆 ​

御花园 — 中轴线北端的皇家花园，堆秀山的太湖石假山、供奉玄武大帝的钦安殿、四百年古柏藤萝，在方寸之间营造出皇家园林的精致与幽深。

珍宝馆（宁寿宫区） — 清代宫廷珍宝的宝库，金瓯永固杯、大禹治水图玉山、皇后凤冠……每一件都是倾国之力的艺术杰作。需另购票10元。

钟表馆（奉先殿） — 收藏中外钟表200余件，从英国的写字人钟到清宫造办处的花园式钟，展示18至20世纪钟表制造的巅峰水平。需另购票10元。

九龙壁 — 宁寿宫区皇极门外的琉璃照壁，九条巨龙腾跃于云海波涛之间。传说其中一块构件是楠木"冒充"琉璃烧制，瞒过了乾隆帝的眼睛。

推荐游览路线 ​

🕐 半日精华游（约3小时） ​

沿中轴线一路向北，不走回头路：

午门 → 太和门广场 → 太和殿 → 中和殿 → 保和殿（顺路看殿后云龙阶石）→ 乾清门 → 乾清宫 → 交泰殿 → 坤宁宫 → 御花园 → 神武门出

这条路线串联了外朝三大殿与内廷后三宫，涵盖故宫最核心的建筑，适合时间紧凑的初访者。出神武门后可直接登上对面的景山万春亭，俯瞰紫禁城全景。

🗓️ 一日深度游（约6-7小时） ​

上午走中轴线精华（同半日游路线），下午探索两翼：

上午：午门 → 三大殿 → 后三宫 → 御花园 下午：从御花园折返，东线游览珍宝馆（宁寿宫区，顺路看九龙壁）→ 钟表馆（奉先殿）→ 东六宫 → 神武门或东华门出

一日路线可尽览故宫精华与两大专馆。建议自带干粮，馆内餐饮选择有限且价格较高。

实用贴士 ​

  • 务必提前预约：故宫实行实名预约制，门票提前7天开售，旺季往往开售即罄，建议定好闹钟准时抢票。
  • 带身份证：入院需刷预约时使用的身份证原件，无身份证无法入场。
  • 周一闭馆：除法定节假日外，每周一闭馆，规划行程时务必避开。
  • 轻装出行：故宫内步行距离长（中轴线全程约2公里），穿舒适的鞋；大件行李不允许带入。
  • 从南向北：午门是唯一入口，参观方向为自南向北，出北门神武门后可顺游景山公园。
  • 拍照时段：早晨开馆后和下午闭馆前光线最佳、游客最少，是拍摄空旷宫殿的黄金时间。

文化故事精选 ​

为什么叫"紫禁城"？ ​

"紫"取自天上的紫微垣——古人认为那是天帝居住的星宫；"禁"指皇家禁地，寻常百姓不得擅入。皇帝的居所对应天帝之宫，故名"紫禁城"。

"推出午门斩首"是真的吗？ ​

这是流传最广的历史误传。午门是举行献俘礼和"廷杖"（当众杖打大臣）的地方，并非刑场。明代确有官员在午门受廷杖而亡，但"斩首"一说纯属以讹传讹。详见午门。

九千九百九十九间半房 ​

传说紫禁城有房屋9999间半，因为天庭的宫殿是一万间，人间的皇帝不敢僭越天帝的数目。实测故宫房屋约8707间，但"九千九百九十九间半"的说法已成为紫禁城最迷人的民间想象。

龙椅背后的秘密 ​

太和殿的金漆龙椅是明代嘉靖年间遗物，椅背雕有蟠龙，椅座为金漆髹龙。它不只是一把椅子，而是皇权的天命象征——袁世凯称帝时曾换上一把西式高背椅，很快就被移走了。
