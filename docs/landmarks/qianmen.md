# 前门大街

![前门大街](https://images.unsplash.com/photo-1542079668-197625d11054?w=1200)

> "老北京商业的缩影，百年老字号的聚集地。"

## 简介

前门大街位于北京市中轴线上，北起正阳门（前门），南至珠市口，全长845米。前门大街是北京最著名的商业街之一，历史悠久，商铺林立，老字号云集，是体验老北京商业文化的绝佳去处。

## 历史沿革

### 形成

前门大街形成于明代永乐年间。明成祖迁都北京后，在前门（正阳门）外形成了繁华的商业区。

### 清代繁荣

清代，前门大街进入鼎盛时期。由于内城不允许开设商铺，商业活动集中在正阳门外，前门大街成为北京最繁华的商业街。

### 民国时期

民国时期，前门大街继续繁荣。电车（铛铛车）的开通使前门大街更加热闹。

### 现代改造

2007年，前门大街进行大规模改造，恢复了清末民初的风貌。铛铛车重新开通，老字号重新聚集。

## 主要景点

### 正阳门

正阳门俗称前门，是明清北京内城的正门，也是北京城最高的城门楼。正阳门由城楼和箭楼组成，是北京城的标志性建筑。

#### 正阳门城楼

- **高度**：约40米
- **结构**：重檐歇山顶，灰筒瓦绿剪边
- **功能**：明清皇帝出巡必经之门

#### 正阳门箭楼

箭楼位于城楼前方，是防御性建筑。箭楼四面有箭窗，用于射击。

### 铛铛车

前门大街上的铛铛车（有轨电车）是旅游观光车，复刻了老北京电车的模样。

- **票价**：单程20元
- **路线**：前门大街往返
- **体验**：叮当作响，复古怀旧

## 游览攻略

### 推荐路线

**前门漫步（约2-3小时）**：

前门地铁站 → 正阳门 → 前门大街（由北向南） → 铛铛车体验 → 大栅栏 → 小吃街

**结合天安门路线（约半天）**：

天安门广场 → 正阳门 → 前门大街 → 大栅栏 → 午餐（全聚德烤鸭）

### 交通方式

| 方式 | 站点 |
|:---|:---|
| 地铁 | 2号线前门站B口或C口 |
| 公交 | 前门站，多条线路 |
| 铛铛车 | 前门大街观光 |

### 实用贴士

1. **铛铛车体验**：推荐乘坐铛铛车，复古有趣
2. **老字号购物**：可以购买布鞋、丝绸等特产
3. **美食推荐**：全聚德烤鸭、都一处烧麦
4. **夜景欣赏**：夜景灯光漂亮，适合拍照
5. **避开高峰**：周末人很多，建议工作日前往

## 文化意义

前门大街是北京商业文化的缩影，老字号承载着北京人的记忆和情感。在这里，你可以品尝正宗的北京烤鸭，购买传统的老北京布鞋，感受老北京的商业氛围。

"头顶马聚源，脚踩内联升，身穿瑞蚨祥，腰缠四大恒"，这句老话形象地描述了老北京人对前门的印象。
