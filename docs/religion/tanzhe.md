# 潭柘寺

> "先有潭柘寺，后有北京城。"

## 简介

潭柘寺位于北京市门头沟区，是北京最古老的佛教寺院之一，也是北京历史最悠久的名刹。潭柘寺始建于西晋愍帝建兴四年（316年），距今已有1700余年历史，比北京城建城还早800年，故有"先有潭柘寺，后有北京城"之说。

## 历史沿革

### 始建

潭柘寺始建于西晋建兴四年（316年），原名嘉福寺，是北京最早的佛教寺院。

### 发展

- **唐代**：华严宗高僧在此讲经
- **金代**：金熙宗曾到此进香
- **元代**：元世祖忽必烈的女儿妙严公主在此出家
- **明代**：明成祖朱棣曾到此进香
- **清代**：康熙、乾隆多次到此进香，潭柘寺达到鼎盛

### 近代

民国时期，潭柘寺逐渐衰落。新中国成立后，潭柘寺得到保护，1980年代修复后对外开放。

## 建筑布局

潭柘寺依山而建，坐北朝南，建筑层层递升。

### 山门

山门是潭柘寺的入口，门前有石桥。

### 天王殿

天王殿是潭柘寺的第一重大殿，供奉弥勒佛和四大天王。

### 大雄宝殿

大雄宝殿是潭柘寺的主殿，供奉释迦牟尼佛。

#### 琉璃鸱吻

大雄宝殿顶上有琉璃鸱吻，是潭柘寺的标志性建筑。

### 帝王树

潭柘寺内有一棵古银杏树，高30余米，胸径约4米，植于辽代，距今已有千年历史，称为"帝王树"。

#### 帝王树传说

相传每换一个皇帝，帝王树就会长出一个新枝，因此得名"帝王树"。

### 配王树

帝王树旁还有一棵"配王树"，也是千年银杏。

### 流杯亭

流杯亭是潭柘寺的特色建筑，亭内有水槽，可以将酒杯放在水中漂流。

### 楞严坛

楞严坛是潭柘寺的戒坛，是僧人受戒的地方。

### 塔林

潭柘寺后有塔林，埋葬着历代高僧。

## 游览攻略

### 推荐路线

**经典路线（约3-4小时）**：

山门 → 天王殿 → 大雄宝殿 → 帝王树 → 配王树 → 流杯亭 → 楞严坛 → 塔林 → 出寺

### 交通方式

| 方式 | 路线 |
|:---|:---|
| 公交 | 地铁1号线苹果园站 → 931路 → 潭柘寺 |
| 自驾 | 莲石路 → 108国道 → 潭柘寺 |
| 旅游专线 | 前门有旅游专线 |

### 实用贴士

1. **门票较贵**：门票50元
2. **帝王树必看**：千年银杏是精华
3. **秋季最美**：秋季银杏金黄，最美
4. **斋饭**：寺内有斋饭供应
5. **爬山**：寺庙依山而建，需要爬山
6. **结合游览**：可与戒台寺一起游览（相距约10公里）

## 文化价值

潭柘寺是北京最古老的佛教寺院，有"先有潭柘寺，后有北京城"之说。

潭柘寺保存了大量珍贵文物，是研究北京佛教史的重要场所。

潭柘寺的"帝王树"是千年银杏，是北京最古老的树木之一。
