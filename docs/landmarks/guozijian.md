# 国子监与孔庙

![国子监与孔庙](https://images.unsplash.com/photo-GTno1mONTfo?w=1200)

> "千年学府，万世师表。"

## 简介

国子监与孔庙位于北京市东城区国子监街，是中国元、明、清三代的最高学府和祭祀孔子的场所。孔庙是祭祀中国古代伟大思想家、教育家孔子的庙宇，国子监是中国古代最高教育行政机关和最高学府。

## 孔庙

### 历史沿革

孔庙始建于元代大德六年（1302年），是元、明、清三代祭祀孔子的场所。孔庙的建筑规模仅次于山东曲阜孔庙，是全国第二大孔庙。

#### 建筑布局

孔庙的建筑沿中轴线对称分布，主要建筑有大成门、大成殿、崇圣祠等。

| 建筑 | 功能 | 特色 |
|:---|:---|:---|
| **大成门** | 孔庙正门 | 面阔五间，单檐歇山顶 |
| **大成殿** | 主殿，祭祀孔子 | 面阔九间，重檐庑殿顶 |
| **崇圣祠** | 祭祀孔子祖先 | 面阔五间，单檐歇山顶 |
| **碑亭** | 存放御制碑 | 元代至清代御制碑 |

### 主要看点

#### 大成殿

大成殿是孔庙的主殿，殿内供奉着孔子的塑像。大成殿的建筑等级很高，仅次于故宫太和殿。

- **殿名由来**："大成"出自《孟子》，意为孔子集古圣先贤之大成
- **建筑规模**：面阔九间，进深五间
- **屋顶形式**：重檐殿顶，黄色琉璃瓦

#### 碑林

孔庙内保存有大量历代御制碑和题名碑，是研究中国古代教育和科举制度的重要资料。

## 国子监

### 历史沿革

国子监始建于元代大德十年（1306年），是中国古代最高教育行政机关和最高学府。国子监不仅培养了大批人才，也是中国古代教育制度的重要见证。

#### 建筑布局

国子监的建筑沿中轴线分布，主要建筑有集贤门、太学门、辟雍殿、彝伦堂等。

| 建筑 | 功能 | 特色 |
|:---|:---|:---|
| **集贤门** | 国子监正门 | 面阔三间，单檐歇山顶 |
| **辟雍殿** | 皇帝讲学之所 | 方形，重檐攒尖顶 |
| **彝伦堂** | 国子监藏书之所 | 面阔七间，单檐歇山顶 |
| **敬一亭** | 祭祀之所 | 六角形，重檐攒尖顶 |

### 主要看点

#### 辟雍殿

辟雍殿是国子监的核心建筑，是皇帝临雍讲学的场所。辟雍殿的建筑形式独特，是中国古代建筑中的精品。

- **建筑形式**：方形，重檐攒尖顶
- **屋顶形式**：黄色琉璃瓦，四角攒尖
- **内部陈设**：皇帝宝座、讲经台等

#### 十三经碑林

国子监内保存有清代镌刻的《十三经》石碑，共189块，是中国现存最完整的石刻经书。

- **刻经时间**：清代雍正年间
- **石碑数量**：189块
- **内容**：《周易》《尚书》《诗经》等十三部儒家经典

## 文化意义

### 儒家文化的传承

孔庙和国子监是中国儒家文化的重要载体，见证了中国古代教育制度的发展和演变。

### 科举制度的见证

国子监是中国古代科举制度的重要见证，培养了大量的人才，对中国古代政治、文化产生了深远的影响。

### 建筑艺术的精品

孔庙和国子监的建筑体现了中国古代建筑的精华，是研究中国古代建筑艺术的重要实物资料。

## 游览攻略

### 推荐路线

**精华路线（约1.5小时）**：

孔庙：先师门 → 大成门 → 大成殿 → 崇圣祠 → 碑亭
国子监：集贤门 → 太学门 → 辟雍殿 → 彝伦堂 → 敬一亭

### 实用贴士

1. **最佳时间**：工作日人较少，适合慢慢参观
2. **讲解服务**：建议租用讲解器或请导游，了解更多历史知识
3. **拍照位置**：大成殿前、辟雍殿前都是绝佳拍照点
4. **周边游览**：参观完后可以到雍和宫、五道营胡同游览
5. **文化体验**：可以带孩子来此感受传统文化氛围
