# 北京推荐旅游路线

> "条条大路通北京，步步皆是好风光。"

## 按天数规划

### 一日游（经典打卡）

| 时间 | 景点 | 活动 |
|:---:|:---|:---|
| 6:00 | 天安门广场 | 看升旗仪式 |
| 8:30 | 故宫 | 从午门进入 |
| 12:00 | 景山 | 登景山俯瞰 |
| 14:00 | 北海公园 | 白塔、九龙壁 |
| 16:00 | 什刹海 | 胡同游 |
| 18:00 | 晚餐 | 后海或南锣鼓巷 |

**门票预算**：约72元

---

### 两日游（精华深度）

#### Day 1：皇城精华
- 天安门广场 → 故宫 → 景山 → 北海 → 什刹海

#### Day 2：园林与学府
- 颐和园 → 圆明园 → 清华/北大 → 鸟巢水立方

**门票预算**：约157元

---

### 三日游（经典全览）

#### Day 1：皇城文化
- 天安门 → 故宫 → 景山 → 北海 → 什刹海

#### Day 2：园林与学府
- 颐和园 → 圆明园 → 清华/北大 → 奥林匹克公园

#### Day 3：长城与十三陵
- 八达岭长城 → 明十三陵 → 南锣鼓巷

**门票预算**：约260元

---

### 五日游（深度体验）

- Day 1：天安门 → 故宫 → 景山 → 北海
- Day 2：颐和园 → 圆明园 → 清华/北大 → 鸟巢
- Day 3：八达岭长城 → 明十三陵 → 王府井
- Day 4：天坛 → 前门 → 国博 → 正阳门
- Day 5：南锣鼓巷 → 什刹海 → 恭王府 → 钟鼓楼

---

## 主题路线

### 皇城文化线
**路线**：天安门 → 故宫 → 景山 → 北海 → 天坛 → 颐和园
**用时**：2-3天

### 胡同文化线
**路线**：南锣鼓巷 → 什刹海 → 烟袋斜街 → 钟鼓楼 → 恭王府
**用时**：1-2天

### 近代历史线
**路线**：天安门 → 人民英雄纪念碑 → 国博 → 北大红楼 → 圆明园
**用时**：1-2天

### 亲子游线
**路线**：故宫 → 科技馆 → 自然博物馆 → 天文馆 → 动物园
**用时**：2-3天

---

**💡 Tips**：北京景点众多，建议根据自己的时间和兴趣选择路线。第一次来北京，推荐选择"三日游"经典路线。
