# 茅盾故居

![茅盾故居](https://images.unsplash.com/photo-Uk19VsygF9Y?w=1200)

> "中国现代文学的奠基人，在胡同深处静静守望。"

## 简介

茅盾故居位于北京市东城区后圆恩寺胡同13号，是一座典型的北京四合院。茅盾先生从1974年至1981年在此居住，度过了人生的最后7年。故居现已成为"茅盾故居纪念馆"，展示着这位文学巨匠的生平事迹和珍贵文物。

## 历史沿革

### 清代府邸

茅盾故居原为清代的一座四合院，后成为普通民居。1974年，茅盾先生迁入此院。

### 茅盾晚年

1974年至1981年，茅盾先生在此居住，继续从事文学创作和学术研究。

#### 晚年成就

| 年份 | 成就 |
|:---|:---|
| 1974年 | 迁入后圆恩寺胡同13号 |
| 1975年 | 继续创作回忆录 |
| 1981年 | 逝世于此 |

## 故居布局

### 四合院结构

茅盾故居是一座典型的北京四合院，分为外院和内院。

#### 建筑布局

| 建筑 | 功能 | 特色 |
|:---|:---|:---|
| **大门** | 入口 | 传统的朱漆大门 |
| **影壁** | 照壁 | 简朴的砖雕影壁 |
| **正房** | 茅盾居室 | 保留原貌 |
| **东厢房** | 书房 | 茅盾生前书房 |
| **西厢房** | 展览室 | 生平事迹展 |

### 主要展室

#### 正房展室

正房是茅盾先生的居室，保留了他生前的生活场景。

- **卧室**：简朴的卧室，摆放着简朴的家具
- **会客室**：会客的地方，摆放着沙发和茶几
- **工作室**：茅盾生前工作的地方

#### 西厢房展室

西厢房是茅盾生平事迹展，展示了茅盾先生一生的重要经历。

## 珍贵文物

### 手稿和著作

茅盾故居收藏了大量茅盾先生的手稿和著作。

| 文物 | 说明 |
|:---|:---|
| **手稿** | 《子夜》《春蚕》等手稿 |
| **著作** | 各种版本的著作 |
| **书信** | 与友人的往来书信 |
| **日记** | 茅盾的日记 |

### 生活用品

茅盾故居还保留了茅盾先生生前使用的生活用品。

- **家具**：简朴的木质家具
- **文具**：笔、墨、纸、砚
- **藏书**：大量珍贵藏书

## 游览攻略

### 推荐路线

**精华路线（约1小时）**：

入口 → 影壁 → 正房（茅盾居室） → 东厢房（书房） → 西厢房（生平展） → 出口

### 实用贴士

1. **提前预约**：建议提前在官网或电话预约
2. **最佳时间**：工作日人较少，适合慢慢参观
3. **拍照位置**：影壁、正房前都是绝佳拍照点
4. **讲解服务**：可以租用讲解器，了解更多历史知识
5. **周边游览**：参观完后可以到南锣鼓巷、什刹海游览

## 文化意义

茅盾是中国现代文学史上的重要人物，他的小说《子夜》《春蚕》等在中国文学史上占有重要地位。茅盾故居不仅是一座名人故居，更是研究中国现代文学史的重要场所。

"中国现代文学的奠基人"，这个称号完美地概括了茅盾先生在中国文学史上的地位。
