颐和园 ​

"虽由人作，宛自天开。" —— 中国造园艺术的最高境界，颐和园正是其巅峰之作

简介 ​

颐和园位于北京西郊，是以昆明湖、万寿山为基址，以杭州西湖为蓝本，汲取江南园林精华而建的大型山水园林，也是中国现存规模最大、保存最完整的皇家行宫御苑，被誉为"皇家园林博物馆"。全园占地约290公顷，其中昆明湖约占四分之三，万寿山雄踞湖北，殿宇楼阁依山就势，湖光山色浑然一体。

颐和园始建于清乾隆十五年（1750年），是乾隆为母亲崇庆皇太后祝寿而建的"清漪园"，工程历时十五年，耗银480万两，为"三山五园"中最后建成的一座。咸丰十年（1860年），清漪园遭英法联军焚掠，珍宝荡然无存。光绪十二年（1886年），慈禧太后挪用海军军费重建，取"颐养冲和"之意改名"颐和园"，作为其夏季避暑听政之所。1900年园内建筑再遭八国联军破坏，两年后修复。1924年颐和园正式对公众开放，1961年列为全国重点文物保护单位，1998年被联合国教科文组织列入《世界遗产名录》。

推荐游览路线 ​

北宫门线（4-5小时，山水之选） ​

北宫门进 → 苏州街 → 须弥灵境 → 佛香阁 → 排云殿 → 长廊 → 石舫 → 乘船至南湖岛 → 十七孔桥 → 铜牛 → 东宫门或新建宫门出

适合人群：初次到访、体力较好者。由山至水，一路下行，尽览前山精华。

东宫门线（4-5小时，人文之选） ​

东宫门进 → 仁寿殿 → 德和园 → 玉澜堂 → 乐寿堂 → 长廊 → 排云殿 → 佛香阁 → 石舫 → 沿西堤漫步 → 新建宫门出

适合人群：喜欢历史人文、想细看宫廷旧迹者。先殿宇后山水，节奏从容。

实用贴士 ​

  1. 联票更划算：联票已含佛香阁、德和园、苏州街、文昌院，单买这些园中园反而更贵，时间充裕建议直接买联票。
  2. 早到避人流：旺季开园即入，先直奔佛香阁或长廊，避开上午10点后的大客流。
  3. 善用游船：石舫至南湖岛的游船（大船40元）既省力又是移动观景台，可从湖上回望佛香阁全景。
  4. 穿舒适鞋：万寿山台阶陡峭，环湖一圈十余公里，一双好鞋比什么都重要。
  5. 预留拍照时间：佛香阁俯瞰（南湖岛）、十七孔桥夕阳（东堤）、长廊光影，都是出片圣地，日落前1小时光线最佳。
  6. 周边联游：可与圆明园安排在同一天，两园相距仅数公里，感受"毁灭与重生"的历史对照。
